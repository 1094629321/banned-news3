 <p>
  5月3日“世界新闻自由日”来临之际，自由亚洲电台台长Bay Fang发表评论文章指出，在当前国际公共卫生危机中，新闻媒体的角色比任何时候都更重要。以下是本台粤语部翻译摘要：
  <br/>
  <br/>
  自由亚洲电台台长Bay Fang的文章说，阻碍了中共及其他专制国家统治者向国民隐瞒其处理疫情的做法。独立的报道让民众获得需要的资讯，也帮助国际社会跨越政治党派和国界共同遏制疫情的发展。
  <br/>
  <br/>
  中国政府处理疫情缺乏透明度，中国外交部发言人赵立坚早前坚称依据现实的数据，指控“可能是美国把疫情带到了武汉”。不过，自由亚洲电台的报道却对中国的官方数字提出了质疑。本台三月报道，从武汉殡仪馆透露的数字计算，当地七家殡仪馆每天可发放3500个骨灰盒，从3月23日开始至清明节前12天，总共便发放42000个骨灰盒，估计当地因新冠肺炎死亡人数远超当局公布的数字十倍。
  <br/>
  <br/>
  北京当局之后修正武汉的新冠肺炎死亡人数，上调死亡数字50%，由原来的1290人，修正到3869人。文章指，在日益不信任中国官媒的情况下， 许多在中国的人民都寻求独立的新闻资讯。在今年头三个月，即中国疫情爆发高峰期间，自由亚洲电台普通话组的网页和社交媒体的访问量剧增，其脸书(Facebook)的视频访问量比去年同期增长逾四倍。
  <br/>
  <br/>
  中国当局早期不但隐瞒疫情，还对发出疫情警报的病毒研究人员、医生和律师进行打压。在3月，疫情肆虐全球，中国当局还驱逐《华尔街日报》、《纽约时报》、《华盛顿邮报》等外国新闻机构的记者。
  <br/>
  <br/>
  Bay Fang 的文章还指，除中国之外，自由亚洲电台针对广播的多个专制国家政府，亦有不同程度的信息控制，并利用疫情大流行来巩固权力。例如柬埔寨最近通过了《紧急状态法》，该法可规范公民的权利，并允许当局对电信、新闻机构和社交媒体进行完全的监管。
  <br/>
  <br/>
  文章说，这些国家的行为不会令人惊讶，因为他们长期以来采取遏抑资讯的中国模式，无视公众利益。此时此刻，这种态度是危险及不负责任。政策制定者和医护人员的抗疫防疫，需要可靠的数据和资讯。能够确保政策制定者和公民获取这些知识，是新闻记者，而非官方的宣传口舌。
  <br/>
  <br/>
  文章最后说: 在我们纪念世界新闻日之际，当前人类面临最具挑战性的时刻，新闻工作者扮演的角色至关重要。为我们提供信息的记者、编辑和所有协助这努力的从业人员，在这次疫情危机中都是“必要人员”。尽管受到威胁、 恐吓和审查，我们(记者)的使命一定要坚持下去，直到真相和公益获得伸张。

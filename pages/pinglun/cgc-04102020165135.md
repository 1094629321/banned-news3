  律师王全璋遭中共构陷关押四年半出狱至今已经一周了。中共不但非法地不让他去北京与妻儿团聚，还以监狱感染疫情需要隔离为名，把他强行送到济南继续非法拘禁。中共安排了三名流氓在楼道里日夜守候，阻止他出门；小区门口，也有十余名公安、打手阻止亲友探访。王全璋在济南打工的堂弟因为到小区探望，被带到了派出所；前来递送朋友为他订购鲜花的快递小哥，也被带到了派出所。
  <br/>
  <br/>
  不仅如此，中共还安排湖北巴东县政府以黑社会的方式，打电话到王全璋妻子李文足远在湖北的三个姐姐家中，直接爆出她们的家庭成员和家庭住址，并且要求与他们谈谈王全璋出狱的政治和安全问题。随后就有当地派出所人员找到王全璋连襟的单位，要求他们“不要参与王全璋的事情”。这是赤裸裸地通过株连进行威胁，是党国恐怖主义的具体体现。什么叫“不要参与”？共产党公然违法胡作非为还要别人配合，中共这是到了何等无耻的程度！
  <br/>
  <br/>
  尽管这些对我来说一点也不陌生，因为当年同样的事情都在我和我家人、亲属的身上发生过，而且有过之而无不及，甚至就连“不要参与”这样的说辞都一样。可是当再次听到时，还是让人不由得怒火中烧。
  <br/>
  <br/>
  中共在“709大抓捕”中，对人权律师和人权捍卫者的打压迫害都是在以黑帮绑架、构陷关押之后，再继续公然的迫害。江天勇律师被迫害关押刑满“释放”至今已经一年多了，仍然在河南信阳老家被无理软禁；刚刚被从狱中“释放”的王全璋律师又被以“隔离”之名，软禁在家。高智晟律师被中共绑架失踪快三年了，生死不知。周世锋、屠夫（吴淦）、胡石根等仍被关押；王默、郭泉、许志永等出狱不久，就再次被中共抓捕关押；方斌、陈秋实、李泽华因为揭露中共掩盖疫情失踪，至今没有下落……。
  <br/>
  <br/>
  因此，对于中共说“隔离两周后，王全璋就会有真正的自由”，我是丝毫不相信的。根据中共黑权力集团以往的做法，中共分明是要把罪恶进行到底，不死不休。

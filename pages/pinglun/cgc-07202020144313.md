  最近，《纽约时报》报道说：“据知情人士透露，特朗普政府正在考虑全面禁止中共党员及其家人赴美旅行，起草中的总统公告还可能授权政府撤销在美中共党员及其家属的签证。特朗普是否接受该计划仍属未知。”此消息传出后如同一记惊雷炸响，在中国多个层面上引起了激烈的讨论，尤其令中共恐惧的是激起了官、吏和军三心的巨大扰动。
  <br/>
  <br/>
  从高官到各阶层党员纷纷在是继续依附中共政权获取利益、成为文明世界清算对象，为中共陪葬；还是保住未来发展前途，避免牺牲未来空间，给自己与子孙后代留下退路这两者间，作着权衡与取舍。“‘退党’二字在搜索引擎上成了热搜词”的说法，在社交媒体上广泛传播。中共外交部的华春莹也按捺不住地使出了泼妇骂街的浑身解数，说： “美方公然地选择与十四亿中国人民作对……为了政治私利，不惜绑架美国国内的民意……甚至到了丧失理智走火入魔的地步……。”
  <br/>
  <br/>
  华春莹的说法实在荒谬之极。中共是中共不是中国，共产党不等于中国人民。美国禁止你中共党员及其家人赴美旅行，就算消息属实，那也只是与你中共作对，怎么成了与中国人民作对？怎么成了站在十四亿中国人民的对立面了呢？众所周知，美国政府是美国人民投票选出的、总统组建的，它只能代表民意，无法绑架民意，否则就要下台。而你中国共产党是中国人民投票选出来代表他们吗？你华春莹明知中国人民从来没有选过共产党，甚至连选票什么样都没见过，你还在这样说，这不是正在公然绑架中国十四亿人民和你们中共一起站在美国的对立面上吗？那么，绑架人民的不是你共产党还会是谁？！
  <br/>
  <br/>
  事实上，中共政权在精神上早已丧失了凝聚力，如今即使在党内也无人相信所谓的“共产主义思想”。多年来，中共一直靠暴力绑架、盘剥中国人民，掠取资源，再通过把从人民那里榨取的民脂民膏分给中共党员干部们，或者一定程度上允许他们利用权力压榨盘剥辖区的人民获得利益来收买他们，维护这个一党独裁的共产专制体制。
  <br/>
  <br/>
  在中共建政以来七十多年的逆向淘汰后，中共体系内从上到下只剩下一群唯利是图、道德沦丧，只会阿谀奉承、投机取巧、攀附权势的官僚了。他们从思维方式到具体做事，一切都只会从利益出发。他们的思维和行为字典中，早已没有了“该不该”，只有“能不能”。
  <br/>
  <br/>
  当依附于共产党只有好处没有坏处时，他们会争先恐后地摇尾乞怜，装出一副“无比忠诚”的样子；一旦更直接、更大的利益摆在面前时，他们就会毫不犹豫地作出新的权衡与选择。此时，“忠诚”二字一定不会出现在他们的脑子里。此时，到底是中共给他们的利益大，还是能够赴美所意味着的一切更重要呢？现在中国无数中共党员和干部们正在思考寻求答案、准备作出各自的决定。全天下都看到，仅仅是一则传言中的禁令，就能引起各方如此热烈的讨论和中共党员们的扰动与重视，实质何在？答案已经不言而喻了。
  <br/>
  <br/>
  这个到目前为止尚未被证实和执行的“美国将禁止所有中共党员及其家属入境美国”的传言，在中共党员干部们中所产生的三心骚动反应充分说明：中共党员们的理想早已根本不是“共产主义”了，他们也根本不相信这个党“为人民服务”的谎言宣传，他们所向往和看重并真正信任的是美国式的自由生活方式，与可以不断追求公正的社会制度。

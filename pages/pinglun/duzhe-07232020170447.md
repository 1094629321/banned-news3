 <p>
  唐龙 （独立媒体人）
  <br/>
  <br/>
  有最新报道说，任志强先生被中共指控严重违纪违法，被开除党籍，罪名是 “对党不忠诚不老实”。其实我们都知道 “对党忠诚老实” 才是真正的罪过， 因为现在习近平说党， 其实就是指他自己，他已经对中共实行一尊独裁了。
  <br/>
  <br/>
  前一段时间， 任志强先生撰文痛斥习近平是一个 “穿着皇帝新装的小丑”， 可谓一针见血，正中七寸。于是对习近平 “忠诚老实”， 那就是对人民和正义的犯罪。为此， 任志强先生现在被 “开除出党”， 其实是他的莫大荣幸和解脱，而与习近平这样丑类结党营私， 才是羞愧和耻辱。我们声援任先生的正义良知， 也将为他能尽早重获自由努力奔走。
  <br/>
  <br/>
  而同时所发生的另一有意义事件，就是美国国务卿蓬佩奥在加州尼克松图书馆发表演讲，他指习近平是 “破产极权主义意识形态的笃信徒”， 呼吁中国人民改变共产党执政方向。应该说蓬佩奥对习近平的定性相当准确， 笔者最近也曾撰文指出习氏就是 “满清，文革和苏俄” 的三合一， 对于21世纪的中国和世界而言，恰似一个邪恶毒瘤。
  <br/>
  <br/>
  借此机会， 也想多谈几句对共产党本质的认识和透视， 因为在这方面的错误和糊涂观念还是很多的。譬如有人就说，那中国各行各业的领头精英，都是共产党员，该怎么解释？两个解释：请问在中国的精英在政治立场和身份上有其它选择吗？几乎没有。即使算有，譬如加入所谓民主党派或干脆无党派，那在中国的政治文化中是“有效选择”吗， 是事业和个人发展的障碍还是助推？答案一般是障碍。
  <br/>
  <br/>
  于是， 把一种在单一政治文化， 几无选择环境下产生的所谓 “精英站队”现象， 看成是 “对党热爱忠诚”， 那就大错特错了，大多数人只是在这种变态政治环境中的被动被迫选择而已， 任志强先生就是这方面的一个生动案例。
  <br/>
  <br/>
  要讲共产党及其主义的本质缺陷和谬误， 不妨从一个简单形象入手，很能说明些问题。即我们可把共产党及其主义看成一个红色十字架， 并沿着其纵向组织结构，与横向意识形态频谱发现其原罪和谬误。
  <br/>
  <br/>
  共产党的组织结构特点， 是愈往上愈荒谬邪恶。 作为一种激进和理想化的政治组织， 其在基层往往对普通人还有一定的煽动力和欺骗性。 但问题却往往发生在高层，因为其高层是独裁和暗斗架构，必定出妖孽，并造成全局的失误和溃败，从斯大林，毛泽东，波尔布特，齐奥塞斯库，到今天的习近平，都无一例外。
  <br/>
  <br/>
  共产党的意识型态， 在政治频谱上属于 “极左”，极端绝对的东西必定狭隘偏执， 无论是极左还是极右。而且极端的东西必定导致其它极端，即绝对权力导致绝对腐败， 炫目耀眼的乌托邦背后， 一定充满血腥和罪恶。 历史的经验和教训已反复证明以上真谛， 为此中国人一定要彻底醒悟起来。背负共产的红色十字架， 只会走向罪恶和深渊。

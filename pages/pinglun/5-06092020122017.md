  朝鲜官方朝中社星期二宣布，从当天中午12时起，韩朝官方通过朝韩联络办公室维持的通信联络渠道、朝韩军方之间的东西海通信线路和通信联络试点线路、劳动党中央委员会总部大楼和青瓦台之间的热线将全面被关闭。朝中社表示，劳动党第一副部长金与正和党中央委副委员长金英哲在前一天的对韩项目部门工作总结会议上下达了这一指示，并强调将“对韩工作”改为“对敌工作”。
  <br/>
  <br/>
  韩联社证实，韩国有关部门在当天上午通过多条朝朝联络渠道拨打朝方电话，朝方均未接听。但联合国军司令部表示，在朝鲜切断韩朝一切官方联络之际，联合国军司令部和朝军之间的直通电话仍保持畅通。通过板门店两侧架设的直通电话机，联合国军司令部与朝军当天照例在板门店进行了试通话检查线路。
  <br/>
  <br/>
  中国外交部发言人华春莹当天在北京的例行记者会上，呼吁韩朝双方通过对话协商，推进和解合作。
  <br/>
  <br/>
  就此，美国朝鲜问题国家委员会项目主任丹尼尔·韦茨发推特表示，“这是平壤当局老掉牙的戏码，不过也是危险的一步。”

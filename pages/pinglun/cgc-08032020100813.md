 <p>
  最近，中共警方破获一宗旧案，任河北省石家庄市栾城区法院代院长的赵智勇（曾用名赵志勇）竟然是23年前持枪抢劫银行运钞车的劫匪。消息一出，令举世哗然。网友惊呼：知道中共烂，只是没有想到竟然烂到了这种程度。
  <br/>
  <br/>
  据说，23年前赵智勇当兵时借回家探亲的机会，结伙持枪抢劫辛集市农村合作基金会的运钞车，导致一死两伤。人们不禁要问，一个抢劫、杀人的凶犯退役后又是怎样进入司法系统，22年后还混到了判定他人善恶的法院代院长位置上的？更可笑的是，赵智勇竟然曾经被中共评为司法系统的标杆，在担任裕华区法院执行局副局长时，还曾获得“执行能手”的称号。纵然他再会隐藏也会留有蛛丝马迹，可是这样的大案竟然过了22年才被破获，这样的大盗竟然过了22年才被发现已经坐在法院代院长宝座上。这件事再次证明，中共所有审查监督干部任用的机制都只是摆设而已。
  <br/>
  <br/>
  在中共拒绝社会监督的极权专制体制下，强盗劫匪一旦有了党赐执照，掌握了权力，就变成所谓“为人民服务”的党的干部。事实上，如果按照中共自己制定的《刑法》，贪污受贿数额超过5000元人民币就要追究刑事责任。中共党政官员中，科级以上的所有干部若经过调查，恐怕很难有人达不到被起诉标准而得幸免。因此，在中共体制内一直有着“满山都是贼，查谁谁倒霉”的说法。所以，贪不贪不是问题的关键，是不是被党拿来杀了当成给猴子看的鸡才是关键所在。对此，百姓早已心知肚明，只是受制于手中没有选票，无法作出选择，加上“口袋罪”的威胁，不敢轻易去说那会被指控为“谣言”的“遥遥领先的预言”罢了。从这个角度而言，说中共政权是由一群劫匪强盗组成，不能不说是切合实际、毫不夸张的客观描述。
  <br/>
  <br/>
  在党垄断一切权力的情况下，很多人申请入党也好，拼命努力考取公务员也罢，都只是想要成为“赵家人”，好依托党的特权更容易地获取更大的利益。如果说他们是“为人民服务”，或者真正相信党的“为共产主义事业而奋斗”的宣传，恐怕连他们自己也觉得好笑。因此，有多大机会把所谓的“国有资产”变成官员们自己的个人财产、具有多少贪污受贿的潜力，便是衡量官位好坏的重要标准。在这种情况下，有权的就会依托权力谋取私利；无权的若不安于当“韭菜”就会铤而走险谋取钱财，甚至不惜杀生害命。归根结底，这一切都是不能维护社会公正的共产专制制度所造成的恶果。
  <br/>
  <br/>
  想起上世纪90年代，由李小鹏等官二代组建的农村合作基金会依托公权力强势放高利贷，然后再利用政府扒梁牵牛、暴力殴打……逼着百姓还债的一桩桩、一幕幕，也让我想到事情可能的又一面：在社会不公、法律如儿戏的中国，还没混到法院院长位置上的赵智勇们看不惯官员子女利用公权力掠夺农民，于是结伙走上黄泥岗。

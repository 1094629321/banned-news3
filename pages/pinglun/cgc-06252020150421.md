 <p>
  <br/>
  在共产专制下的中国，很多有良知与责任感的律师因关注社会公正帮助弱势群体维权，或者是勇于为受中共当局非法打压迫害的信仰者辩护而遭中共有系统的迫害。中共迫害律师的第一步一般是通过司法部门无理吊销律师的执业证（也叫律师执照）。这些因从事人权工作维护法律尊严而被党国吊销执照的律师戏称自己为“律师后”。
  <br/>
  <br/>
  中共迫害他们并非因他们做了什么出格的事情。就拿这次中共抓捕丁家喜、许志永等一批公民来说，仅仅因为他们相约在福建厦门一起吃了一次饭。不了解情况的人可能会觉得这有点离谱，怎么会仅仅因为一起吃顿饭就被抓了呢？事实上，近十年来中共已经把很多觉醒公民的任何聚会都视为街头运动了，就连生日聚会也不例外。朋友们还记得当年胡佳兄弟带着女儿胡谦慈过生日而被抓的事情吧。可见中共对于公民的觉醒已经恐惧到了草木皆兵的程度。
  <br/>
  <br/>
  这次丁家喜、许志永等因聚餐被抓已经超过半年了，这段时间谁也不知道他们被关在哪里？最近刚刚传出他们“被批捕”的消息。据说，丁家喜律师后被烟台公安从福建带走失踪半年后，被关押在山东临沂市临沭县看守所。许志永则仍然下落不明，只是也传出“被批捕”的消息。毫无疑问的是，这半年来中共是以所谓“指定监视居住”的名义，把他们关在某处的黑监狱里。伴随的肯定是精神摧残和酷刑折磨，这些早已是中共黑监狱里的“标配”了，凡是被中共这样关押过的人无一幸免。
 </p>
 <p>
  所谓“指定监视居住”的地方就是中共法外对公民乱施大刑的地方。很多被中共关在黑监狱的人都曾有盼望着尽快进入看守所，以免受酷刑折磨的想法。本来，看守所是被称作“人间地狱”的地方，他们为什么盼望着快点到那里去呢？不言而喻，黑监狱里的酷刑不知要比看守所里残酷多少倍。
  <br/>
  <br/>
  据丁家喜律师后的妻子罗胜春说：“彭剑律师已经向临沂市公安局局长寄送了法律文书，因‘健康码’等原因，他无法亲自赶赴临沂。”类似于这样的跨省跨地区的抓捕打压迫害，非是党国高层统一指挥不可。罗胜春所担心的也正是我们所担心的，酷刑折磨基本可以确定已经发生，只是不知道他们被伤害程度如何，现在身体情况怎样……
  <br/>
  <br/>
  天作孽，犹可违，人作孽，不可活。天网恢恢，疏而不漏。相信不久的将来，中共一定会为它所做的一切恶行付出应有的代价。为了这一天的早日到来，我们要携手一起努力。在黎明前的黑暗时刻，请朋友们多多关注被中共邪恶政权视为敌人的丁家喜、许志永等勇于担当的勇士们。

 <p>
  武汉肺炎疫情凸显全球对远程工作、教学等，所需电子产品和技术的依赖。欧洲联盟担心，在智慧型手机等生产上愈来愈依赖他国供应的原物料。因此，欧盟日前提出确保稀土矿物来源的新战略。
  <br/>
  <br/>
  欧洲联盟（EU）3日提出确保稀土矿物来源的新战略，将以成立欧洲原物料联盟减少对中国、智利和南非等国家的依赖。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  武汉肺炎疫情凸显全球对远程工作、教学和通信所需要的电子产品及技术依赖，欧盟也加入与美国、中国、日本等国家的竞争，要确保通信、健康、防卫及航天领域所需的原料来源不会短缺。
 </p>

 <p>
  欧盟意识到，必须在供应绿色和数字转型的关键原料上减少风险，这些关键的原料从2017的27种，增加到2020年的30种，会被欧盟视为“关键”的原因，主要是供应不足，或是集中在少数的供应商手中。
  <br/>
  <br/>
  欧盟执行委员会副主席塞夫柯维奇(Maros Sefcovic)呼吁欧洲要“意识到严酷的现实”。他说：“我们使用的大多数金属的75％至100％来自欧盟以外的国家，中国提供了我们98％的稀土供应。”
  <br/>
  <br/>
  此外，欧盟消费的98％硼酸盐由土耳其供应，68％的钴来自刚果民主共和国，78%的锂来自智利；南非则满足欧盟71%的铂金需求。
  <br/>
  <br/>
  塞夫柯维奇指出，欧盟必须彻头彻尾改变作法。因为欧盟很大程度上仰赖环保及社会标准较低、较不自由、贫困且采取不永续经济方式的国家，提供有限的原物料。
  <br/>
  <br/>
  欧盟预计，到2050年，为了生产电动车电池及储存能源，将需要比现在多约60倍的锂、15倍的钴。因此，欧盟必须加速在原物料供应上的独立自主。塞夫柯维奇强调： " 我们必须有更多方面的供应源，更充分利用欧盟境内资源，同时也会在欧盟实施最严格的环境及社会标准。 "
  <br/>
  <br/>
 </p>

 <p>
  欧盟执委会认为，欧盟境内矿藏还没有充分开发。内部市场专员布雷顿（Thierry Breton）说："欧洲有很多资源，譬如铝，钴和锂，但是欧洲的潜力远未得到开发。"
 </p>
 <p>
  他举例由于欧洲大陆的产能不足，在欧洲开采的原料，例如锂，必须离开欧洲进行加工。而在大量投资上，延误发放许可证以及被舆论敌意的视为肮脏工业，通常使得潜在投资者裹步不前。
  <br/>
  <br/>
  欧盟提出的新战略是要成立欧洲原物料联盟，重组产业界、投资人、欧洲投资银行、各成员国等，以确保矿物原料供应链。供应多样化需要建立伙伴关系，欧盟执委会希望明年起，与加拿大或澳大利亚及有兴趣的非洲国家展开伙伴关系合作。
  <br/>
  <br/>
  欧盟还打算研究各种创新方法以重复利用、修复及回收含稀土矿物的产品，目标是在2022年之前，确认境内储存及废弃物中关键原物料的可能供应量，并找出有利于回收这些原物料的计划。

  <br/>

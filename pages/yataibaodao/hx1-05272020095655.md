 <p>
  中国全国人大将在28日绕过香港立法会通过“港版国安法”，被外界视为是提前27年走向“一国一制”。台湾的外交部长吴钊燮26日接受美国一家电视台采访时表示，北京趁国际社会忙于抗击疫情，推动港版国安法，下一步可能动武拿下台湾。
  <br/>
  <br/>
  台湾外交部长吴钊燮26日接受亲特朗普总统的美国福克斯新闻（Fox News）主播玛莎·麦卡勒姆（Martha MacCallum）视讯访问。这是半个月内第二次。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  议题环绕在北京第13届人大28日将通过“港版国安法”的效应。福克斯主播问到他对美国可能出手反制中国的看法。吴钊燮表示，尊重美国政府为挽救并保护香港自由民主而准备对中国采取的行动。
 </p>

 <p>
  <br/>
  <b>
   台外长担忧中国会对台湾动武
  </b>
  <br/>
  <br/>
  吴钊燮说：“我认为台湾也试图做相同的事。我们一直紧盯情势发展，也告诉中方‘别这么做’。我们告诉中国，他们应该履行30年前给予香港自由民主的承诺。我们会尽我们所能支持香港，以利香港维持自由繁荣。”
  <br/>
  <br/>
  吴钊燮的说法，响应了蔡英文总统日前所说“台湾同所有民主阵营的伙伴们，都和香港人民站在一起。”
  <br/>
  <br/>
  此外，吴钊燮提到，各国忙着处理疫情时，中国政府似乎在全球多处扩张势力并采取行动。除在南海、东海增加军事行动外，中国也对台施压，并要求多国不要与台湾往来，特别是西半球国家。
  <br/>
  <br/>
  吴钊燮说，中国政府明显趁大家忙得焦头烂额时出手，各界必须保持警觉，“如果香港倒下，中国政府在香港施行国安法，我们不知道下一步会怎样，可能就是台湾。我们对此非常慎重，试图保护自己。”
  <br/>
  <br/>
  被问到中国是否会在这段时间对香港或台湾采取军事行动时，吴钊燮说，中国政府曾承诺在香港实行“一国两制”、香港制度50年不变，如今却企图完全接管香港行政部门，“我们也担心中国可能对台采取军事行动”。
  <br/>
  <br/>
  吴钊燮表示，中国政府面临国内危机时，常在外部找代罪羔羊转移焦点，台湾很容易成为中国眼中的代罪羔羊。
  <br/>
  <br/>
  吴钊燮：“我们当然不希望中国认为台湾很脆弱，可用军事手段迅速攻下台湾，所以我们试图在外交层面争取美国与全球各国支持，同时试图强化防卫能力，借此吓阻中国。”
  <br/>
  <br/>
  美国总统特朗普26日在白宫被问到是否会对中国实施制裁时说：“你们在本周结束前，就会知道我们采取了什么行动 -- 我认为是非常强而有力的行动。”

  <b>
   北京制法为了“师出有名”
  </b>
  <br/>
  <br/>
  台湾朝野也关注西方国家对于港版国安法通过的因应行动。执政的民进党籍立法委员赵天麟27日质询外交部，是否掌握美、英等国对中国提出何种制裁？外交部次长徐斯俭答询说：“美国可能会取消香港的优惠关税地位，英国可能给有英国海外护照的香港居民英国公民身份，因应有可能出现港人出走的情况。但是美国如果要对香港有所制裁的话，其实对美国经济利益也会有负面影响。”
  <br/>
  <br/>
  政治大学国际关系研究中心美欧所研究员严震生接受自由亚洲电台采访则认为，香港问题去年底告一段落，新冠肺炎之后，抗议者又要重回街头，大陆要设法找到法源处理。
  <br/>
  <br/>
  严震生：“香港是大陆的一部份，台湾不是。香港只是一个特别行政区，香港人常把自己比成台湾，我们也把它当作台湾，完全是引喻失义。（北京）处理完香港和未来会不会处理台湾是两码事，而且他也不是趁什么疫情，他本来就要处理的，他只是让你香港自己处理搞了十几年都没办法处理好基本法的这个部份，那他干脆用一个，我们讲说漏洞吧，由国家机器来做，透过人大通过立法。以后处理香港就师出有名。”

  <br/>
  <b>
   外媒报导解放军模拟夺东沙岛 可能投入2航母
   <br/>
  </b>
  <br/>
  另外，军事活动从台海延伸到南海，美中动作频频。日本媒体共同社先前披露，中国解放军订于8月在南海军演，模拟夺取台湾控制的东沙岛；香港媒体《南华早报》也报道说，中国届时将派遣1艘航空母舰行经东沙岛。英国媒体《太阳报》更示警表示，解放军可能投入山东号、辽宁号2艘航母至东沙群岛附近演习，借此演习夺取东沙群岛，甚至是入侵台湾。
  <br/>
  <br/>
  <b>
   美军26日动作频频
  </b>
  <br/>
  <br/>
  而根据关注全球飞机行踪的“飞机守望”（Aircraft Spots）推特账号的军情资讯显示，美国空军2架B-1B超音速战略轰炸机与一架KC-135R空中加油机，26日自关岛安德森空军基地出发，前往台湾周边的南海空域执行任务。同一时间，美国太平洋空军也在脸书公布B-1B超音速战略轰炸机执行任务的照片，指称来自戴斯空军基地的2架B-1B枪骑兵轰炸机在5月26日于南中国海执行任务，这是轰炸机特遣队任务的一部分，展示了美军在国际法允许的范围内飞行和操作的能力。文中强调，这是美国国防部自由选择的时间和节奏。
  <br/>
  <br/>
  此外，另一张曝光的美国空军航行路线图显示，美军KC-135加油机26日也一度飞近香港，是否和近期北京强推“港版国安法”有关，引发各界关注。
  <br/>
  <br/>
  台湾国防部则公布，将在2023年完成建构全台雄风反舰飞弹机动发射车91辆。外界认为这是国军对解放军的反制措施，对中国航母及舰队加强吓阻力。国军依据实际战场需求，利用其机动性提高存活率，及增加联合制海战术运用的弹性。

 <p>
  两年前被判刑13年的中国民间组织“中国人权观察“理事长秦永敏，健康状况一直备受关注。家属被狱方拒绝探望之际，近期收到相信是他的亲笔信。耐人寻味的是，这封弥足珍贵的信花了大半年才寄到家属手上。
  <br/>
  <br/>
  两年前被判囚13年，目前在湖北潜江监狱服刑的秦永敏再度传出音讯。与他相熟并要求匿名的一名人士透露，妻子赵素利接到他写给家里的信。
  <br/>
  <br/>
  知情人士：“秦先生给赵素利的信主要是对赵素利的思念和牵挂，还有担心她的身体，叫赵素利不要给他账上打钱了，要她照顾好自己的身体。他说，赵素利你是我唯一的亲人，所以说你要照顾好自己的身体，等到我们再相聚的时候。能感觉到他对赵素利的思念和牵挂，我真的很感动。赵素利的身体也不好，并且24小时遭监控。赵素利身边也没有亲人，她也没有能力抗争，她也不敢发声。”
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  秦永敏因”颠覆国家政权罪“入狱以来，健康状况备受关注。三哥秦永昶表示，弟弟寄给他的只是家书，没有披露他在狱中的情况。
  <br/>
  <br/>
  秦永昶：“没有没有。 这信纯粹是家事，没有其他的事情。写那些事情这信就寄不出来，就不会寄给我们。现在写家里的事情它都拖了半年多才给我们。如果写那些东西信就发不出来了。“
 </p>

 <p>
  <br/>
  耐人寻味的是，从署名日期12月27日开始计算，这封信花了超过半年才送到家属手上。
  <br/>
  <br/>
  秦永昶：”莫名奇妙。一封信怎么走了半年呢？以往也是很长时间，但是没有那么长时间。你问我，我怎么知道。我也不知道为什么寄不出来，但是可以想象得到，它愿意给你发信就给你发信，它不愿意发信就不给你发信。信是要经过审查的。肯定不高兴，但是有什么办法呢？“
  <br/>
  <br/>
  家属上次与秦永敏会面是今年1月，也就是说，现在距离上次会面已超过半年。
  <br/>
  <br/>
  秦永昶： “它们说是疫情的原因所以不让会见，没有别的，是不是真实理由我不知道。肯定是每个月都有提（会面）要求，但又有什么用呢？我们只不过是要求去见。它告诉你不能见，你就没有办法了。”
  <br/>
  <br/>
  66岁的秦永敏患有高血压。上次会面，他告诉家人自己的牙齿脱落了几颗。
  <br/>
  <br/>
  秦永昶：“状况还可以，但是他长期都是高血压，发的时候问题就比较严重，要是不发情况还好，但是他一发有时候蛮高的。监狱里面的条件我们看不出来，因为也不允许我们谈监狱里面的事情。高血压如果得不到及时治疗肯定是有危险的。“
  <br/>
  <br/>
  秦永敏的朋友说，疫情不应成为监狱阻止会面的藉口。
  <br/>
  <br/>
  知情人士：”湖北疫情早就结束了，各种隔离也已经撤销了，所以说，应该恢复正常的会见。各地的审讯早已公开开庭了，为什么潜江监狱还一直以疫情为由，拒绝要求秦永敏会见？ 这是当局有意阻挠他们见面，也剥夺他们见面的权利，这当然应该受到强烈谴责。”
  <br/>
  <br/>
  秦永敏长年关注中国人权状况，曾多次被捕，合共入狱超过20年，当中在1998年因颠覆国家政权罪成，坐牢12年。

 <p>

 <p>
  台湾的蔡英文、赖清德20号宣誓就职，已有47国、263名政要表达祝贺。除了首次有美国国务卿蓬佩奥书面表达祝贺，白宫副国安顾问博明、美亚太助卿史达伟在祝贺时称“蔡总统”，前所未见。对此，中国外交部、国防部、国台办三个部委同声谴责。
 </p>
 <p>
  受新冠肺炎疫情影响，蔡英文第二任就职典礼仅邀请各国驻台代表。14个友邦及19个理念相近及友好国家共65位政要以预录致贺视频，在台北宾馆典礼会场及总统府发言人脸书播放。台湾外交部透露，已有47国或地区，263名政要以发表声明、致函、推文及拍摄视频等方式祝贺台湾总统就职。
 </p>
 <p>
 </p>
 <p>
  <br/>
  美国国务卿蓬佩奥（Mike Pompeo）发推特和书面贺信，称许“蔡总统带领活力民主台湾的勇气和愿景，堪称印太地区及全世界的表率。”美国副国安顾问博明、美国亚太助卿史达伟录制视频，并公开称呼“蔡总统”，都创下台美关系的“第一次”。
  <br/>
  <br/>
  美国在台协会脸书表示，由于疫情缘故，美国派了一个“虚拟代表团”参加台湾的总统就职典礼。接下来每一天，AIT将跟分享美国代表团祝贺视频。
 </p>

 </p>

  <b>
   博明：台湾向世界表明自由民主的精神是普世价值
   <br/>
  </b>
  <br/>
  AIT第一支视频就是有“中国通”之称的白宫副国安顾问博明(Matthew Pottinger)，他在2分半视频中，全程以流利中文表达祝贺：“今天我非常高兴有机会祝贺蔡英文总统在选举中取得历史性胜利，在台湾历史上，她所得票数超过所有台湾的政治候选人，我更要赞扬台湾人民对来之不易的民主的维护。”
  <br/>
  <br/>
  博明还说：“你们依靠的正是台湾的言论自由、法律的统治、和自由公正的选择，美国也依靠着同样的机制。台湾向世界表明，自由民主的精神，绝不仅仅是美国或西方的，而是普世的。”
  <br/>
  <br/>
  博明称许在武汉肺炎大爆发之前，台湾就应用从SARS流行病得到的教训和累积的经验，台湾的学校仅关闭2周，在2300万人口中，台湾至今病例少于五百例，其中大多数是输入病例。台湾有很多方面值得学习，美国将继续与台湾交流，将继续敦促像世界卫生等其他国际组织等，挽救人的生命置于政治考量之上，选择人民的自由而非政府的压制。期待与台湾开展经济、文化、教育、安全等方面的合作，通过合作体现台美的共同利益和价值观。

  其他录制视频的美国政要还包括，前司法部长艾许克罗（John Ashcroft）、国防部印太事务前助理部长薛瑞福（ Randall Schriver）等。另有联邦众议院议长佩罗西（Nancy Pelosi）、联邦参议院代议长葛拉斯里（Chuck Grassley）及多名美国参众议员，透过致函、推文、影片等方式致贺。
  <br/>
  <br/>
  台湾15友邦中，梵蒂冈教宗方济各（Pope Francis）发贺电，尼加拉瓜由副总统穆丽优录音祝贺，巴拉圭由副总统韦拉斯格斯以影片方式祝贺。其他友邦史瓦帝尼、危地马拉、海地、洪都拉斯、马绍尔、瑙鲁、帕劳、贝里斯、圣克里斯多福、圣露西亚、圣文森、吐瓦鲁均由国家元首以影片方式祝贺。
  <br/>

  视频中，贝里斯总理巴洛表示：“蔡总统阁下才德兼备，即便面对政治挑战，希望且深信她终将凯旋得胜。”
  <br/>

  非友邦的友台国家包括日本、斐济、印度、韩国、新加坡、加拿大等由国会议员或前行政官员祝贺，欧洲多国政要，国会议员、友台小组主席、前任官员等祝贺。

   前驻美代表：中美紧张助台美发展  当“树欲静而风不止”台湾要有智慧自保
  </b>
  <br/>
  <br/>
  美国首次有国务卿公开对台湾总统就职表达祝贺，也是首次有白宫副国安顾问录制视频公开称“蔡总统”。台湾前驻美代表程建人说，以前就职没有碰到疫情，这样的作法确实特别，其中有多位是长期反中人士，拜登也搭上美国反中的快车。
  <br/>
  <br/>
  不过程建人说：“从过去一段时间看来，不是台美之间的问题，而是中美之间的问题。美中从贸易之后，又是科技、疫情问题，双方弄的非常不愉快。这样的情形，给台美关系发展的空间。”
  <br/>
  <br/>
  程建人提到，以前美国总统和台湾总统之间，互相的信件往来都是有的，自两蒋时代就有，只是没有马上公开。
  <br/>
  <br/>
  中国大陆党媒《环球时报》20日则以《罕见！外交部、国台办、国防部同日表态！》为题报导，中国外交部指出，美国政府一些官员和美国一些政客向蔡英文发视频贺辞，严重违反“一个中国”原则和中美三个联合公报规定，严重干涉中国内政。中方对此表示强烈愤慨，并予以谴责。

  <b>
   台绿委：中国一次骂47国很厉害
  </b>
  <br/>
  <br/>
  《环球时报》的报导还称，中国国防部则重申，“台湾问题是中国内政，事关中国核心利益和14亿中国人民民族感情，不容任何外来干涉。我们坚决反对任何国家与台湾开展任何形式的官方往来和军事联系。我们绝不允许任何人、任何组织、任何政党、在任何时候、以任何形式、把任何一块中国领土从中国分裂出去！”
  <br/>
  <br/>
  对此，台湾执政的民进党籍立法委员范云接受自由亚洲电台采访反问：“中国可以一次骂47个国家，实在太厉害了。我想他骂的内容数十年来没有什么改变，只是把台湾愈推愈远吧！中国内部也有各种危机，到底他还能怎么围堵台湾，全世界继续在看。”
  <br/>
  <br/>
  范云认为，疫情改变了全世界看待边境、看待国际局势的方式，特别对中国的态度有很大的改变，发现站在中国的利益上，反而自己国家的健康受伤害。
  <br/>
  <br/>
  范云：“这是一个很大的反省，过去大家以为可以跟中国个别打交道，只要处理经济、投资，可以不用管人权跟民主，现在发现，这反而是逃不了，因为他的独裁给大家带来连动的灾难，事实上大家的命运都连结在一起，你不可能说，你只要处理经济，不要处理别的问题。”
  <br/>
  <br/>
  范云说，台湾必须站稳自己的立场，后疫情时代，用价值交朋友是最稳健的方式，分享同样民主多元的价值，世界才会真的看到台湾对他们的意义，才会选择跟台湾站在一起。

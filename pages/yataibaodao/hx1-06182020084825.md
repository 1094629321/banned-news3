  新冠肺炎疫情造成全球800万人感染，台湾至今445人确诊，434人已解除隔离。台大医院救活了3名呼吸衰竭的重症病人，最奇迹的是A先生曾心脏停止跳动，以人工心脏代替33天。在医院为他举行的出院庆生会上，他感谢台大医护英雄，将他从鬼门关抢救回来。

  </b>

  <br/>
  <br/>
  古世基医师提到，台大医院启动多团队的照顾，除了感染状况、免疫、内科、营养、复健、护理团队等介入帮助，且在暴露感染的高风险下，组成“拍痰团队”协助病友拍痰、翻身等，连主治医师都协助拍痰。全身防护的医护人员，则以在防护衣上写名字，互相辨识身分。

 <p>
  A先生在住院51天后才三次采检新冠肺炎阴性，进而解除隔离。台大医院这位重症康复具有指标意义，令医护团队振奋。古世基：“他整个叶克膜用的时间，大概全台湾最久，33天，37天脱离呼吸器，加护病房住院47天，总住院到今天为止75天。”
  <br/>
  <br/>
  <b>
   75天新冠肺炎住院治疗费 国家健保埋单
  </b>
  <br/>
  <br/>
  外界关心叶克膜使用长达一个多月会否有后遗症，台大医生表示，叶克膜装置后，最重要因为血液抽出来每分钟3千至5千CC，机器运转打回，一端从脚或脖子，一端放在心脏，管路长，任一微小感染可能直接进入血液造成感染，因此比较要担心的是摆置叶克膜的过程有无并发症出现，叶克膜拆除后没有太多后遗症。

  <br/>
  <br/>
  <b>
   北京疫情三文鱼惹祸？张上淳：科学证据不足
  </b>
  <br/>
  <br/>
  对于北京最近又出现第二波新冠疫情，官方指向与市场里的三文鱼有关。对此，台湾疫情指挥中心的专家张上淳认为：“他们缺少更多的科学根据。你只在一个砧板上面（检测到病毒）。你如果认为鲑鱼（三文鱼）有受到原产国的污染的话，那就要赶快去验更多的鲑鱼里面有没有？如果没办法证实在其他鲑鱼身上找到，那你怎么说他是输入的呢？”

 <p>
  至于如何评估北京这波疫情？张上淳说，目前看到的推测可能五月底、六月初就开始有，比起武汉肺炎初期在武汉发生时，不去处理、不去隔离、不去承认有，现在当然不会像武汉这么严重。
  <br/>
  <br/>
  至于有说法称夏天疫情可能趋缓。张上淳说，这是全新的病毒，大多数人都没有抵抗力。只要有人带病毒，经密切接触，就可能传给周边的人，大家都没有抵抗力就会再爆发。

  “索马里兰驻台湾代表处”9日在台北揭牌，索国总统阿布迪透过视频表示，台索两国都是民主阵营，此关系立基于共同价值、对政治与经济自由的信仰，将完整推动发展双边关系。面对北京抗议，索马里兰驻台代表穆姆德强调，索马里兰是主权独立国家，有权利决定谁交往。
  <br/>
  <br/>
  位于“非洲之角”的索马里兰（Somaliland台湾译“索马利兰”）共和国驻台代表处（The Republic of Somaliland Representative Office in Taiwan）9日在台北揭牌。会场陈列两国国旗，播放台湾和索马里兰国歌，全体肃立，场面隆重盛大。

  <br/>
  由于索马里兰不被国际承认，台索进一步发展关系，即便不是建交，只是成立驻台代表处，都深受瞩目，国内外媒体塞爆会场。
  <br/>
  <br/>
  索马里兰驻台代表穆姆德首先致词感谢愿意与索马里兰在贸易、安全和社会发展等各领域建立关系的贵宾，共同见证索台关系的里程碑，和历史性的揭牌仪式。

  <br/>
  <br/>
  穆姆德提及索国正大力推广、欢迎国际投资，包括在农业、旅游和矿产等领域的潜能。

  <b>
   北京反对建立关系？ 索马里兰代表：经济合作对中国没有威胁
  </b>
  <br/>
  <br/>
  台湾驻索马里兰共和国代表处已于8月17日正式开馆，双边并签署技术合作协定。索马里兰位于索马里境内北部一片地区，人口约350万，面积约17万平方公里，相当于160个香港，或5个台湾，具有战略位置，紧邻中国大陆设立的首座海外军事基地吉布地。
  <br/>
  <br/>
  尽管中国大陆与索马里（Somalia，台湾译“索马利亚）”有正式邦交，外交上并不承认从索马里亚独立出去的索马里兰，但是对于台湾和索马里兰互设代表处，仍公开表达“坚决反对”。索马里兰驻台代表穆姆德9日被问到此事时表示，“索马里兰是一个独立的国家，不管我们想跟谁建立关系，索马里兰是独立的，从索马里兰的观点，我们是独立的，我们很高兴可以跟台湾，还有其他国家建立关系，开展经济合作关系。”

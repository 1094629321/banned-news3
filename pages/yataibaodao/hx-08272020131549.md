 <p>
  台湾27日发布首次大规模新冠肺炎抗体检测成果显示，针对高峰期的高风险族群近5千人检测，仅4人报告为阳性，感染率为万分之8。研究团队认为，这证明台湾落实对入境者隔离检疫政策，成功阻止病毒传播到社区。
  <br/>
  <br/>
  彰化县政府和台湾大学公共卫生学院27日联合发布COVID-19（2019冠状病毒疾病，俗称武汉肺炎）血清抗体检测研究报告，目标为万人检测，先公布今年2、3月间近5千名高风险者的报告，结果曾受感染的阳性率为万分之8，以此推断台湾的社区是安全的。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  彰化县卫生局长叶彦伯说，研究发现，确诊个案18人经过好几个月之后，所有个案都产生具保护力的中和抗体：“我们做血清抗体的结果，所有的个案都产生了具保护力的中和抗体，这非常重要，这对我们疫苗研发来讲，是非常正面的讯息。尤其是在间隔将近100天之后，还有中和抗体保护效果，是一个好消息。”
 </p>
 <p>
 </p>
 <p>
  <iframe frameborder="0" height="620" scrolling="no" src="https://www.facebook.com/plugins/video.php?href=https%3A%2F%2Fwww.facebook.com%2FRFAChinese%2Fvideos%2F623551924965500%2F&amp;show_text=0&amp;width=622" width="622">

 </p>

  </i>
 </p>
 <p>
  <br/>
  <br/>
  这项研究是在今年2到3月全世界开始大流行，也是台湾疫情因为大量境外输入病例，疫情进入高峰期的时候，针对彰化最有可能感染的人群，包括确诊个案的接触者、入境居家检疫者，和最容易接触到感染者的医护人员、卫生局（所）防疫工作人员、警察消防人员做检测， 受检者总共4841人。
  <br/>
  <br/>
  叶彦伯说：“其中只有4位产生了中和抗体，也就是说，我们确定他有感染，又产生有中和抗体保护力的人数就这4人。所以阳性率，是万分之8点3。在一个最容易感染的族群都已经这么低了，一般民众放心，社区是安全的，也不需要再对一般民众做抗体的检测。”

  <br/>

   台湾首次大规模血清抗体调查 各界拭目以待
  </b>
  <br/>
  <br/>
  叶彦伯强调，这是国内第一次用这个科学性、血清抗体的调查，来证明彰化很安全，同时也证明台湾很安全。因为彰化是第一个台湾本土死亡个案发生的地方。
 </p>
 <p>
  协助对彰化近5千多例检测，进行数据分析的台大公卫学院教授陈秀熙指出：“从境外跟本土阳性率的比较，我们可以很清楚知道，我们防疫成效可以达到 73%，证明居家隔离在第一波接触者传染之后，就阻断了它往下变成社区流行的最主要的机会。”

  <b>
   研究送6位同行审查
  </b>
  <br/>
  <br/>
  台大公卫学院教授詹长权指出，研究结果已送6位同行审查，目前收到4位回复，他举例其中之一：“这个就是美国加州大学柏克莱分校公共卫生学院吕淳祺院长 (Michael Lu)，在会议开始前半个多小时寄给我，他用一个客观第三者的角度跟我们讲说，我们台湾那么低的、万分之8的阳性率，是台湾的成就。有这么坚实的科学证据，要让世界知道，因为很多人都说台湾做的好，但没有数据，彰化提供了数据。”
 </p>
 <p>
  研究团队提到，原本进行癌症研究，新冠肺炎疫情爆发后应政府的要求，在二月初与教育部和科技部合作研究。彰化在两、三个月采检五千案例，一般两年都做不到这个采检数量。

 <p>
  不过，该项被称为“万人血清抗体检验计划”爆出诸多违反研究伦理的争议，包括提供民众的说明书中，并未写明是“研究计划”。
  <br/>
  <br/>
  <b>
   彰化县卫生局做法与中央不同 遭质疑分庭抗礼
  </b>
  <br/>
  <br/>
  此外，彰化县卫生局自行推动所谓“精准筛检”，主动采检无症状居家检疫者的做法，因与中央指挥中心规定不符合，日前遭指挥官陈时中宣称要移送政风调查。
  <br/>
  <br/>
  “万人血清抗体检验计划”原订25日公布结果却临时喊卡。陈时中要求研究团队讲清楚为何拖延，不要让外界误以为遭施压和盖牌。
 </p>

 <p>
  由于彰化做法倾向普筛，与中央做法路子不同，在台湾社会也引发辩论。许多人一直挑战指挥中心“不普筛”的思路，甚至有言论认为台湾不普筛是为了隐匿疫情。因此有人解读彰化的做法是在跟指挥中心唱反调。一度引发双方关系紧张。不过双方似乎达到共识，不再相互喊话并强调要团结。
  <br/>
  <br/>
  研究团队迟了两天发布研究结果，中央流行疫情指挥中心发言人庄人祥受邀出席新闻发布会。庄人祥会后接受自由亚洲电台采访指出，“研究者总算排除万难把它公布了，厘清不是指挥中心这边施压或盖牌。”
  <br/>
  <br/>
  庄人祥还说，“因为这疾病有将近50%机率，是因为无症状或症状前感染者所造成的，所以我们居家检疫隔离措施，是避免个案散播到社区去，这个研究也可以稍微回答中央过去这个政策是对的。”
 </p>

 <p>
  加拿大奥委会（COC）和加拿大残奥委员会（CPC）开了全球第一枪，他们发声明表示，如果奥运不延迟，加拿大不会派遣运动员参加七月举行的东京奥运会。加拿大政府和体育界人士均支持这项决定。
  <br/>
  <br/>
  刚过的星期天，国际奥委会和日本首相安倍晋三首度松口表示将考虑重新安排本届奥运，未来4个星期内会有所决定。但加拿大奥委会和残奥会率先发出声明称：「除非奥运延期，否则2020年夏天不会派员参加东京奥运。」它们呼吁国际奥委会推迟奥运一年，如此可以确保运动员和公众的安全。
  <br/>
  <br/>
  周一加拿大总理特鲁多在记者会上支持加拿大奥委会的决定。  “我知道这对很多人来说都是心碎的，运动员丶教练丶工作人员和体育迷来说都是难过的，但这是正确的决定，只要遵循着保护自己和其他人的原则，就是在拯救生命。”
  <br/>
  <br/>
  加拿大代表团官方在推特上随後也陆续发出帖文说：“远比表现丶成绩和奖牌更重要”“今日先延期，明日再征服（Postpone today. Conquer tomorrow）。
  <br/>
  <br/>
  加拿大女子奥运足球代表队队员黛安娜·玛斯森(Diana Matheson)说，身为加拿大代表团之一，很高兴看到加拿大奥委会对国际奥委会施压。  “我相信国际奥委会会做出延期的决定，所以我们还是很有机会可以参加这个难得的体育盛事，就算我们要再等上一年才能比赛，这也是该做的事情，很高兴加拿大奥委会率先做出这个呼吁。”
  <br/>
  <br/>
  加拿大的疫情危机不断升级，总理特鲁多再次呼吁国民尽量不要出门，如果一定要出门，一定要保持社交距离，与他人间隔至少两米。  “我们看到一些照片显示人们聚集在一起，认为自己是无敌的，但你不是。实在令人受够了。你应该回家丶待在家里。我们不断教育国民做好自己本分，但若再不听劝，不排除处以罚款或逮捕。”
  <br/>
  <br/>
  除了给予企业劳工经济援助外，加拿大联邦政府也宣布投资1.92亿元帮助研究生产新冠状病毒疫苗，正积极扩大检测能力，全力防堵疫情恶化。
  <br/>
  <br/>
  总理特鲁多也首度在记者会上提到会参考中国和台湾的防疫经验。驻加拿大台北经济文化代表处陈文仪则说，台湾防疫成果很愿意与加拿大分享，加拿大和台湾可以尽快展开具体双边公卫医疗合作，并感谢杜鲁多总理之前公开支持台湾加入世界卫生组织。
  <br/>
  <br/>
  这几天许多加拿大媒体均报导台湾防疫的成功经验。例如：加拿大国家广播电视公司报导，台湾人民生活一切正常，但加拿大各级学校及商店却关门了，相较来说，台湾遭受疫情的冲击远比加拿大轻。报导中抛出问题称：加拿大人锁在家中的情况会不会成为新常态？抑或，加拿大未来可以像一万公里外的台湾？
  <br/>
  <br/>
  在台湾经营一家医院丶目前人在加拿大的陈惠薰表示，总归一句，加拿大反应太慢。 “加拿大还没有什麽疫情的时候就应该要开始预防，台湾就是做得很早。台湾很早就不让武汉人进入，进来一定要量体温，加拿大有吗？根本没有，进来的人都趴趴走，可能都不知道自己已经得病。”
  <br/>
  <br/>
  陈惠薰也说，加拿大医疗体系隶属各省政府管辖，联邦政府难以统一调配，这和台湾中央政府统一执行的方式也不一样，恐怕效率上难以和台湾相比。
  <br/>
  <br/>
  自由亚洲电台记者   柳飞

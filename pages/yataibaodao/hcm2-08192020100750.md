 <p>
  台湾彰化卫生局主动筛检出，一名自美国返台少年在居家检疫第10天确诊新冠肺炎。由于不符合指挥中心筛检作业程序，指挥中心下令相关单位介入调查。此举在台湾舆论引起热议。
  <br/>
  <br/>
  台湾确诊病例(案485)是名长期居住美国的少年，8月5日独自返台探亲，一直没有疑似症状，8月15日，也就是少年返台隔离的第10天，被彰化卫生单位采检后确诊。
  <br/>
  <br/>
  彰化卫生局及时找出无症状病例，却意外成为指挥中心要调查的对象。中央流行疫情指挥中心发言人庄人祥(18日)第一时间表示，“指挥官陈时中已经下令，请政风单位针对卫生局正式调查。”主要厘清两点，一是中央政策是没有症状不用采检，为何地方卫生局仍要采检？其次是要掌握还有多少人也接受采检？
 </p>
 <p>
 </p>
 <p>
  <br/>
  此举引发台湾舆论哗然。台大公卫学院院长詹长权在个人脸书发文，“看得懂世界卫生组织要传达什么讯息，对全球疫情很担心。看不懂政府为什么这么做？台湾怎么了？很难过！”
  <br/>
  <br/>
  国健署前署长邱淑媞在个人脸书提出，“政风单位不是专办贪污索贿或喝花酒吗？卫生防疫检验让病毒现形，这是什么大罪？”
  <br/>
  <br/>
  <b>
   政风处专办腐败案  调查“确诊局长”遭非议
  </b>
  <br/>
  <br/>
  拥有美国哈佛大学公共卫生学院科学博士的国民党前立委陈宜民接受本台访问时指出，中央防疫需要地方协助，有时对于一个超期部署的地方官员，身为指挥官，应该把地方卫生局长当作作战部队最倚重的长官，怎么会做这样的事。
  <br/>
  <br/>
  陈宜民：“今天更不要说，有些县市他们要超前部署，过去都有先例可循，这没有好去(挑惕)没有症状、没有遵守(SOP)。我觉得这件事情小题大作是绿色恐怖。”
 </p>

 <p>
  <b>
   彰化卫生局长：没有规定不能采检无症状患者
  </b>
  <br/>
  <br/>
  遭调查的彰化县卫生局长叶彦伯下午打破沉默受访时指出，居家检疫者不管有没有症状，若要就医采检都会依照就医采检的程序，确保到医院做采检。卫生局全程掌控，不会有问题。叶彦伯也坦承，“这是正常标准作业程序，不需要向中央报告。”
  <br/>
  <br/>
  彰化卫生局从4月起就开始“精准筛检”。台媒《联合报》披露，叶彦伯曾说过，“没有一条规定对于无症状者不能采检，无症状让人担心的就是风险。”
  <br/>
  <br/>
  <b>
   陈时中做出回应
  </b>
  <br/>
  <br/>
  下令政风处调查的卫福部长陈时中19日在防疫记者会上亲上火线澄清。陈时中回应指出，指挥中心第一时间看到此案，第一个疑问就是在居家检疫规定中，有状况、生病时不能任意外出，怎会到医院或是医师到家里检验呢？是根据什么检验？“有生病或是有症状时，可以联络卫生局就医。几个案例认为自己有症状，自行跑去就医，违反我们居家检疫的规定，我们曾经开罚过。”
  <br/>
  <br/>
  对于外界高度质疑为何是由政风单位出面调查，陈时中解释，政风单位在各政府部门中是针对廉政或宣导辅导订立政策的单位，具有法律背景，跟各部门的单位也不相同，是另外一条鞭。
  <br/>
  <br/>
  陈时中：“现在大家对于无症状感染者有很多意见，过去更有指挥中心对于零确诊或无症状感染者的‘吃案’说。为了避免引发后续，因此也希望透过内部调查，让整体状况透明清楚，透过第三方、相对公正的方式希望能厘清事实。”
  <br/>
  <br/>
  陈时中强调，对于卫生局，不是当作犯罪行为去查，而是搞清楚事实，且不是由卫福部自己人来查，而是希望经由独立单位让事实更清晰，然后根据此结果做讨论或检讨。
  <br/>
  <br/>
  <br/>
  自由亚洲电台记者黄春梅 台北报道  责编 胡力汉 申铧  网编 瑞哲

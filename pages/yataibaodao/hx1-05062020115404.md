 <p>
  五月下旬，台海两岸都有重要的政治大事发生。受新冠肺炎疫情影响，中国大陆的“两会”和台湾蔡英文总统第二任就职，都走精简路线。与此同时，美军轰炸机队和加油机6日再度现踪台湾东北海域。美国航母是否近期会在亚太地区巡戈，受到台湾朝野关注。
  <br/>
  <br/>
  大陆的两会和台湾蔡英文就职第二个任期在台海两岸都是具指标意义的事件。由于美军最近频繁在台海附近出没，台湾立法委员王定宇很关注美国航母是否会在近期来到台湾附近。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  台湾立法委员王定宇：“能说就说，不能说我们就谢谢再见。美军航母尼米兹号是不是在520前后会来到台海周边或者南海？”
 </p>
 <p>
  国防部长严德发：“我们所了解的，他们在海上的兵力都在做一些积极的工作。”
  <br/>
  <br/>
  王定宇：“尼米兹号已经出港了，也做了一些测试性的演习，他是不是往亚洲来了？那根据他的航速，是不是在520前，美军航母尼米兹号会来到台海周边？”
  <br/>
  <br/>
  严德发：“这个是美方他们部署的调整，我们在做观察。”
 </p>

  <b>
   520前夕 美轰炸机队四度巡弋台湾外围
  </b>
  <br/>
  <br/>
  蔡英文第二任就职典礼5月20日举行，中国大陆政协、人大两会则紧接在5月21日、22日召开，民进党籍立法委员王定宇6日上午质询国防部官员，关注美军战略轰炸机队，最近在4月22日、5月2日、5月4日，三度接近台湾外围空域。
  <br/>
  <br/>
  立委王定宇： “飞到我们的东北角、飞到冲绳、飞到宫古海峡，美军这样的飞行，还有这样飞行的长度，这三趟，还有加油机，他有没有什么战术意义在里面？”
  <br/>
  <br/>
  空军参谋长郑荣丰说，“美军的活动，我们是不能作评论的，但是我们判断它是动态兵力运作的模式，它是有战略的意义。”
  <br/>
  <br/>
  郑荣丰还说，美军电侦机在南海与台海附近绕行，主要是该区域有军事活动，水上与水下，台湾也会侦巡。

 <p>
  立委质询才结束，美军驻关岛2架B-1B超音速战略轰炸机、KC-135加油机6日飞向中国东海执行任务，掠过台湾东北海域。据推特上专门关注飞机行踪的账号“飞机守望”公开披露的讯息，这是美军近半个月来第四度有轰炸机编队接近台湾外围空域。
  <br/>
  <br/>
  对此，国防部发言人史顺文少将指出，“台海周边海空域情况，国军都全程掌握。”
  <br/>
  <br/>
  <b>
   蔡英文就职典礼不邀外宾  友邦政要视频祝贺
  </b>
  <br/>
  <br/>
  蔡英文五二零第二任就职，受到新冠肺炎疫情的影响，决定不同于第一任就职时，在总统府前广场搭舞台，广邀外宾观赏户外大型表演活动，今年决定不邀请外宾访问台湾，只邀请各国驻台使节出席。
  <br/>
  <br/>
  总统府发言人丁允恭6日接受自由亚洲电台采访指出：“这一次总统、副总统就职典礼，基于防疫工作的考量，我们不会像过往模式举办大型群众活动。我们预计会在总统府大礼堂宣誓就职后，总统会在台北宾馆进行就职演说。”
 </p>

  丁允恭还说：“就职典礼的部份，也没有邀请外宾来台，友邦跟友好国家政要，以拍摄影片方式祝贺。相关规划会依照中央流行疫情指挥中心指引，简单隆重的办理。”
  <br/>
  <br/>
  与此同时，中国5月21日至27日的政协会议，以及5月22日到28日召开的全国人大会议，也被外界称为“史上最短”的两会。
  <br/>
  <br/>
  港媒报导，港区全国人大代表5日在中联办开例会，港区人大代表团团长马逢国会后表示，在北京期间将采取封闭式管理，除了入住的北京饭店和召开大会的人民大会堂外，不能随便进出和接受采访，会议结束后集体回香港。
  <br/>
  <br/>
  <b>
   大陆各地两会加强防疫
  </b>
  <br/>
  <br/>
  此外，疫情之下，大陆各省市的两会，多采用主会场搭配分会场视频连线的方式召开，并缩短会期，避免不同会场之间的人员流动。会场防疫措施包括，成立疫情防控小组，进入会场前必先量体温、戴口罩、喷消毒液、隔开就座、分开用餐，主动申报健康情况、活动轨迹，甚至核酸检测，并签订健康承诺书等。部份则采行线上会议。部份省份要求出席会议的人士，会前14天不可聚餐、旅行，即自主健康管理，有些地方则实行会场、酒店两点一线管理，避免不必要的外出和聚集活动。

 <p>
  <b>
   学者：外界关系蔡英文讲什么  对岸如何回应
  </b>
  <br/>
  <br/>
  对于两岸的政治大事一前一后，政治大学经济学系教授林祖嘉6日接受自由亚洲电台采访说：“大家都很关心蔡总统讲什么，对方回应是什么，当然很重要，但是时间会不会摆的这么巧合，真的很难说，因为主要是疫情，所以时间往后拉，那大陆那么多人要找个时间开两会不容易，所以是不是故意弄在这个时间真的很难说，不知道为什么。”
  <br/>
  <br/>
  林祖嘉认为，两岸领导人谈话重点，很可能都会摆在疫情后，如何补贴纾困和协助产业转型的问题。
  <br/>
  <br/>
  因为疫情都朝向精简方式进行，林祖嘉说：“疫情不可能来，一定是这样，大家网路上解决，简单一点，双方少讲一点刺激彼此的话，也许好一点。”

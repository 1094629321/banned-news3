 <p>
  美国国防部长埃斯珀与中国国防部长魏凤和6日通电话，根据美方释出的新闻稿指，美方关切中方在南海与台湾问题破坏稳定表达关切；而中方透过新华社发出的新闻稿指，美方对中方“污名化”等问题表达中方原则立场，要求美方要停止错误言行。
  <br/>
  <br/>
  根据美国国防部的声明指出，美国国防部长埃斯珀(Mark T. Esper)呼吁中国在新冠疫情问题上提高透明度，并且对中国在台湾和南海附近破坏稳定的活动表达关切。美方并呼吁中国履行1984年《中英联合声明》原则下的国际义务。埃斯珀表达中国遵守国际法、规则和规范，并履行其国际承诺的重要性。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  声明稿指，埃斯珀强调他对美中防务关系的优先顺序，重点将放在预防和管理危机、降低风险以及在两国共同利益的领域开展合作。
 </p>

 </p>

 <p>
  两位部长一致认为，保持开放的沟通渠道和发展危机沟通和降低风险所需系统的重要性。美国和中国之间建立建设性、稳定和以结果为导向的防务关系的原则。
  <br/>
  <br/>
  美国国防部发言人霍夫曼（Jonathan Hoffman）受访时提出，埃斯珀仍然希望今年年底与魏凤和举行会晤。埃斯珀上个月也曾提出访问中国的意愿。
  <br/>
  <br/>
  至于中方透过官媒发布的声明表示，魏凤和就南海、台湾、美方对中方“污名化”等问题表达中方原则立场，要求美方要停止错误言行，加强海上风险管控，避免采取可能使局势升温的危险举动，维护地区和平稳定。埃斯珀说，在两国关系紧张时，两军要保持对话磋商，管控危机，防止误判，降低风险。
  <br/>
  <br/>
  <b>
   中国释出“免战牌” 美外交与国防战略节奏一致
  </b>
  <br/>
  <br/>
  台湾国防院副研究员苏紫云解读美中两国防长的通话认为，应该要回溯中国外交部长王毅日前接受新华社专访所强调的：“要避免对抗，畅通渠道，坦诚对话，拒绝脱钩，保持合作。”外界多认为这是“战狼外交政策调整”。
 </p>
 <p>
  苏紫云认为，王毅的谈话很明显是对美国发出“免战牌”，希望跟美国保持经济上的挂钩，希望跟美国经贸上的合作，双方有坐下来可以沟通的机会。“两国国防部长的谈话，很清楚地相对于中国外长王毅的谈话，魏凤和的谈话硬中带软，他还是想甩锅给美国，指出美国在印太区域这里的活动，才是所谓区域不稳定的主因，但是，这是倒果为因的说法。”
 </p>
 <p>
 </p>

 <p>
  苏紫云说这其中衍生出的另一个观察，特别以北京大学“南海战略态势感知计划”而言，可以显示中共在采取外交或安全行动前铺了许多哏。从今年五月开始“南海战略态势感知计划”，单方面释出美军活动的讯息，营造美军是在南海不稳定的因素。
  <br/>
  <br/>
  苏紫云：“把南海不稳定的紧张态势，企图嫁祸给美军，塑造美军为 ‘麻烦制造者’的角色。接着3个月左右时间，都被中国的外长跟防长间接地用来指控美军一个依据，这是整套的大外宣。可惜从国际舆论跟各国的反应看来，中共的甩锅做法或是战略洗脑的方式应该是失败的。”
  <br/>
  <br/>
  中国外交部发言人汪文斌7月28日在例行记者会上就曾经引用相关数据指称，“今年上半年，美军机在南海活动多达2000多次。”
  <br/>
  <br/>
  <b>
   苏紫云：印太战场主赖科技战非地面战 美打破“惧战”迷思
  </b>
  <br/>
  <br/>
  对于美国日前主动公布，从加州向太平洋海域试射一枚未安装核弹头的洲际弹道导弹。美国国防部长埃斯珀明确指出，美国该与中国战斗时不会畏战，并要确保美国要战就要赢。
  <br/>
  <br/>
  苏紫云分析，过去美国是所谓的“战略模糊”，这是比较建制派的说法，也就是90年代美国对华政策20年之间的遗绪，对于中国崛起抱持耐心看待的角色，2017年逐渐转变。蓬佩奥在尼克松中心的演说就是外交上的宣告，美国国防部长的谈话就是一个很明确的战略讯号。美国有决心、有意志保护盟友的安全跟承诺。
  <br/>
  <br/>
  苏紫云：0636“美国不会掉进越战的迷思‘惧战’，国内的问题可能会变成美国担忧的部分，现在印太区域战场的环境其实不同于越战、韩战、阿富汗战争地面作战，而是海空军战场。”
  <br/>
  <br/>
  苏紫云认为，海空军战场主要依赖科技，最重要是美国国防部长的谈话，这代表在外交与军事上已经采取同一的节奏跟战略，这一点北京要慎重地思考。
 </p>

 <p>
  <b>
   张延廷分析中共具内线作战优势 美武器、精准度领先
  </b>
  <br/>
  <br/>
  台湾前空军副司令张延廷接受本台访问时分析，美国秀肌肉是“斗而不破、竞而不破”的招式，美国在这边具有领头羊、领导地位、区域安全的主角无可厚非。恰如其分、适可而止，这样就能“双赢”。
  <br/>
  <br/>
  张延廷：“他希望’不战而屈人之兵’，但是现在中共大了，他在亚洲地区想当很有份量的领导者。美国在‘第一岛链’有他的利益。两个是不是会对撞？彼此都要理性、稍微冷静下来看这问题，对立升高会‘双输’。”
  <br/>
  <br/>
  在美中防长通话聚焦在南海与台湾问题，今年五月退役前，曾是空军最后一位曾飞进中国大陆侦照的飞行员张延廷分析，中国有其地理位置优势，毕竟“敌远我近”，在中共的家门口。再者，他有“火力集中”、所谓的“内线作战”优势，他可以由内往外攻叫做“内线作战”优势。
  <br/>
  <br/>
  张延廷：“对美国是‘敌近我远’，美国的武器、精准度、美国太空人造卫星定位的能量超过中共。双方互有优劣、不能说谁有绝对优势、没有！两方都有相对的优势。所以这样来想变成‘两败具伤’。”
  <br/>
  <br/>
  张延廷分析，现在战争是“两快一慢”，“一快”是如果来的话会很快、突如其来，所以“首在决胜、远在速胜”；另“一快”战争节奏很快，开始之后后面节奏会很快，他夺取制空权、制海权，导弹的精准攻击非常快。“一慢”就是战后的平复、创伤的平复是很慢的，所以他再三呼吁，现在战争不要轻易爆发。等到11月美国总统大选后，由新总统决定，现在就要降温，不要升高对立、冲突、争端。

 <p>
  中国外交部长王毅30日到德国访问，并将会见德国总理默克尔。不过，他在德国威胁捷克参议长维特奇率团访台要付出“深重代价” ，引起捷克严重反弹，捷克外交部31日表示，将召见中国驻捷克大使，捷克外长也批评王毅所使用的语言已经超出了外交惯例。
  <br/>
  <br/>
  中国外交部长王毅到德国进行访问，被媒体问到捷克参议院议长维特奇（Miloš Vystrčil）率团访台一事时，王毅表示，捷克在台湾问题上，挑战一个中国原则，就是与14亿中国人民为敌。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  中国外交部发言人赵立坚31日在例行记者会上谴责捷克反华势力侵犯中国主权时，转述王毅的话说，“王毅国务委员昨天在德国已经明确表态，对于捷克参议院议长的公开挑衅及其背后的反华势力，中国政府和中国人民决不会听之任之、坐视不管，一定要让其为自己的短视行为和政治投机付出深重代价。”
  <br/>
  <br/>
  究竟是甚麽深重代价 ? 捷克外交部表示，将召见中国驻捷克大使，要求对王毅的话作出解释。捷克谴责中国干涉该国的内政，外交部长佩特日切克（Tomáš Petříček）说：“我希望中方能解释这些话，这次访问当然会影响我们与中国的关係，但我认为这句话太过份了。”
 </p>

  王毅访问德国之前曾访问法国，虽然法国总统马克龙与外长勒德里安都针对中国人权状况，特别是香港及新疆维吾尔人的处境，向王毅表达严正关切，但都被王毅以新疆与香港特区中所發生的事情，都是属于中国内政，他国不应该干涉驳回。
  <br/>
  <br/>
  王毅在出席法国国际关係研究院（IFRI）活动时也为国际指责香港国安法辩护，认为有实施的必要性，不仅可维护香港居民的基本权利, 也有助稳定一国两制，关于新疆维吾尔人受迫害的说法，王毅强调当地近年已没有出现暴动。
  <br/>
  <br/>
  德国总理默克尔（Angela Merkel）28日则表示，欧洲和中国应该在彼此可以合作的话题上继续对话，比如气候变化等，也应在他们有不同意的困难话题上继续对话，例如香港目前正在發生的事情。
  <br/>
  <br/>
  王毅这趟欧洲行，被各国政要强烈关切香港国安法、新疆维吾尔族人权，以及受迫害的基督徒、穆斯林及藏传佛教等问题。

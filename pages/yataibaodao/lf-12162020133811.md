 <p>
  加拿大丶德国丶法国丶英国等9个国家的国会议员日前共同致信所属国外交部长，敦促各国谴责中国对澳大利亚的霸凌行径。与此同时，加拿大总理特鲁多表示，希望两名遭中方无理拘捕的加拿大公民能尽快获释，与家人过年团聚。
 </p>
 <p>
  <br/>
  圣诞新年将至，加拿大总理特鲁多接受媒体采访时提到，希望被中国拘捕的康明凯(Michael Kovrig)和斯帕弗(Michael Spavor)能尽快回家，和家人团聚过节：“我非常希望他们快回来，这即将是他们在中国狱中度过的第三个圣诞节。他们的经历太恐怖了，但他们和其家人所展现出来的毅力是不可思议的，我们正尽一切努力让他们回家。”
  <br/>
  <br/>
  两名加拿大人因孟晚舟被拘捕而遭中国拘禁超过两年，这导致加拿和中国关系急剧恶化，加拿大民众对中国的负面印象也达到历史新高。根据加拿大亚太基金会上个月发布的民调显示，83%加拿大人希望政府能以强硬态度对待中国。仅有35%的受访者认为，中国经济实力增长是机遇而不是威胁，远低于2018年的60%。
  <br/>
  <br/>
  中国驻加拿大大使丛培武15日参加亚太基金会举办的对谈，被问及加拿大民众对中国观感的问题。丛培武说，这是某些媒体和政客刻意反中所造成的影响：“(媒体)它们不报道真实新闻，却选择传播歪曲误导的消息。而一些政治人物更为了自己的利益，也参与这种反华行动。例如，美国自己新冠状病毒疫情不可收拾，却想将责任推给中国。”
  <br/>
  <br/>
  当被问及孟晚舟已经被保释，两名加拿大人却还不见天日，丛培武说，意识到这是导致加拿大民众对中国不友善的一大原因。但他强调，两人涉嫌危害中国国家安全，和孟晚舟案件性质不同，最近两人在狱中的饮食照顾更好了，让他们可以增强免疫力以抵抗新冠病毒。
  <br/>
  <br/>
  加拿大外交部长商鹏飞则隔空反击，再度强调两名加拿大人不当被拘捕是不能被接受的，而不仅加拿大遭中国霸凌，澳大利亚丶英国等都有类似的遭遇，所以最好的方法就是盟友团结一致制裁中国：“我认为，制裁就是要与核心国家集团一起，就能发挥最大作用。过去加拿大与其他国家共同制裁白俄罗斯和俄罗斯，所以对制裁中国也正在与国际盟友协商合作。”
  <br/>
  <br/>
  15日，“对华政策跨国议会联盟”（IPAC）18个共同主席采取联合行动，敦促所在国外交部长发表声明支持澳大利亚，反对中国对澳大利亚的欺凌和胁迫。目前，加拿大丶德国丶法国丶英国丶意大利丶捷克丶丹麦丶新西兰以及欧盟已发出信函谴责中国政府。
 </p>

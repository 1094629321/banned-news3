 </p>
 <p>
  在新疆长大的乌孜别克族小学老师希迪克（Qelbinur Sidik）告诉美国有线电视网，她在一教育营担任教师的第一天，看到两名士兵用担架将一名维吾尔族年轻女性遗体抬至室外。

 </p>
 <p>
  “男警晚上喝酒的时候，会互相吹嘘自己怎么强暴与虐待女孩”希迪克说。
  <br/>
  <br/>
  希迪克说，2017年3月她被指派到2处再教育营教导“文盲”，当时新的营地大约有100名男女，“脚和手都拴着铁链”。她描述，刚到达营地的年轻囚犯看似身体健全、眼睛清澈明亮，但他们过不久就会被折磨到生病、甚至死亡。她表示营地里经常有尖叫声回荡，当她询问警察原因时，一名男警察告诉她有囚犯正在遭受酷刑。
  <br/>
  <br/>
  另一名妇女齐亚乌敦（Tursunay Ziyawudun）告诉媒体，她自2012年与哈萨克斯坦丈夫哈力克（Halmirza Halik）搬到邻国哈萨克斯坦长居，她在2017年返回新疆处理官方文件时被拘押，当时她并无犯罪纪录。齐亚乌敦1个月后获释，但2018年3月又被召回，开始她长达9个月的噩梦。
  <br/>
  <br/>
  齐亚乌敦透露，自己有一次被殴打后被两位女警带到另一个房间，命令她躺在桌子上，“她们将一根电击棍插入我的体内任意扭转，并且电击我，我昏了过去。”
 </p>

  10天后，齐亚乌敦被男警带出牢房，她听见隔壁房有一名女孩在尖叫，并看到5或6名男子走进那间房间。“接下来我遭到轮奸，当下我就明白了，隔壁房那些警察也正对那女孩做这种事。”齐亚乌敦说，并表示这在她在拘留营的期间内发生多次。去年齐亚乌敦被送往美国接受治疗并接受子宫切除手术。
  <br/>
  <br/>
  英国广播公司于2月初披露新疆集中营内存在系统性性侵，引发全球哗然。齐亚乌敦也告诉英国广播公司，集中营内女性需接受不明原因的体检、吃药、每15天被强行注射一种引起噁心和麻痺等不良反应的“疫苗”，并且被强迫在体内安装节育器。美联社也有报导曾指出维吾尔族人被强制绝育的现象在新疆很普遍。
  <br/>
  <br/>
  对于这些指控，中国政府皆予以否认，并驳斥“种族灭绝”的说法，申称新疆不存在再教育营。中国外交部发言人汪文彬就批评英国广播公司的报导与事实不符、毫无依据，并指接受采访的都是演员。

 <p>
  武汉肺炎祸害全球，各国疫情依旧严峻，第一个爆发疫情并将武汉肺炎病毒扩散至全球的中国，却在8日举办对抗疫情表彰大会，中国国家主席习近平还发表演说，反驳全球对中国隐瞒疫情的说法。欧洲疫情感染人数都在上升当中，世界卫生组织（WHO）7 日则警告，全球必须要为下一次疫情大流行做更好的准备。
  <br/>
  <br/>
  中国早期隐瞒武汉肺炎疫情，吹哨者李文亮医生等人被整肃，以致疫情为祸全球，成为世纪浩劫后，又以抗疫外交推销中国模式，遭到举世批评；但是中国国家主席习近平8日在全国抗击新冠肺炎疫情表彰大会上反驳说，中国在抗疫以来，一直公开透明履行国际义务，向世卫主动通报防疫信息，并即时发布基因序列，又在物资紧张时，向国际社会提供援助。
 </p>
 <p>
 </p>
 <p>
  <br/>
  针对中国举行全国对抗武汉肺炎疫情表彰大会的庆祝仪式，世卫突发卫生事件执行主任莱恩（Mike Ryan）表示，不清楚中国的表彰大会，但肯定中国前线医护人员付出的努力，虽然疫情在武汉爆发是一个可怕的开始，但经过这么多努力，疫情维持在非常低的水平，值得其它国家学习。
  <br/>
  <br/>
  不过，他强调，全球疫情还未完结，不应该这么快就自满。中国仍有输入病例，需谨慎以对疫情再次爆发的风险，即使零星的病例都可能扩大变成群聚感染。

 <p>
  虽然世卫秘书长高级顾问艾沃德（Bruce Aylward）肯定中国取得阶段性成果，他说， "在这次中国的疫情处理中，我认为有三件事特别显着，首先是中国从国家到地方对医疗设施的广泛投资，以及中国人为工作牺牲、保证社区安全的集体责任感 。"但他也指出，大家仍要继续努力。
  <br/>
  <br/>
  纽约时报五日报导，从中国、俄罗斯到伊朗，武汉肺炎的大流行引爆全球情报界发动间谍大战，相互刺探甚至窃取对方的武肺疫苗研发成果，有如冷战时期太空竞赛的翻版；其中又以中国的间谍行动特别复杂，熟悉疫苗间谍战相关内容的美国和欧洲官员指出，中国的手法较为复杂，其特工会偷偷利用世卫的资讯，做为以网路骇客方式窃取疫苗资讯的指引。
  <br/>
  <br/>
  世卫会搜集研发中的疫苗数据，尽管这些资讯最后都会公诸于世，但中国骇客有机会早一步得知，世卫最看好哪些武肺疫苗研究计画。美国早在二月间就察觉到中国的企图，导致白宫对世卫的态度转趋强硬。
  <br/>
  <br/>
  世卫总干事谭德塞(Tedros Adhanom Ghebreyesus) 呼吁各国投资公共卫生领域，称世界必须为下一次流行病作出更好的准备，他说，  "全球必须要为下一次疫情大流行做更充分的准备  历史证明疫病爆发及大流行是无法避免的一部分，所以这次全球流行病不会是最后一次，全球必须做好更充分的准备。"
  <br/>
  <br/>
  整个欧洲的感染人数都在上升当中，新增的确诊患者2/3都是年轻人，但是法国病情恶化尤使欧洲各国深感忧虑，德国甚至称法国是反面典型。法国的确诊病例已经达到33万5524例，确诊连续暴增，8日的新增病例是6544例，9月4日新增8975例，创单日最大增幅，政府说，病毒传播量成指数型成长，法国专家发冬季疫情高峰警报，英国的确诊病例6日也新增2988例，意大利则有277634确诊病例，新增1297例，西班牙则超过47万例。

 <p>
  中共总书记习近平本周出席上海浦东开发开放30周年大会，并考察江苏南通等长三角一体化发展，官媒对此进行高调报道。不久前，习近平还赴广东出席深圳经济特区建立40周年庆祝大会。其频繁活动，引发外界多种猜测。
  <br/>
  <br/>
  中共总书记习近平本周四（11月12日）在上海浦东开发开放30周年庆祝大会上说，要抓住机遇、乘势而上，全面贯彻中共十九大和十九届二中、三中、四中、五中全会精神，科学把握新发展阶段，坚决贯彻新发展理念，服务构建新发展格局，坚持稳中求进工作总基调，勇于挑最重的担子、啃最硬的骨头，努力成为更高水平改革开放的开路先锋。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  习近平再次强调“构建以国内大循环为主体、国内国际双循环相互促进的新发展格局”，他还说，明年是中国共产党成立100周年，上海是中国共产党诞生地，要传承红色基因、践行初心使命，不断提升党的建设质量和水平，确保改革开放正确方向。
 </p>

  主持会议的是上海市委书记李强，中办主任丁薛祥、副总理刘鹤、中组部部长陈希以及发改委主任何立峰等高官。结束上海之行后，习近平前往江苏考察。
  <br/>
  <br/>
  <b>
   学者 : 考察最发达地区与美中关系有关
  </b>
  <br/>
  <br/>
  一个月前，习近平曾考察广东并出席深圳经济特区建立40周年大会，此次再到上海等地考察，北京清华大学政治学系前讲师吴强认为其中有三层含义：“第一层含义实际上是在中美关系恶化的状况下，以此考察中国经济最繁荣，技术也最发达的三个地区，似乎想了解一些实际情况，以此强化他在党内对美强硬对抗的姿态。”
  <br/>
  <br/>
  吴强说，习近平与上海等“长三角”地区高官，进行近距离接触：“他跟这些地区的官员，实际上是做一个讨价还价，给这些地方官员放权让利，换取这些地方的支持。这种关系体现在‘十四五’规划，刚刚结束的十九届五中全会当中，把这些发达地区作为习的个人执政基层，为中共二十大作准备，为他第三个任期作准备。”

 <p>
  <br/>
  在一段三分多钟的视频中，习近平走出会场向欢迎他的人群表示：“我看也是中年以上的多，你们都是亲历者，浦东的三十年，我们改革开放四十多年，怎么走过来的，刚才我看了（展览）特别感触，就是那个叫烂泥路，这边烂泥路，这边是东方明珠塔，当时那么差，现在浑然一体。”
  <br/>
  <br/>
  12日下午，习近平在江苏南通市五山地区滨江片区，考察滨江生态环境保护情况。官媒说，习近平这次来调研长江经济带和长三角一体化发展，看看环境整治情况。
  <br/>
  <br/>
  江苏维权人士张建平对本台说，南通市近年的发展速度最快：“习近平这一次到南通来，主要还是内循环的问题。对全国而言，现在江苏经济状况是最好的，在江苏又是南通最好，一枝独秀（在江苏）。同时我们还要看到，习总书记到南通，有些维权人士被控制在家里，让书记听不到民意。”
  <br/>
  <br/>
  对于习近平下半年频繁到各省考察，官媒进行密集报道，旅美学者宋扬接受本台采访时说：“从共产党传统作风看，最高领导人如果在短时期内，频繁的到地方去进行所谓的考察和调研，那么很可能是中央和地方的关系出了一点问题，另外，最高当局可能在酝酿某一个重大的政策，他也有可能去进行所谓的调查研究，如摸底吹风等。”

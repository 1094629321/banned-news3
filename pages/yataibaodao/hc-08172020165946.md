 <p>
  几天前，瑞士驻华大使馆在微博上发了一条“萌文”，邀请中国网友在中瑞建交70周年之际，为一部瑞士电影配音。没想到，博文的评论区却意外“翻车”，大家纷纷要求瑞士政府公布中国“盗国贼”名单。这看似毫不相干的吐槽，反映出了中国民众对官员财产不予公示的愤慨。
  <br/>
  <br/>
  上周五，瑞士驻华大使馆在微博上发文说，为了庆祝中瑞建交70周年，他们邀请中国网友用汉语或瑞士三大官方语言之一，参加瑞士电影《海蒂和爷爷》(Heidi)的配音大赛。
 </p>
 <p>
 </p>
 <p>
  可能让主办方没想到的是，这条看上去蛮“接地气”的博文却成了中国网友的发泄对象。网络截图显示，很多热门评论都要求瑞士政府“公布盗国贼名单”。截至北京时间周一晚间，大部分相关评论都已被删除，但仍有少数幸存者。一位网友说：“贪官名单呢？钱都倒腾到瑞士去了吗？”还有不少网友纷纷吐槽早前的评论被大量删除的事实。
  <br/>
  <br/>
  要求中国官员公示财产的“新公民运动”支持者、湖北维权人士刘家财认为，这起网络事件反映出了国内民众对于“人民公仆”大肆敛财的愤慨。
  <br/>
  <br/>
  “中国底层民众的生活目前非常困难。一些网民在这家大使馆（发布的博文下面）要求公布‘盗国贼’的名单，实际上就是在谴责中国的这些贪官：‘人民都要吃草了，你还要人民节约，要节约的是政府官员。’”
  <br/>
  <br/>

   瑞银藏了中国的多少“黑钱”？
  </b>

 <p>
  这一事件的大背景是，一向保守中立的瑞士最近公开批评中国说，中国侵犯人权的情况持续上升，正背离开放之路，而香港《国安法》也影响了‘一国两制’的落实。若中方坚持，西方国家誓将果断回应。
  <br/>
  <br/>
  有舆论分析，这起事件让人联想到了去年网传的一篇题为《令人咋舌，瑞士银行公布消息，100位中国人存款合计7.8万亿！》的自媒体文章。该文声称，瑞银此前公布消息说，约有100名中国人的瑞银存款共计达到7.8万亿元人民币，也就是人均780亿元。
  <br/>
  <br/>
  记者注意到，这篇文章似乎曲解了瑞银集团(UBS)和国际会计师事务所普华永道(PwC)此前联合发布的一篇报告。报告指出，中国亿万富翁在2017年达到了373人。他们的财富共计高达1.12万亿美元，折合人民币约7.8万亿元。他们当中的97%都是“白手起家”的，很多人来自科技和零售行业。
  <br/>
  <br/>
  不过，瑞士驻华大使馆评论“翻车”事件和上述自媒体文章，仍然反映出了民众对瑞士银行藏匿了中国官员家族巨额“黑钱”的普遍认识和强烈不满。
  <br/>
  <br/>
  作为全球最大的财富管理公司，瑞银管理的客户总资产在2018年就达到了2.3万亿美元。美国金融杂志《机构投资者》(Institutional Investor)去年发文说，2018年，亚洲客户为瑞银的财富管理业务共注入了247 亿美元的资金，占比超过全球的三分之二，而中国离岸客户注入的资金占了亚洲客户的近一半。
  <br/>
  <br/>
  现居美国的中国经济社会学者何清涟指出，中共官员在海外藏匿巨额财富，显然是在未雨绸缪。
  <br/>
  <br/>
  “中国的高官，尤其是涉及经济和外事的一些有办法的官员，都拥有海外财产。越是中国官方敌视的国家，他们反而觉得更可靠，而这也是他们出于对西方国家（金融）体系的信任。”

 <p>
  《纽约时报》近日报道说，中国国家主席习近平、中国全国人大常委会委员长栗战书以及中共政治局常委汪洋各自的三位直系亲属，光在香港就拥有超过5000万美元的豪华房产。上周，中国华融资产的原董事长赖小民受贿、贪污、重婚一案一审开庭。他被指控在2008年至2018年担任多个公职期间，非法收受折合人民币近18亿元的财物。中央社报道，这打破了已宣判的中国官员的贪污金额记录。
  <br/>
  <br/>
  根据2017年实行的《领导干部报告个人有关事项规定》，各级党政机关、事业单位和央企的领导干部，都应向相应组织的人事部门报告他们的工资、家庭房产、投资和境外存款等事项。早在1995年，中国就建立了官员财产内部申报制度，此后相关规定经过多次修订。
  <br/>
  不过，维权人士刘家财认为，这与财产公示有着本质区别。
  <br/>
  <br/>
  “这就像既当运动员、又当裁判员一样。凡是对他不利的信息，他是不会公布的。这样的话，人民怎能监督得了他呢？而且所谓的‘反腐运动’，也是中共内部的权力斗争，并不是真正的反腐。”
  <br/>
  <br/>
  总部位于德国的国际反腐败组织“透明国际”(Transparency International)的《全球清廉指数》显示，中国去年在180个国家和地区中排在第80名，较前一年有小幅提升。

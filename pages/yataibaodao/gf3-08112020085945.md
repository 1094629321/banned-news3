 </p>
 <p>
  湖南长沙一批银行下岗职工，不满资方在十几年前体制改革中为了减员增效，  实施被指违法的内退政策。职工代表要求与银行高层会面遭到拒绝，双方爆发冲突。
  <br/>
  <br/>
  银行拒绝安排高层与下岗职工代表会面，并派保安劝他们离去。双方发生拉扯推撞，混乱中有职工倒地
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  王华一：“我们同去的一个男同志被保安摔倒在地。我要求进去面见行长给他们堵住了，这个时候一只罪恶之手从我后面向我前胸袭过来。我只感觉到一阵剧
  <br/>
  痛。他们所有员工一哄而上，全部围攻过来了，然后踢了我几脚。我当时为了自卫而反抗，我的颈椎受伤了。”
  <br/>
  <br/>
  王华一原本在工商银行工作，2000年40多岁时下岗。为了讨回公道，她与大约10名遭遇相似的“内退”职工代表上周到工商银行长沙总行要求会见领导层。在
  <br/>
  冲突中受伤后，王华一连续住院数天。她说，由于经济原因，伤势没好转也只好提前出院。
  <br/>
  <br/>
  王华一：“（工行）来了四个人，送我一点慰问金和水果，但是说，没有钱，不能借资。我说，你堂堂国企央企，有没有说，不能借资的，不可能。他要求我
  <br/>
  自己拿钱，我说我没钱。因为我们长期是低收入，已经20年了。”
 </p>
 <p>
 </p>
 <p>

  <br/>
  2000年，工商银行着手改革体制。王华一说，当时资方为了让年富力强的员工自愿下岗，软硬兼施。
  <br/>
  <br/>
  王华一：“当时他们就是在大会小会上宣传，再加上一些威胁的语言，说现在给你们这么好的机会，有这么好的条件。他们回去（下岗 ）没有上岗仍然拿那么多钱，如果这次给你们条件你们不回去的话，以后就是考试考核，竞聘上岗，末尾淘汰。”
  <br/>
  <br/>
  但后来工行一再改变政策，无法兑现向下岗员工许下的承诺。他们和在职员工待遇差距不断扩大。王华一估计，心有不甘而加入维权行列的下岗员工多达数千人。部分人曾到北京工行总行上访，但不得要领。
  <br/>
  <br/>
  王华一：“我们今天的待遇和同时期、同年龄的、同工龄的员工相比，女同志的话，少了1500到2000，男同志就少了2000到4000. 我们现在养老金的收入是3000余元。3000余元在我们长沙市就够请一个保姆。”
  <br/>
  <br/>
  就王华一等下岗职工的遭遇，本台以电话尝试接触工商银行总行信访办主任李玉舟。
  <br/>
  <br/>
  记者：“请问是李玉舟先生吗？”
  <br/>
  <br/>
  李玉舟：“不是。你说的话我听不懂。你这叫普通话呀？找个会说普通话的。”
  <br/>
  <br/>
  记者又给湖南省工行信访部门打电话。
  <br/>
  <br/>
  工行职员：“这个事情据我所知，还进行了沟通的啦。他们虽然年纪大了一点也是成年人。我虽然知道这件事，但是应该由他们反映情况。第一你没有参与，第二你不属于这个群体里面的人员。第三，他们也没有委托你干什么。”
  <br/>
  <br/>
  湖北原银行职员伍立娟16年前下岗后一直积极维权。她说，王女士等人的遭遇只是冰山一角。
  <br/>
  <br/>
  伍丽娟：“如果不内退，现在这些工商银行的职工干部，（工资）最低都有五六千，八九千上万都有的。工商银行把这些下岗内退的名额，用到了亲属和子女
  <br/>
  身上。‘不查不知道，一查真奇妙’。”很多（工商银行分行）都成为家族亲戚亲
  <br/>
  属连带关系的工商银行。”
  <br/>
  <br/>
  伍立娟说，在目前的大环境下，下岗人员很难与资方抗衡，对此她感到诅丧。
  <br/>
  <br/>
  <br/>
  记者：高锋  责编：胡力汉 嘉远 网编：瑞哲
 </p>

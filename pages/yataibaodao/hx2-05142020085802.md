 <p>
  美军太平洋舰队的官方脸书主动披露一艘驱逐舰航经台湾海峡。台湾国防部也已证实。这是美舰今年第六次通过台海。台海两岸马上将有一系列的政治活动前夕，美军的行动格外引发关注。
  <br/>
  <br/>
  美国海军太平洋舰队（U.S. Pacific Fleet）主动在脸书粉丝团上披露，麦克坎贝尔号驱逐舰（DDG-85）通过台湾海峡。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  在台湾，国防部发言人史顺文14日接受自由亚洲电台采访，也证实此事，不过国防部对美舰通过的时间和舰种都未提供说明，只表示：“美军一艘作战舰上午由北向南航经台湾海峡，执行一般航行任务，目前仍继续向南航行，国军运用联合情监侦掌握区域情势，状况正常。”
 </p>

 </p>

  <br/>
  台湾中央通讯社报导，根据国防部的公开资讯显示，这次公布的美国军舰通过台湾海峡，是今年以来第6次。分别为1月17日的CG-67、2月15日的CG-62、3月26日的DDG-85、4月11日的DDG-52、4月24日的DDG-52、今天的DDG-85。2019年美舰共9次通过台湾海峡。
  <br/>
  <br/>
  <b>
   美军不断回应中国军事活动
  </b>
  <br/>
  <br/>
  台湾智库谘询委员赖怡忠接受自由亚洲电台采访指出，中国最近军事活动很多，美国就是回应中国的军事活动。所以中国每次有飞机或船舰一过来，美国飞机或船舰过几天可能马上有回应的动作。
  <br/>
  <br/>
  赖怡忠：“美国军舰通过台湾海峡基本上是从约2018年开始的寻常任务，主要是稳定台海局势，警告老共不要轻举妄动。”
  <br/>
  <br/>
  台湾大学政治学系名誉教授张麟征接受自由亚洲电台采访指出，两岸关系不可能好，因为台美关系太好了，令很多台湾老百姓安全感很低。
  <br/>
  <br/>
  张麟征：“台湾跟老美好的不得了，表面上是好的不得了，但老美是拿你当牌在打，什么时候将你弃之不顾，你不知道。现在看起来两岸不会马上有什么军事冲突，但是现在不会，不表示未来不会。现在老美照顾你，不代表以后老美会照顾你。现在中美在较劲，不管防疫也好、贸易也好、安全方面、战略方面、围堵、反围堵也是一样，基本上要看中美之间的角力博奕什么时候会完。”
 </p>

  <b>
   美军频繁现台海两目的：钳制中国、安抚台湾
  </b>
  <br/>
  <br/>
  对中国、美国军机军舰频繁地轮流出现在台湾周边，张麟征作政治解读：“大陆来，美国来，军舰军机绕着台湾走，假设最后引爆擦枪走火的冲突，战场在那里？台湾人不会发抖吗？战场一定是在台湾！”
  <br/>
  <br/>
  张麟征分析，“大陆军机军舰绕着台海周边走，是在威慑、告诉美国，这是我的核心利益，所以你不要太越红线。那美国的军舰军机绕着台海周边走，是安抚台湾人心，想对冲大陆对台湾安全上的威慑，要给大陆制造麻烦、钳制大陆持续壮大。但是大陆绝对不想跟美国冲突，美国虽然姿态做足，也不愿意跟大陆发生冲突。那贸易方面还可以较量一下，其他的军事冲突暂时不太可能。”
  <br/>
  <br/>
  前国防部副部长林中斌接受自由亚洲电台采访也预测，即便解放军和美军之间，应对的动作很多，但是双方绝对不会开战。
  <br/>
  <br/>
  林中斌：“一百二十个放心，不会打起来！以前武统声浪几乎到了天上，全国一致。现在这个开始变化，这个很睹定，他们用的理由最后还是要武统，但是现在不宜。以前没有说现在不宜，以前说蔡英文再搞台独，马上要打了，现在的转变是‘现在不宜’。”
 </p>

  <b>
   台湾前国防部副部长：不可能开打
  </b>
  <br/>
  <br/>
  林中斌举例说，《超限战》作者乔良近来也说，终究还是武统，但“现在不宜”。重点在“现在不宜”，北京也发现了所谓“战狼式的外交”，最好收敛一下，否则对他们的利益是有害的。
  <br/>
  <br/>
  林中斌透露他透过私人管道了解到，蔡政府的一些官员，最近积极询问怎么能在中美较劲当中，不要卷入？这很有意义，而且在这个时间问 。
  <br/>
  <br/>
  林中斌：“我所看不是那么紧张，两岸恶化、恶化到顶了，开始要转向了。恶化对双方不利。蔡英文已当选，没有必要顾虑不反映人民仇中意见而选不上。台湾人民仇中是北京引起，不是我们挑起的。”
  <br/>
  <br/>
  林中斌指出，最主要是北京自己的作为让台湾人民非常害怕，跟香港一样还得了！最妙的是，台湾很多第三代外省人，在上次总统选举中支持蔡英文，居然跟爸妈闹翻了。这是以前没有的事。

  林中斌：“这非常有象征意义。我相信北京应该注意到第三代外省人应支持统一 、支持北京，但是没有！他们宁愿跟爸妈闹翻也要反中。这事情要北京自己问一下你这台湾工作怎么做的？我相信北京已经在做内部的反思了。所以有很多都是新的情况，总结起来不会打起来，放一百二十个心吧。”
  <br/>
  <br/>
  最近日媒报导，中国大陆有可能在今年8月在南海的东沙群岛进行夺岛演习。林中斌解读：“通常一个政府在转弯之前，一定先做一下相反的动作，两边都要照顾到。一个国家大的就像一艘航母，你要转不能转太快，否则要翻船。就像邓小平以前讲的，我们要反右，但最主要要防左。 维持现状是安抚内部的必须言论。”
  <br/>
  <br/>
  至于美方军机、军舰动作不小，林中斌认为，美国防疫方面令人忧心，特朗普在防疫上遭到美国内部批评声浪，必须另有作为。指向疫情失控是中国所害，如今中国要危害台湾，美国要跟中国对抗到底，将内部责难焦点转向台海。
  <br/>
  <br/>
  林中斌说，美国没有要真正开战 ，这对美国没有好处。特朗普一上台，就调整方向，对外作战的都停下来，这是他一向的政策。他连驻外的军队都要撤回，他还会开辟另一个战场，那不是疯掉了？

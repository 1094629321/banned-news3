 <p>
  日本冲绳附近的钓鱼岛，中、日、台三方都宣示拥有主权。中国大陆叫钓鱼岛，台湾叫钓鱼台，日本叫尖阁诸岛。最近日本石垣市有议员提案要更改钓鱼岛的地址，再度引发争议。
  <br/>
  <br/>
  日本冲绳县石垣市长中山义隆9日在市议会提案，将钓鱼台列屿（日本称“尖阁诸岛”）行政区域地址，从原本的“石垣市登野城”改为“石垣市登野城尖阁”，预定本月22日表决。执政的自民党在市议会占多数席次，议案通过可能性高。由于日本、台湾、中国都宣称拥有钓鱼台主权，三方争议再起。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  台湾驻日本代表谢长廷9号表示，已两度向日本表达严正立场和关切，台湾拥有钓鱼台列屿的主权，这个立场不会因为日方变更地址而有影响。谢长廷还说，石垣市长中山义隆表明更名并没有特别的意思，只是因为石垣岛也有一个地名叫做“登野城”，与钓鱼台列屿（尖阁诸岛）登记的地名相同，两者容易产生混淆，才决定改名，并且更名一事只是地方议会的决议，并非中央决策，也不是现状有很大的变化。
 </p>
 <p>
 </p>

  台湾保钓团体9日前往位在台北的日本台湾交流协会抗议。国民党也发声明指出，无论从历史文献、地理及法理来看，钓鱼台主权属于中华民国无庸置疑。1972年美国片面将琉球群岛及钓鱼台列屿的“行政权”移交日本，风波至今未平。日本自始无权主张拥有钓鱼台列屿主权。马英九执政时期对钓鱼台列屿主权提出严正声明，且在当时签订的《台日渔业协议》中，坚持主权，确保台湾渔民在钓鱼台周边捕鱼的权益。
  <br/>
  <br/>
  国民党发言人洪于茜接受自由亚洲电台采访指出：“执政党无视日本公务船一再非法干扰、驱赶台湾渔民，又无法强硬维护主权跟渔权。我们在此严肃要求总统蔡英文和民进党政府不可再软弱，务必采取一切必要方式，维护中华民国在钓鱼台列屿的主权和渔权，以及台湾渔民的人身安全，否则会愧对国人。”
  <br/>
  <br/>
  <b>
   宜兰民代连署更名反制
  </b>
  <br/>
  <br/>
  宜兰县苏澳区渔会表示，钓鱼台是渔民的传统作业海域，日本改名居心叵测，更伤害苏澳镇与石垣市作为姐妹市的情谊。

  钓鱼台在台湾被划归在“宜兰县头城镇大溪里钓鱼台路1号”，这块门牌还保存在大溪里长家。近日有宜兰县议员发动连署要求把钓鱼台改名为“头城钓鱼台”，对日方更名行动进行反制。宜兰县长林姿妙也表示要与议会带队登岛挂门牌，并邀请蔡英文总统一起登岛宣示主权。对此，蔡英文打破沉默，10日出席民进党中常会对外作出回应。
  <br/>
  <br/>
  蔡英文：“钓鱼台是中华民国的领土，这是政府一贯的立场。呼吁各方秉持‘搁置争议、共同开发’的原则，以和平方式解决争议，共同维护区域的和平稳定。”
  <br/>
  <br/>
  蔡英文表示，感谢林姿妙的邀请，并强调，中央政府一定会依情势审慎处理。一方面确保主权立场，另外一方面也能够维持区域和平稳定。
 </p>

 <p>
  <br/>
  中国大陆则派出海警船巡弋钓鱼台海域宣示主权。中国外交部重申，钓鱼台列屿是中国固有领土，中国拥有钓鱼台主权的事实，不会因为任何国家将该列屿改名而有任何改变。日方对钓鱼台列屿的片面举动，并无助于区域安全与稳定，望日方审慎处理。
  <br/>
  <br/>
  综合日本媒体报导，中国海警船近期频繁驶入钓鱼台海域，5月发生连续3天入侵日本“领土”。自民党外交部会拟定一项决议文，要求政府透过外交管道向中国提出严正抗议。决议文指出，中国公务船异常的行动明显，日本绝不容许这种状态，针对中国海警船在钓鱼台海域追逐日本渔船的行为，是“海上的追踪狂行为”。日本防卫大臣河野太郎日前也公开表示，中国持续威吓台湾等地、中国公务船屡次航行在钓鱼台海域等举动，令人强烈担忧，他并暗指安倍政府有必要重新思考是否邀中国国家主席习近平以国宾身分访日。
  <br/>
  <br/>
  <b>
   学者：钓鱼台争议再起  台湾最好袖手旁观
  </b>
  <br/>
  <br/>
  东海大学日本区域研究中心主任陈永峰10日接受自由亚洲电台采访认为，今年钓鱼台更名争议的直接原因，与武汉肺炎影响习近平4月访日行程后，中国公务船多次进入钓鱼台海域驱赶日本渔船有关，这是过去罕见发生的事。
 </p>
 <p>
 </p>

 </p>

  陈永峰：“本来习近平应该是在樱花开的时候，要去东京访问，当作国宾招待。新的天皇要以国宾招待的人，后来不能去。应是东京方面的提议，北京马上有反应，禁止日本所有的不只观光客，包括商务客都不能进上海、北京。东京方面又反应，到三月中旬左右，真正封了中国来的客人，否则本来只有封浙江省跟湖北省两省。动作非常慢，百分百跟习近平不能访日有关。从那时开始，到底中国军方主导或中国行政部门主导不知道，就一直有中国行政船进到本来是日本控制的钓鱼台周边海域。”
  <br/>
  <br/>
  陈永峰指出，日本挑动改名是针对中国不是针对台湾，究竟是日本地方政府自己决定更名，还是和中央有默契则不得而知。
 </p>
 <p>
  陈永峰说，中国向来对钓鱼台的主张采“三段论法”，也就是，钓鱼台是台湾的一部份，台湾是中国的一部份，所以钓鱼台是中国的。因此，蔡英文在这个问题上是比较困难的。如果台湾什么都不说，事实上，中国没有办法把钓鱼台跟北京连在一起。不过台湾前总统李登辉曾主张，钓鱼台是日本领土不属于台湾，避免麻烦。
  <br/>
  <br/>
  陈永峰说，蔡英文说钓鱼台是中华民国的也没有错。台湾认为钓鱼台是台湾附属岛屿。依1895年《马关条约》，应跟着台澎割让给日本。第二次世界大战日本战败，《马关条约》被废弃，钓鱼台被美国占领，后来钓鱼台又随冲绳还给日本，从那之后一直在日本行政控制底下。
  <br/>
  <br/>
  如何不卷入日中关系紧张，影响台日关系？陈永峰认为：“我们所有的外交关系跟中华民国扯上关系的时候，就一直讲一样的话就可以了，这是我的主张。蔡英文这样对应就可以了，因为并没有办法直接改变事实。另外一方面，钓鱼台现在事实上仍在美军控制下，所以去处理那个问题，不是只有跟日本之间的问题，台湾最好的方法就是站在旁边看。”
  <br/>

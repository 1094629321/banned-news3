 <p>
  山东警方针对 “厦门聚会案”展开跨省抓捕以来，部分涉案人出走国外，包括异议人士刘四仿。家属怀疑受到株连，快要流离失所，年幼的儿子也面临失学。
  <br/>
  <br/>
  去年年底，来自中国各地的异议人士云集厦门聚会，其后演变成大抓捕。原籍四川，在广东落地生根的异议人士刘四仿为了逃避追捕，早前出走美国。他周四（6日）向本台表示，留在广东佛山的妻儿近日被房东逼迁，怀疑是受到株连。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  刘四仿：“房东直接就说是当地警察找他，强迫他们不能租房子给我家人。当局这次的理由是我去年年底，在厦门参加了一个公民聚会。我现在是在美国，当局痛恨没有把我给捉拿归案，所以对我家人下手来报复我。典型的株连和迫害。”
  <br/>
  <br/>
  房东本来要求两人马上搬走，经过交涉，才让他们住到本月底。而更让刘四仿烦恼的是8岁儿子的学业。
  <br/>
  <br/>
  <b>
   欺压异议人士子女 连学校都没得读
  </b>
  <br/>
  <br/>
  刘四仿：“上个月的时候学校找我太太谈这个事情。意思很明确，当然也没有直说，意思就是‘没有办法让你孩子继续在这学校读书了’。当时我太太也非常伤心。她知道学校的难处，但没办法，她跟孩子讲，‘如果你必须留在这边上学，那么学校就可能没办法再办下去了，所以我们就只能离开这个学校’。‘孩子表示能理解，我听到之后心里特别感到难受。”
  <br/>
  <br/>
 </p>

 <p>
  刘四仿本来任教于儿子就读的私人学校。与此同时他也是歌曲创作人，与音乐搭档徐琳创作了多首脍炙人口的维权歌曲。三年前，两人先后公安扣查。其后徐琳“寻衅滋事”罪成判刑3年，刘四仿则获取保候审，并且在当局压力下离开了佛山。
 </p>
 <p>
  刘四仿：“我是2017年被抓过，当时做了个妥协，取保一年，不发表作品或者写文章，公开发表言论等等。关于2019年厦门的会议也好，公民聚会也好，我们认为这是中国宪法所赋予的公民权利，当时当局认为这是对他们权威或政权构成的挑战。”
 </p>
 <p>
  维权律师隋牧青表示，刘四仿在厦门聚会扮演重要角色，自己很清楚，一旦落网后果不堪设想。
  <br/>
  <br/>
  <b>
   刘四仿协助安排厦门聚会会场 当局视为眼中钉
  </b>
  <br/>
  <br/>
  隋牧青：“他离开佛山后又重新开始参与各种公民活动，厦门那个会议，据我所知他（刘四仿）是参与帮助安排会场，录音录像等等。所以后来他们得知，有一些人被抓以后，因为他在广州，离香港很近，就迅速逃到了香港，从香港飞到了美国。”
  <br/>
  <br/>
  妻子卢丽娜本想带着儿子与刘四仿会合，可是事与愿违。
  <br/>
  <br/>
  隋牧青：“他在香港等太太和孩子去会合，就差一天左右，妻儿被警方拦截了，在广州东站坐广九直通车，好像是在边检的地方遭拦下来了，然后就边控，被拦下来了，不能出境了。”
  <br/>
  <br/>
  记者尝试联系佛山市公安分局要求就刘四仿妻儿的遭遇解释，但电话一直无法接通。

 <p>
  今年以来，新疆强迫劳动的问题再度成为国际舆论关注焦点。澳洲战略政策研究所指控83家跨国公司的供应链工厂存在僱佣新疆再教育营裏的维吾尔族工人的问题，全球超过250个人权与劳工团体也发起结束中国维吾尔地区强迫劳动联盟运动，美国甚至对新疆部分商品发布禁令。针对西方的指责，中国政府星期二发布调查报告称，新疆少数民族劳动就业呈现明显的自愿性、自主性和自由性。
  <br/>
  <br/>
  中国新疆发展研究中心20日发布《新疆少数民族劳动就业调查报告》指出，新疆少数民族民众劳动就业呈现明显的自愿性、自主性和自由性。世界维吾尔代表大会则强调，这个报告企图转移国际社会谴责中国强迫维吾尔族人充当边疆劳役的事实。
 </p>
 <p>
 </p>
 <div>
 </div>
 <p>
 </p>
 <p>
  针对西方一些智库发表报告指出，新疆存在“大规模强迫劳动”现象，新疆发展研究中心邀请专家学者，对新疆少数民族民众劳动就业情况进行调查。
  <br/>
  <br/>
  《调查报告》称，近年来，新疆南部喀什、和田、阿克苏地区以及克孜勒苏柯尔克孜自治州的各项产业快速发展，但仍不能满足当地民众就业需求，越来越多城乡富余劳动力，将目光投向了工资收入更高、工作条件更好、生活环境更佳的新疆北部城市和内地发达省市。
 </p>
 <p>
 </p>

 <p>
  世界维吾尔代表大会發言人迪里夏提反驳这份调查报告，他说，“报告居然宣称很多劳动力是把目光投向收入更高、条件更好的内地發达城市，这完全是颠倒黑白的谎言，为什麽中国政府不将他们安置到当地提供更高的工资和更好的福利，相反的，中国为了促进更多的移民进入到当地，却提供了很优厚的福利，高工资、免费提供住房、保障所有相应配套的福利，同时还保证拥有退休金，这麽好的福利待遇，为什麽不提供给这些维吾尔族的劳动力呢 ? ”
  <br/>
  <br/>
  《调查报告》也证实，在过去一个时期里，新疆少数民族民众自发性就业确实存在质量低、收入低、稳定性低“三低”的问题。
  <br/>
  <br/>
  《调查报告》也就近年来，不少专家学者研究了少数民族在北京、天津、武汉、南京、东莞、西安等地劳动就业状况，说明他们普遍认为，少数民族外出劳动就业完全是自愿、自主和自由的，自愿决定外出劳动，自主决定去哪里就业，自由选择何种工作。
  <br/>
  <br/>
  针对调查组强调，在相关学术文献中，没有查到任何关于“强迫劳动”的词语或近似表述，迪里夏提认为，“当局发布这个所谓的自愿调查报告，企图是转移国际社会谴责中国强制强迫维吾尔人充当边疆劳役的事实，报告严重歪曲现实，将现代版的奴隶美化成所谓的自愿。”
  <br/>
  <br/>
  世维会指出，所谓问卷调查完全是一个闹剧，任何人在中国政府的质询下，只能按照当局的需求去回答问题，如果提出不同或表达自己的想法，这个人就会失去自由，关进集中营内，接受政治洗脑。
  <br/>
  <br/>
  <br/>

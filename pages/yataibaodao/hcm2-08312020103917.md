 <p>
  香港“831太子站事件”发生一周年这天，超过200名来自港、台的民众，自发地来到台北的香港手足开设的保护伞餐厅前，拉起“挺香港人链”。此外，在台中、高雄也举办了类似的活动。
  <br/>
  <br/>
  “光复香港，时代革命，五大诉求，缺一不可！”
  <br/>
  <br/>
  位于台湾大学对面、台北市新生南路上的保护伞餐厅外，31日傍晚六点左右就陆续聚集不少穿着黑衫的民众，来声援香港“831事件”。现场的民众也唱起了《愿荣光归香港》这首去年反送中期间流行的歌曲。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>

  <br/>
  祈求民主与自由，万世都不朽，我愿荣光归香港！”
  <br/>
  <br/>

 <p>
  31日晚间的这一场活动“无大台”，来自香港和台湾的学生们南北串连分别在台北与高雄发起“挺香港人链”一起手拉手。主办的学生表示，“831太子站事件满一周年”，希望在台湾的人，能一起手拉着手、点亮灯光，在海的另一头，为香港组成一条自由之路。
  <br/>
  <br/>
  主办同学：“他们没办法发言的时候，我们虽身在台湾，我们愿意为香港手足来发声。我们愿意用我们的生命，来延续香港青年他们的热情。面对中国的暴政，虽然还不到50年，香港人已经体会到‘一国两制’的可怕，‘一国两制’就是对中国政府的妥协。”
  <br/>
  <br/>
  另一位发起人周同学则说，活动选在公馆相当有意义，除了这里是保护伞餐厅之外，去年香港“反送中”运动以来，台湾也在台大前的公馆地下道发起连侬墙。831这个日子有不同的意义。
  <br/>
  <br/>
  周同学：“我觉得这是个特别的日子，除了纪念香港831事件，也是我们重新回到这里(公馆)，在连侬墙撤除一年后，回到公馆这个地方，公馆也多了保护伞餐厅，由香港手足开的店。他们在异地依然心心念念着香港，希望能在这地方为香港尽一份心力。”
 </p>

  <b>
   香港自由恶化 移居台湾港人呼吁珍惜自由
  </b>
  <br/>
  <br/>
  周一台北的天气忽而大雨、忽而放晴，不过，还是有超过200名民众自发地到场来声援“831事件”。来自香港的江先生与两个同为香港籍的朋友结伴而来，去年香港“反送中”爆发后，香港的自由状况日益恶化，也让三人决定移居到台湾。
  <br/>
  <br/>
  江先生：“这边的香港人发起这个活动就是要提醒香港还有台湾所有的居民，究竟在香港发生什么事，和中国合作会有什么后果。这是大家都要反思的问题，不只是香港人。现在台湾还有自由，你们要珍惜现在的情况。我们来台湾是因为这边可以有自由的生活，要不然我可以回去。”
  <br/>
  <br/>
  牵着爱犬来参加人链活动的赖先生夫妇，更是一个月前才成为台湾“新住民”，他们也是在“反送中”运动之后，决定在台湾安身立命。

 <p>
  台湾读高三的刘同学刚放学就来加入人链行列。从“反送中”运动以来，刘同学一直关心着香港抗争活动。尽管已经过了一年，随着台湾媒体关注度降低，同学之间谈论也少了，不过，刘同学还是愿意以行动来声援香港。
  <br/>
  <br/>
  刘同学：“香港的处境某方面跟我们蛮相关，尽管香港是中国的特别行政区，但是台湾不是。但是看在中国的眼里，其实台湾就是下一个要收回的地方。我觉得他们的处境跟我们一样，在我们遇到中共打压的时候，香港为我们发声，我们也该为香港发声。”
  <br/>
  <br/>
  除了台北之外，台湾香港协会也在台中旧火车站前发起“撑香港人链”活动，在高雄则有港人团体“维多利打狗”发起“831高雄挺香港人链”声援活动，号召民众到捷运站牵手串成人链。根据台媒苹果日报报导，高雄市行政暨国际处长项宾和代表高雄市长陈其迈到场致意，并承诺市府已在研拟协助港人行动方案，会架设可联系公共平台，为需支援与协助的港人尽一份力。

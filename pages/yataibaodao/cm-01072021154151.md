 <p>
  香港53名泛民人士1月6日被抓捕后，现在大部分已经获得保释。美国国务卿蓬佩奥（Mike Pompeo）发表声明批评，并指美国正考虑实施制裁。
  <br/>
  <br/>
  美国国务卿蓬佩奥（Mike Pompeo）在1月7日凌晨发表的声明中，谴责中国共产党蔑视自己的人民和法治，要求这些人应被无条件立即释放。他也提到，香港人正承受中共的压迫下受苦，美国不会袖手旁观，并表示美国正考虑对涉及拘捕行动的个人和实体实施制裁，也将探索对香港驻美国的经贸办事处施加限制。
  <br/>
  <br/>
  “美国将考虑对涉及此次香港大规模拘捕行动的任何个人和实体实施制裁及采取其他限制。”蓬佩奥说。
  <br/>
  <br/>
  在50多名被捕人士中包含一名美国公民 – 亚洲人权委员会主席，同时也是香港民主派组织“民主动力”的成员关尚义（John Clancey），他在当日深夜已获准保释。对此，蓬佩奥也强调，“美国不会容忍对美国公民的任意拘留或骚扰。”
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  <strong>
   美考虑制裁 中港政府不怕
  </strong>
 </p>
 <p>
  <figure class="image-richtext image-inline captioned" style="width:1751px;">
   <img alt="戴耀廷获得保释。（AFP）" src="https://www.rfa.org/mandarin/yataibaodao/gangtai/cm-01072021154151.html/000_8yc7bz.jpg/@@images/2ef7e507-8fe8-4bde-8133-0fe784d97a5c.jpeg" title="2"/>

    戴耀廷获得保释。（AFP）

  <br/>
  <br/>
  对此，港府回应有宪制责任维护国家安全，无惧任何海外政府的制裁威胁。中国外交部发言人华春莹在7日的例行记者会上，也表示支持香港政府“依法打击违法犯罪行为”，再度痛批美国干涉中国内政。
  <br/>
  <br/>
  “中方将采取一切必要措施坚决维护自身主权、安全利益。美方必将为其错误行为付出沉重代价。”华春莹说。
  <br/>
  <br/>
  美国非政府人权组织自由之家高级研究员莎拉.库克 （Sarah Cook）告诉本台，即使新旧政府交接在即，美国政府对侵犯人权的个人或实体的制裁不会因为白宫易主有所影响，加上此类制裁针对特定对象，在有效性上具有一定优势。
  <br/>
  “针对这些特定官员的有特定的行动，对他们的生活、财务、出入能力有真正的影响，至少会造成（他们）严重的不便，但不会影响更广泛的人口。”她说。
  <br/>
  <br/>
  在香港国安法实施后，美国政府在去年八月宣布制裁11名中港官员，其中包含香港特首林郑月娥，她在接受香港国际财经台（HKIBC）访问时提到因为制裁所以停用银行户口，“有大量现金在家”。
  <br/>
  <br/>
  <strong>
   专家：香港议题不会因白宫易主被忽视
  </strong>
 </p>
 <p>
  <strong>
   <figure class="image-richtext image-inline captioned" style="width:5354px;">

     林景楠获得保释。（AFP）

 </p>
 <p>
  除了蓬佩奥外，总统当选人拜登提名的下一任美国国务卿布林肯（Antony Blinken）在社交媒体推特上发文，强调拜登政府与香港人民站在一起，对抗北京对民主的打压。
  <br/>
  <br/>
  “大规模逮捕泛民主抗议者的作为，是对那些勇敢为普世权利发声人士的攻击。”布林肯写道。
  <br/>
  美国智库战略暨国际研究中心（CSIS）中国权力专案主任葛来仪（Bonnie Glaser）以布林肯的发文为例告诉记者，目前对香港局势的担忧已是两党共识，因此美国是否会对香港实施进一步制裁，不会因为特朗普任期即将结束而淡出焦点。
  <br/>
  <br/>
  “特朗普政府采取了许多行动来惩罚北京和香港那些正在破坏该城市自治的人，他们会将接力棒传递给拜登政府，拜登政府也会继续关注。”葛来仪在信件回覆中写道。
  <br/>
  <br/>
  但葛来仪强调，美国是否会接纳来自香港的难民，以及人数多寡，是尚须厘清的议题。美国众议院在12月通过《香港人民自由与选择法案》（Hong Kong People‘s Freedom and Choice Act），为逃离政治迫害的在美港人提供“临时保护身分”。此法案在新国会任期是否能通过，也是新政府与国会上任后的关注焦点。
  <br/>
  <br/>
  <strong>
   美须联合他国 向中施加更大压力
  </strong>
 </p>
 <p>
  除了美国之外，包含欧盟、加拿大、英国都批评香港的拘捕行动。欧盟执行委员会发言人也向媒体强调，不排除对中国的镇压行径施加进一步制裁。
 </p>
 <p>
  库克告诉本台，拜登政府强调国际合作制衡中国势力，上任后若能联合各国施压，透过多边合作必能带给中共及香港政府更多国际压力。
 </p>
 <p>
  “如此可以清楚传递，这是许多不同利益的国家共有的原则性问题，这无疑增加了压力和效力，因为不仅是这些官员无法来美国或不能将钱存入美国银行，而且他们也将无法前往加拿大或英国旅行。”
  <br/>
  <br/>
  至香港时间1月8日凌晨截止统计，涉嫌违反《香港国安法》被捕的53人大多数已被保释，暂时未有人被起诉。
 </p>

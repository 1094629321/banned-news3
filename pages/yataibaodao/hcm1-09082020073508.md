 <p>
  中国8日在北京人民大会堂举行全国抗击新冠肺炎疫情表彰大会，中国国家主席习近平为钟南山等4人颁赠勋章。在此同时，世卫紧急项目执行主任瑞安在回应记者答问表示，中国仍持续有输入个案，没有自满的空间。唯有等到所有地方结束才叫真正结束。
  <br/>
  <br/>
  司仪：“在非典型肺炎和新冠肺炎疫情防控中，做出巨大贡献。”
  <br/>
  <br/>
  根据中国官媒新华社报导，中国国家主席习近平为中国工程院院士钟南山、中央指导组专家组成员、中国工程院院士、天津中医药大学校长张博礼以及担任湖北省卫健委副主任张定宇和解放军少将，中国工程院院士陈薇等国家勋章和国家荣誉称号获得者一一颁授勋章奖章。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  习近平在致词中提到，面对突如其来的严重疫情，党中央统揽全局、果断决策，以非常之举应对非常之事。习近平表示，成立中央应对疫情工作领导小组，派出中央指导组，建立国务院联防联控机制。提出“早发现、早报告、早隔离、早治疗”的防控要求。习近平还声称，抗疫斗争伟大实践再次证明，新中国成立以来所积累的坚实国力，是从容应对惊涛骇浪的深厚底气。
 </p>

  不过，世界卫生组织(WHO)在前一天的记者会，被记者问及看法，世卫紧急项目执行主任瑞安表示，祝贺中国的前线工作者和社区取得如此成就，但正如在其他国家了解到、都认识到，唯有等到所有地方都结束，才叫真正结束。
  <br/>
  <br/>
  瑞安：“正如你说的，中国还持续有境外输入的个案，因此没有自满的空间。随着很多国家，特别是中国恢复正常生活，疫情有再次爆发的风险。”
  <br/>
  <br/>
  瑞安说，零星个案会变成群组，群组可以变成社区传播，社区传播可能导致医疗系统不堪负荷。今年上半年看到这个情况，某些国家也继续出现这个情况，世卫要避免再次发生。

  <b>
   中国强化独有政治模式 企图淡化疫情扩散责任
  </b>
  <br/>
  <br/>
  除了8日的抗击疫情表扬大会之外，前一天，新华社还发表一篇习近平领导中国战“疫”的特稿，还制作一支习近平抗“疫”日历视频。结论强调，“非常之年，习近平以强烈的历史担当，坚定践行对人民的承诺，确保完成决战决胜脱贫攻坚目标任务，全面建成小康社会。”
  <br/>
  <br/>
  北京异议人士胡佳接受本台访问分析，目前所有的证据都显示，新冠肺炎疫情最初的源头来自武汉华南海鲜市场，中国现在的统治者很希望在疫情扩散方面能摆脱他的责任与负面形象，要显示出在疫情防控方面，打了一场全世界最漂亮的仗。
  <br/>
  <br/>
  胡佳：“他要在这里把自己的责任降到最低化，其次，要显示现有的政治制度，现有的中国政治模式要抗击疫情是有得天独厚的优势，中国社会主义制度优势。这点首先要让中国公民、被统治者强化这种意识。”
 </p>

  胡佳指出，除了对内之外，对抗世界各国对中国在疫情初期没有治理好，通过全国性的扩散，最终演变成世界性的灾难的指控，中国要在这方面抹平、虚化、推打回去。
  <br/>
  <br/>
  胡佳形容，现在中国媒体每天开足马力，都是在宣传世界各地在疫情下沦陷的国家，最主要的报导对象是美国，现在又加入印度。因为印度与中国有边界争端，现在也处于紧张状态，而且现在印度是个案增长量最高的国家。
  <br/>
  <br/>
  胡佳：“他的确起到一定作用，因为我现在看到很多很多中国境内普通公众的言论说，看来’民主不是万病的良医’，有些时候专制社会有专制社会的优势，有些人还真的相信共产党所谓‘集中力量办大事’。”
  <br/>
  <br/>
  胡佳说，中国统治者现在用在疫情防控发挥效果，不仅在国内、也向海外强化他的制度优势。某种程度在中国境内在宣传、灌输、洗脑方面有一定效果。胡佳指出，对内宣传是中共根本性的东西，因为他要让这个国家稳固的话，中国不会因为发生一场战争而迎来社会变革，肯定是因为内部这样去影响。

  <b>
   2020建成小康社会难度高 中国靠说自己故事“粉饰”
  </b>
  <br/>
  <br/>
  台北海洋科技大学通识教育中心助理教授吴建忠受访表示，从中国隐匿疫情开始，在发展所谓的中国叙事部分，包括如何从丧事变喜事概念来操作，这是中国共产党一直以来的套路。从一开始的抗疫英雄、习近平的亲自指挥、部署，这两天官媒颁奖，抗疫的成功，中共一直很努力要把抗疫的“成就”告诉给世人。
  <br/>
  <br/>
  吴建忠：“我们看到这半年以来，不管是水灾、粮荒、疫情等，其实‘大大’已经讲话，他讲完话已经一锤定音。以目前王毅在欧洲处处碰壁的情况下，他必须适度补足外宣的内容叙事。从这半年以来中共就不断透过这块在做叙事的精进与补强故事的情况。”
  <br/>
  <br/>
  吴建忠举例，就像过去中共对文革的定义，总体来讲还是成功，虽然有些过错。但是只要一锤定音，基本上所有调性都会跟上官媒的步伐。这次“抗疫”叙事的部分，随着年底全面建成小康社会要完成，10月要召开19届五中全会，中共必须“走自己的路、说自己的故事。”把习近平防疫不力的表现或角色，转成他是“防疫英雄”，把所有功绩都记在他身上。

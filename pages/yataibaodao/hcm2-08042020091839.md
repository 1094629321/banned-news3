 <p>
  台湾传出“零本土”病例的记录可能被打破后，包括政界以及公卫界呼吁在机场进行大范围筛查的声浪高涨，希望能做到阻绝境外病例输入，避免造成下一波社区大流行。
  <br/>
  <br/>
  比利时工程师在台停留超过80天后，日前被检验出新冠病毒阳性。根据卫生局调查，这名男子5月到6月都在彰化，但6月19到29日都在台北，中间住在旅馆，也去过酒吧、餐厅、健身房，接触者非常复杂。
  <br/>
  <br/>
  台大公卫学院副院长陈秀熙在报告中指出，这名工程师推估暴露时间在6月下旬，与他在台北的时间相符。若再晚一周检测，恐怕就会被遗漏。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  比利时工程师被检验出“阳性”，台湾仍然不愿对是否为“本土”病例骤下结论。在此之前，台湾在7月初也发生一起日本女留学生返日被检验出“弱阳性”，核酸检测CT值介于37、38的数值(台湾标准35为阳性，美国标准为40)。 日本女性到底是不是在台湾感染，仍有悬念；7月28日一名泰国移工从台湾出境返回泰国确诊。指挥中心在1号表示，“确诊泰国移工的二采结果，目前还没有收到，不过现在已经知道，他女友的检验结果是阴性。”
 </p>

 </p>

 <p>
  陈秀熙强调，如果已经看到疫情上升，再来做境外筛检，那时候为时已晚。“我们现在如果里面有外来的感染者，被接触的人有被感染的风险。包括我们对出去(境)这种高危险群，都该被做检测，免得每次出去到其他国家被检测出来。”
  <br/>
  <br/>
  台大医院品质管理中心主任陈世英也认为，边境在现有机制下必须考虑调整加强，特别面对的是越来越严重的国际疫情，外面一直想冲进来的病毒越来越多，要加强防卫，才不会造成社区的疫情。
  <br/>
  <br/>
  日前中央流行疫情指挥中心指挥官陈时中曾在记者会亲自说明，强调核酸检测（PCR）在症状出现最多前2天，才能检验到病毒，潜伏期检验容易发生伪阴性，如若实施入境全筛将恐有35%感染者进入台湾。
  <br/>
  <br/>
  陈世英则解释，所有检查都会考虑到“伪阳性”、“伪阴性”。他希望大家记住，任何一个检查，不会只以检查本身作为判别依据，检查的科技有相关配套的规划，科技能帮助检疫诊断更精准。
 </p>

 <p>
  蓝营重量级地方首长、新北市长侯友宜2日也公开呼吁，在台湾大家要安心，国内的疫情要锁住，前面普筛阻绝于境外，绝对是个很重要的关卡。
  <br/>
  <br/>
  行政院长苏贞昌4日回应表示，台湾一直的措施就是边境风险要严管、国内松绑。过去这个作法也让台湾在各种评比上都被肯定防疫做得最好。“至于外国又发生第二波疫情等，我们都密切注意，应该要怎样应对、怎样做，我们都尊重相关指挥中心与专业团队、专业判断。”
  <br/>
  <br/>
  台湾社区到底有无潜在感染个案 台大月底解谜
  <br/>
  <br/>
  台湾能保持110天“零本土”病例实属难得，不过，这是否意味台湾的社区没有潜伏个案，一直都没有科学数据验证。
  <br/>
  <br/>
  台大公卫学院与彰化县卫生局合作，针对过去武汉肺炎病毒在彰化县内的感染情况，从六月开始展开“万人血清抗体检测”。
  <br/>
  <br/>
  陈秀熙在受访时提出，过去因为境外移入个案的居家检疫过程中，一直不是由检测为主的方式，对于这些没有症状的个案，没有办法检测出来。在台湾的社区一直都可能有潜在感染的个案，只是个案的比例有多大，就是要看抗体血清的调查。目前已检测四千多人，结果预计在8月下旬公布。

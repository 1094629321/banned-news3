 <p>
  广西维权律师覃永沛被控“涉嫌颠覆国家政权”一案移交法院，近日有望开庭之际，其外甥却遭山西警方拘留。家属怀疑这是当局在案件开庭之前，打压覃永沛的一种手段。
  <br/>
  <br/>
  覃永沛的外甥朱性铭上月出门后一直音讯全无。正当他在广西的家人感到彷徨之际，周四接到来自山西潞城市公安局书面通知，说朱性铭涉嫌开设赌场，目前已被刑事拘留。
  <br/>
  <br/>
  覃永沛的妻子邓晓云说，家人收到通知书后马上致电山西公安，但对方拒绝透露案情。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  邓晓云：“这个罪名非常莫名其妙，因为他是在火车上被抓的，开赌场不是当场抓的吗？而且我从来没听说过他赌钱，他做的是室内装修跟房产中介生意。我就没听说过他跟赌有关，他姐昨天（周四）打电话到公安局问，公安局明确表示是在火车上抓人的，具体案情警察不告诉她，叫她找律师联系沟通。”
  <br/>
  <br/>
  去年10月，覃永沛被南宁市警方带走，随即被刑事拘留。其后朱性铭在舅父开设的法律资讯服务公司帮忙，估计因此被当局盯上。
 </p>
 <p>
  邓晓云：“他是被抓以后，他外甥才进来帮忙的。有顾客上门要接待，律师来了他也会去接送。所以说，他也成为眼中钉了。他外甥早就被监控了，他找来的律师也都被国宝警告、威胁。他们可能以为外甥在替我办事，然后才在路上抓他的，可能是路经山西的时候被警方上火车抓他下来。”
 </p>

 <p>
  <br/>
  覃永沛案近日已移送法院。据了解，当局的卷宗根据覃永沛在网上发布的数以万计推文把他入罪。家属却相信，覃永沛之所以被控煽动颠覆国家政权，是由于他举报桂林市公安局长仇祖和以及七星公安分局公安胡凯，并把有关材料上载到互联网，因此遭到报复。
  <br/>
  <br/>
  代理律师李贵生拒绝评论朱性铭被抓是否与覃永沛有关，但相信两起案件会独立处理。
  <br/>
  <br/>
  李贵生：“那小伙子我见过，可能是个20多岁的小伙子吧。我觉得各归各吧，每个人对自己的行为承担责任吧。”
  <br/>
  <br/>
  律师近日忙于为覃永沛案阅卷，估计最快本月会开审。
  <br/>
  <br/>
  李贵生：“疫情的影响很大，跟没有疫情的时候很不一样，现在中国的看守所都
  <br/>
  不愿意把人从看守所带出来，为了防止病毒感染，通过视频开庭效果不好。”
  <br/>
  <br/>
  以敢言著称的覃永沛两年前被注销律师牌照后，连同多名维权律师对外提供法律资讯服务。在他失去自由后，当局不断对家属施加压力，包括传唤他们问话。外界估计，这与当局未能掌握足够罪证把覃永沛入罪有关。

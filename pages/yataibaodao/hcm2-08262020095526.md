 <p>
  美国在台协会(AIT)26日与台湾的外交部及国家通讯委员会共同举办美台“5G共同宣言”论坛，美国在台协会台北办事处长郦英杰重申美国国务卿蓬佩奥提出“5G干净路径”的倡议，不使用像华为、中兴电讯这样不受信赖的通信厂商。
  <br/>
  <br/>
  美国在台协会台北办事处长郦英杰26日在出席“台美5G共同宣言”论坛指出，去年5月来自全球30多个国家的政府官员，以及欧盟、北大西洋公约组织和产业界的代表，一起讨论5G建设策略时，都认为各国在评估5G厂商时，必须将国家安全、经济和商务纳入考量。会后由捷克的会议主席公布“布拉格提案”，提供各国在设计、建构和管理其5G基础设施时可供参考的建议和原则。郦英杰对台湾认同这些重要措施表示欢迎。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  郦英杰强调，政府和电信商目前就5G部署所做的选择，将深深影响未来数年，甚至数十年。各国需要对5G设备和软体公司怀有信心，相信他们不会威胁其国家安全、隐私、智慧财产权或人权。
 </p>
 <p>
 </p>

 <p>
  郦英杰：“如果电信商听从中国等威权政府的指示，信任将不复存在，因为中国缺乏独立的司法机构或法治制度，对隐私上有限的保护会造成安全漏洞。”
  <br/>
  <br/>
  郦英杰提到美国国务卿蓬佩奥在今年4月29日宣布，美国国务院将开始要求所有进出美国外交机构的5G网路流量，都需要经过一个“干净路径”。“不使用极度有问题、不受信赖的通讯商所提供的传输、控制、运算或储存设备，例如，华为和中兴通讯。”
  <br/>
  <br/>
  郦英杰提出，“5G干净路径”体现了最高的安全标准，避免不受信任的高风险厂商借机破坏、操纵或拒绝服务公民、金融机构或关键基础设施。他很高兴台湾是“5G干净路径”倡议的成员。
  <br/>
  <br/>
  <b>
   吴钊燮：资安即国安 台湾加入干净路径
  </b>
  <br/>
  <br/>
  台湾外交部长吴钊燮提出，5G的运用一方面驱动产业创新，另外，也对资安带来新挑战。“资安就是国安，为了维护国家整体安全，台湾保持高度资安意识，在布建5G的资安考量上，达成超前部署，完全排除具有资安疑虑的软、硬体设备和服务。”
 </p>
 <p>
 </p>

 <p>
  吴钊燮称，台湾因为有这样的预先准备，可以很有自信地说，台湾是各国推动资安合作的最佳伙伴。台湾也期待与各国，尤其是理念相近国家共同努力，建构安全稳健可信赖的5G通讯网络。
  <br/>
  <br/>
  吴钊燮表示，台湾表达加入5G“干净路径”(Clean Path)倡议的意愿，美国政府也将台湾五家电信产业列入“干净5G”清单，将台美资安合作制度化。吴钊燮说，台美是紧密的全球伙伴，双方会在既有合作平台进一步深化友好合作伙伴关系。
  <br/>
  <br/>
  <b>
   台湾面临复杂资安威胁
  </b>
  <br/>
  <br/>
  国家通讯传播委员会(NCC)主委陈耀祥致词时表示，台湾正面临国际非常复杂的资安威胁。在去年2019年台湾接到来自国际资安的通报有3千多件，包括恶意程式、钓鱼网站等相关情报。陈耀祥指出，“资通安全已经是国安议题，5G网路更是国家关键设施。”台湾的国家安全会议、行政院资安处还有NCC组成“资安铁三角”，成为台湾网路安全的第一道防线。
  <br/>
  <br/>
  美国在台协会与中华民国驻美国台北经济文化代表处(TECRO) 发表5G安全共同宣言提及，美、台透过“全球合作暨训练架构”(GCTF)紧密合作，提升全球对5G网路风险的认知，并期盼能与理念相近的伙伴持续合作，共同发展合适的5G标准、准则与最佳实务作为。
  <br/>
  <br/>
  <br/>
  自由亚洲电台记者黄春梅 台北报道 责编 许书婷 申铧 网编 瑞哲

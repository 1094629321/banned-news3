 <p>
  台湾现任副总统陈建仁将在5月20日卸任。对于中国大陆武汉肺炎疫情再起，陈建仁祝福能尽早控制好不要再爆发。而对于教廷与中国交好，陈建仁引圣经表达了解教宗对照顾2千多万中国天主教徒的渴求，“丢了一只羊也要放下99只羊去寻找那一只”。
  <br/>
  <br/>
  陈建仁是中央研究院院士、流行病学家，台湾首位无党籍和天主教徒的副总统。2019年3月蔡英文总统争取连任之路，出现党内挑战者赖清德。陈建仁主动表态完成阶段性任务后就会卸任，让蔡英文有更大空间挑选副手，被认为避免了民进党内隐然的风暴。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  陈建仁下周卸任后，选择到中央研究院担任特聘研究员。因续任公职，依法不可以获得月俸18万元（4万多元人民币）卸任礼遇。虽然外界质疑中研院薪水比卸任礼遇较高，陈建仁14日与媒体茶叙表示，他到中研院的薪资是院长决定，不在他，而他很难形容自己对学术研究的疯狂程度。
 </p>
 <p>
 </p>

  <b>
   为追求终身挚爱放弃副总统礼遇
  </b>
  <br/>
  <br/>
  陈建仁：“当你活到一个年纪的时候，薪水不是你最重要的考量。最重要的考量是，在你人生的过程当中，你坚持的、你最爱的是什么？”
  <br/>
  <br/>
  陈建仁说，他最爱的就是做研究，看到有研究有成果时，那种喜乐不是金钱和名誉所能比较。他以科学家牛顿为例，每次牛顿发现一个新的成果，就高兴的不得了，牛顿曾说，这辈子自己就像喜欢收集海边美丽贝壳和小石头的小孩子，只要有些收集就感到无比欢欣。
  <br/>
  <br/>
  陈建仁：“相对牛顿来说，我是很渺小的，我的研究只要是捡到海边一两颗美丽的砂粒，我就会很高兴。学术研究者追求的就是对新知探求的喜悦，这有一点像是名侦探柯南在解决一个凶杀案时的兴奋，那个兴奋说老实话，是我所喜欢的。”
 </p>

 <p>
  <br/>
  任内曾三度出访教廷会见教宗，如何看待教廷与中国逐渐走近的关系？陈建仁指出，教廷是为了让所有在大陆的天主教徒，获得信仰自由的保障。
  <br/>
  <br/>
  <b>
   理解天主教教宗的苦衷
  </b>
  <br/>
  <br/>
  陈建仁：“就好像圣经里讲，如果你有100只羊，如果掉了1只羊，还是会放下99只，把那只找回来。所以站在牧灵的（角度来说），教廷一定要去照顾2千多万天主教徒在中国的情况，我们表示我们有体会这个情况，教廷跟中国的互动，我也蛮相信他们就像好的牧羊人一样，不会放弃任何一只羊。”
  <br/>
  <br/>
  过去四年，蔡英文赋予陈建仁的艰巨任务，包括协调军公教年金改革、同性恋婚姻立法、武汉肺炎防疫工作等。在婚姻平权方面，当时社会上很大的反对声浪来自基督教、天主教教徒，而陈建仁又是虔诚的天主教徒。
 </p>
 <p>
  <br/>
  <b>
   推动婚姻平权受责难、攻击
  </b>
  <br/>
  <br/>
  对此，陈建仁坦言，“蛮多责难、攻击、谩骂都会出现，太太信箱常收这样的讯息。但是，我有一个坚持的信念，就是所有人都有追求幸福的基本权利，而婚姻平权就是应该这样做。教会有教会的观点，可是站在人权保障上的观点上，我们应义无反顾的勇往直前。”
 </p>

 </p>

  <br/>
  而在年金改革上，陈建仁提到，他有美国友人感到台湾能完成年金改革非常不容易，这在美国也是相当困难。陈建仁感谢并向所有军公教人员致歉。
  <br/>
  <br/>
  17年前台湾爆发SARS时，陈建仁接下卫生署署长的重担。这次新冠肺炎疫情爆发，陈建仁幕后督军。他说，每一位台湾人民，都是燃烧自己、点亮台湾，也都是抗疫英雄。今天台湾抗疫成绩能够闪亮，就是2370万人民、2370万根蜡烛的烛光，所闪烁出来的光辉。谢谢全台湾人在他卸任时送给他最好的礼物，就是“武汉肺炎防疫成功”。
  <br/>
  <br/>
  路透社提问，台湾无法进入世卫，中国是一大阻力，是否想对中国说些什么？
  <br/>
  <br/>
  陈建仁表示，现在全世界受到武汉肺炎影响，很重要要解决问题的方式是“公开透明”。台湾疫情可以控制得好，就是因为公开透明。在自由国家一切透明的情况下，台湾的指挥中心指挥官能得到民众信赖，比较容易团结在一起。但是到现在，还没有看到太多中国武汉肺炎在科学上详细的数据，包括多少个案曾被检测？检测结果多少人是阳性？都看不到。
  <br/>
  <br/>
  <b>
   陈建仁呼吁中国对疫情透明公开
  </b>
  <br/>
  <br/>
  陈建仁认为，全世界若要团结对抗武汉肺炎，各国应该把自己的作为、数据，和未来要怎么做，公开让大家知道，看看彼此怎么从失败中学习改进。
 </p>

 </p>

  <br/>
  陈建仁提到：“我晓得中国最近武汉肺炎又在武汉开始，而且在吉林出现案例，我也在这里祝福他们能够尽早控制好武汉肺炎，不要再有疫情大爆发的现象。世界上任何一个国家有武汉肺炎病例，就是对台湾的威胁，所以也希望全世界都控制好武汉肺炎的疫情。”
  <br/>
  <br/>
  评估台湾今年是否可能参与世界卫生大会相关会议时，陈建仁说，关键还是在于世卫组织干事长和同仁是否能够尊重专业、中立性，及尊重全世界人类健康的人权。“如果过度考虑政治问题，机会就不大。”陈建仁呼吁，让台湾参加世卫组织和世卫大会，是台湾、世界卫生组织和全世界，最好的三赢策略。
  <br/>
  <br/>
  至于卸任后是否会到中国交流有关新冠肺炎的情况，陈建仁说，卸任副总统基于国安必须要遵守国家相关规定。他如果有出国计划，必须要有总统府同意，他虽然不接受卸任后的年薪礼遇，但在国安上仍需遵守规定。

 <p>
  <b>
   4年最大收获：对台湾更具信心
   <br/>
  </b>
  <br/>
  陈建仁坦言，过去他对台湾的未来感到不确定。这四年最大收获，是他相信台湾会往愈来愈好的方向发展。他也观察蔡英文处事更为圆融和谐，是人民之福。
  <br/>
  <br/>
  陈建仁在脸书提到，当年自己35岁升上教授，是光复后医学院最年轻的教授。他很高兴告诉父亲，父亲却说“有什么稀罕”，就挂上电话。后来父亲提醒他稻穗愈结实愈往下垂，做人要平淡、平实、平凡，应该是认为他太骄傲，要他学习谦虚。
  <br/>
  <br/>
  陈建仁在脸书写道，“2016年在689万选民的支持下，我很荣幸作了台湾的公仆！洪山川总主教曾经告诉我教宗方济各的名言：‘真正的权力就是服务’，好领袖应该像好牧人，带领羊群到青草地、清溪畔，用牧杖和木棒保护羊群，免受豺狼的侵害。我就像印度加尔各答的圣德肋撒姆姆（特蕾莎修女）（St. Mother Teresa）所描述的：‘我是上主手中的小铅笔，祂在思索、祂在撰写。祂作了每一件事，有时候还真是困难，因为这是一支折断的铅笔，祂还必需把它削得更短！’现在是该回到中研院作为上主手中的小铅笔，继续作研究、发表论文、写书的时候了！”

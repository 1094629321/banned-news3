 <p>
  台湾独派政党和公民团体指出，罢韩案的成功可见台湾民众拒绝亲中，呼吁蔡英文应全面检讨马英九政府时期至今未修正的中资规范，防堵漏洞。
  <br/>
  <br/>
  “全面检讨中资规范！反制渗透刻不容缓！”
  <br/>
  <br/>
  台湾独派政党基进党和公民团体8日召开记者会指出，马英九政府以“两岸经济一体化”、“从经济整合走向政治统一”的思维订出宽松的中资来台规范，其中，《中资来台许可办法》已六年未修正，《开放中资业别》更是已九年未修正。他们要求“马规蔡不应随”，国安会、经济部应检讨中资规范。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  <b>
   制造业95% 、服务业和公共建设逾50%已开放中资
   <br/>
  </b>
  <br/>
  经济民主连合召集人赖中强指出，“根据陆委会网站的统认资料，我们所有的制造业中，有高达95%的类别项目开放中资投资，包括敏感的资料处理网站代管、大众交通运输业电脑系统设计服务业、新市镇开发、积体电路制造业、半导体封装及测试业、电子及半导体生产用机械设备制造业、被动电子元件、发电输电配电制造、工具机等涉关国安的敏感项目，与影响台湾在世界产业链地位的关键产业。这些难道不需要检讨吗？我们真的要开放中资投资这些行业？如果不开放，为什么不干脆跟大家讲清楚，把这个法规的开放项目废掉？”
 </p>
 <p>
 </p>
 <p>
  <div class="image-inline captioned" style="width:1276px;">
   <div style="width:1276px;">

  <br/>
  赖中强还说，服务业对中资开放部分，已经有50%当年是马英九政府用行政命令开放的。当时服贸协议公听会上争执非常多的有批发零售业、汽车货运业、仓储业、电信业、电脑系统设计服务业、入口网站、资料处理业。
  <br/>
  <br/>
  赖中强表示，更离谱的是关于公共建设、促参法，就是BOT方式引进民间资金参与公共建设。马政府任内认为可以引进中资，参与以下这些公共建设，包括公立文化机构及设施。
  <br/>
  <br/>
  赖中强说：“中正庙（中正纪念堂）的戏剧院、国家音乐厅要不要开放中资BOT？大众捷运系统、新市镇开发、工业区开发，甚至依法指定的古迹、登陆的历史建筑及其设施、轻轨捷运缆车系统、停车场、公路、快速道路，我们真的要开放中资来台湾参与BOT？”
  <br/>
  <br/>
  赖中强指出，请立法委员去问行政院长苏贞昌、交通部长林佳龙，要开放这些给中资吗？如果答案是否定的，法规为什么不废掉？马政府时期离谱的决定为什么不划上句点？蔡英文第二任期应该积极解决。
 </p>
 <p>
  <b>
   大同公司不获人们信赖
  </b>
 </p>
 <p>
  台湾基进党立委陈柏惟也质疑，蔡英文2016年上台未检讨这些法令。第二任以817万高票当选，应加速处理危及国家安全的法律，且在北京通过港版国安法之后，应处理港资背后的中资渗透问题，全面阻绝中国政府对台湾民主运作的不当干预，否则令人民质疑换民进党上台理由何在？特别像大同案、红媒为何不处理？
 </p>
 <p>
 </p>
 <p>

 </p>
 <p>
  <br/>
  陈柏惟说，被外界质疑有中资介入的大同公司将在30日进行股东改选。大同公司承包了台湾的全国户役政系统、内政部移民署入出境电脑设备暨相关硬体建置及维护案、个人生物特征系统建置及维护案等，甚至还有台湾一级情报机关的电子公文系统。虽然目前外界无法确认是否有中资，但大同案已造成民众普遍不信任，凸显现行法规的不足。
  <br/>
  <br/>
  陈柏惟：“蔡政府难道不了解国际媒体看台湾总统选举，是看民意普遍是‘亲中’或‘亲台’立场？连美国《华尔街日报》和《纽约时报》在罢韩案之后都报导‘台湾拒绝亲中的市长’。”
  <br/>
  <br/>
  陈柏惟说，当韩国瑜选上高雄市长，国中小学就有中国歌舞团去表演，高雄市官员和议员去参加一个“假的小三通”。所以不要以为防范中资只是中央政府的事，如果连美国日本都在检讨中资，台湾是不是应更严格？
  <br/>
  <br/>
  台北大学犯罪学研究所助理教授沈伯洋指出，从肺炎疫情到罢韩投票通过后，他发现有越来越多中国帐号，在网路上不断分化台湾族群，现在最主要的就是要防范“中资”来台。
 </p>
 <p>
 </p>

 <p>
  <b>
   <br/>
   台学者：网红、直播主都成分裂工具
  </b>
  <br/>
  <br/>
  沈伯洋说：“从昨天到今天的观察，可看见有越来越多的中国（社媒）帐号（开始进行）分化，只针对特定族群，产生仇恨性言论，希望有更多的抗议事件在台湾发生。”
  <br/>
  <br/>
  沈伯洋还提到，中国透过三个方面，包括监控、人力整合和个资搜集在台湾进行渗透。例如中国海康威视等监视器不断输出技术。另外，沈伯洋说，最近发现中共在台湾布建67条人力整合线。还透过台商、特定政党对宫庙进行渗透。另外中国政府对网路上的直播播主、Youtube网红的渗透，更不遗余力，将这些网红网罗到秘密社团，他们又制造更多台湾裂解点，这也是为何这些仇恨性言论在所谓特定“同温层”才看得到。
  <br/>
  <br/>
  沈伯洋说：“个资收集要完整才可对人群做分类，这就要靠很多公司在台湾营运，例如影音串连平台，可以搜集使用者很多偏好、APP、网页。如果能做这种个资收集，就可以知道消费者喜欢去的餐厅、地点、生活 作习。例如大同股票如果资金来源开放中资，不用骇客攻击，一天之内可以把台湾的个资带走。”
  <br/>
  <br/>
  沈伯洋强调，关于中资开放的规范，太久没有进行检讨。如果蔡英文第二任都不做，就是拱手把台湾的安全让给中国。疫情稳定之后，台湾将从最安全变最危险的地方。
 </p>

 <p>
  捷克参议院议长维特齐（Miloš Vystrčil）访台效应发酵，传出有捷克知名钢琴制造商遭报复性退订单。捷克总统则批评维特齐访台是“幼稚的挑衅”。维特齐则亲上火线在CNN节目上与捷克总理巴比斯辩论，捍卫台湾行是做对的事，并揭发总统齐曼三大谎言。
  <br/>
  <br/>
  捷克参议院议长维特齐89人访问团，结束6天访问台湾行程于4日离台。降落捷克机场受到数十民众接机欢呼。台湾不分朝野也视为是台捷一次成功的国会外交，连中国外长放话要让维特齐付出重沉重代价，也获得德、法、斯洛伐克、欧盟、美国白宫等反驳。不过，传出维特齐返回捷克后，有北京客户向捷克知名钢琴制造商佩卓夫（Petrof）取消一笔总值530万捷克克朗（约人民币160万）订单，维特齐则驳斥根本没有这订单。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  维特齐返回捷克后，遭亲中的总统、总理公开批评。捷克总统齐曼（Milos Zeman）6日接受捷克电视台Prima访问，形容维特齐访台是“像个男孩般挑衅中国”、“幼稚的挑衅”，可能会对国内企业造成伤害，他将不会邀请维特齐参加捷克的高层外交政策官员会议。
 </p>

 <p>
  在齐曼发表声明之后，维特齐6日亲上火线，与总理巴比斯（Andrej Babiš）同台接受CNN与当地电视联合制作的节目专访。根据台湾中央社报导，巴比斯批评，维特齐的台湾行将政治与商业混为一谈，并违反了一中原则，他也不欣赏维特齐自认能“拉直捷克共和国的脊椎”；巴比斯说，因为维特齐，现在捷克正等着中国的反应。
  <br/>
  <br/>
  维特齐强调，没有书面文件禁止捷克官员前往台湾，但捷克政府的政策却依循中国解释的一中原则，问题是，独立的捷克是否有义务受中共指示，规范谁才应该是商业伙伴？维特齐认为，若捷克有人与“非民主国家”建立长期、密切的商业联系，甚至给予支持，那种国家除了使捷克成瘾之外，“根本没有用处”。
  <br/>
  <br/>
  维特齐指出，齐曼总统总共说了3个谎言，第一个是，他前往台湾之前，与齐曼讨论过“中华民国”与“中华人民共和国”的议题，齐曼仅说“不建议去台湾”，并没有表示“不准”；第2个谎言是，齐曼总统对会议的见解有错，也不真实。
  <br/>
  <br/>
  维特齐说：“最令我遗憾的第3个谎言，是指控我称『台湾是一个独立国家』，这不是真的，我在台湾就已经说过，我不会干涉（统独议题），这是捷克政府的事。”维特齐也强调：“我坚信访问台湾并不违反我们的国家政策，但我也坚信我们绝不能屈服于『老大哥』。”
 </p>

 <p>
  对于传出有捷克钢琴制造商遭到退单，东吴大学中东欧研究中心执行长郑得兴接受自由亚洲电台采访分析，“重点可能也不是这一笔订单，而是『我会让你失去中国市场』这个概念。”
  <br/>
  <br/>
  不过郑得兴认为，中方想透过“经济制裁”效果应该有限，中国一年平均约30万游客到捷克旅游，中国政府可能会想采取禁止中国观光客去捷克观光的手段，日本、韩国、台湾，都受过这种对待。
  <br/>
  <br/>
  郑得兴说：“很不幸的，刚好碰到了（新冠肺炎）疫情，这招也没有制裁到痛点。过去几年，中国虽然号称捷克的第二大贸易伙伴国，可是这个贸易逆差太大了，中国人赚捷克的钱，比捷克人赚中国的钱差很大，差不多每一年捷克对中国的贸易逆差，达到一百多亿美元之多，所以经济制裁能不能达到痛点很难说，参议长维特齐应该都是有算过的。”
  <br/>
  <br/>
  前台湾驻捷克大使汪忠一接受自由亚洲电台采访也认为，“维德齐访台全世界都很关注，尤其引起欧洲各国的讨论，他后续效应当然很大，总统齐曼因为过去一直非常亲中，他当然要做一个表态，我想这是不得不对他自己的立场，或是一个看法的表态而已，所以你看齐曼这是类似一个情绪上发言，没有什么实质上的意义 。”
  <br/>
  <br/>
  媒体报导，齐曼是中共70周年阅兵唯一获邀出席的外国元首，汪忠一提到，齐曼2013当选总统，推动跟中国贸易，身边幕僚跟中国关系密切，尤其相当倚重中国华信集团的董事长叶简明作为他的经济顾问，也作为捷中经济合作平台，引进中方资金。在2016年3月，习近平到捷克访问，双边关系达到高潮，当时习近平承诺对捷克扩大投资，北京与布拉格市签订姐妹市，还开了上海、北京、海南、成都等四条中国到捷克直航航线，将每年30几万中国观光客倍增至60万，为布拉格观光业带动商机。

 <p>
  但汪忠一说，最后中国大陆并没有落实承诺对捷克的两百多亿投资，捷克人民心中开始存疑。尤其中国大陆都以并购为主，包括并购啤酒厂、足球队、传媒等，并没有为捷克带来就业机会，反观台湾的鸿海、宏碁、华硕等企业纷纷前往设厂，创造数万个就业机会。
  <br/>
  <br/>
  汪忠一认为，维特齐此行，如果能促成中华航空台北、布拉格直航，将具有重要指标意义，其他产业投资一步一步落实，更能显示台捷经贸具体绩效。
  <br/>
  <br/>
  汪忠一提到，总理巴比斯属于主要的执政党、“2011不满行动联盟（ANO） ”，但并没有超过国会半数，在组阁上遭到困难，当时总统齐曼就提供协助，帮他找到共产党（KSCM），另外还有社会民主党（CSSD），共同组成联合内阁，所以总统虽为虚位元首，对总理具有影响力。
  <br/>
  <br/>
  汪忠一说，捷克众议院是下议院，对行政机关具有预算审核的权责，参议院则具有否决权，参议院81席，维特齐所属为公民民主党（CDS），目前是第二大政党，布拉格市长贺瑞普（Zdeněk Hřib）所属海盗党近来声势上涨，与公民党争二、三之间。
  <br/>
  <br/>
  郑得兴说，值得观察的是，捷克参议院每2年改选三分之一，在下个月举行，众议院选举明年十月选行，及2023年总统大选，都是维特齐访台效应的几个观察点。
  <br/>
  <br/>

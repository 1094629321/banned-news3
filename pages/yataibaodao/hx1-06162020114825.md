 <p>
  在中国北京著名的艺术家聚集区宋庄镇，最近有数百名艺术家收到镇政府发出的2个月限期搬离的通告，另有人收到律师函要求更改租赁合同的通知。他们接受自由亚洲电台采访表示，政府承诺50年租期，怎么说收回就收回。当年号召艺术家进驻宋庄的栗宪庭向本台表示，会尽全力寻找解决办法。
  <br/>
  <br/>
  2017年北京强势驱离所谓“低端人口”的噩梦，最近降临在俗称“画家村”的宋庄。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  自由亚洲电台采访住在宋庄镇双塘的一名不愿具真名、化名“星光”的艺术家说，东区、双塘街、北塘、小堡环岛以南等整个区域都收到了通知：“很多地方，几乎宋庄所有工作室都收到贴条说在两个月内要拆除，但是什么情况还不太清楚。我估计几百户肯定是有的。原来签50年（租期），当地政府不承认，想把这50年合同给修改了。没人同意要变20年，肯定不同意。”
 </p>

  星光提供本台的政府公告显示，发出的单位是宋庄镇人民政府。公告说，“京国土通州分局明确告知没收你单位（个人）在该违法占用土地上的建筑物及其他设施。自本通知下发日起60日内自行清理个人物品并搬离此建筑，逾期未搬离造成的损失由你单位（个人）自行承担。”
  <br/>
  <br/>
  星光提到，当年要不是前北京市长王岐山、刘淇，到现任的蔡奇，都鼓励艺术家进驻宋庄，打造国际艺术村，不可能发展到今天这么大规模。如果没有政府允许、鼓励，怎会有这么多画家、艺术家拿一生积蓄到这里打造工作室，甚至有人花几千万人民币盖了美术馆。去年政府才说宋庄被规划成艺术小镇，今年就要拆迁，朝令夕改，真相完全被扭曲。

  此外，宋庄镇中心小堡区不愿具真名、化名“无言”的艺术家接受自由亚洲电台采访表示，最近其他村纷纷传阅一张律师函，自家门上也被贴了一张。
  <br/>
  <br/>
  无言：“说房屋合约有问题要修改，6月多少号没说，到小堡村委员会更改合同或重新商议。你如果不去，我们要承担法律责任。”
  <br/>
  <br/>
  无言提供的律师函是由北京嘉润律师事务所发出，名为《变更或修订农村土地的承包/使用/租赁合同的告知函》。内容提及“贵方与小堡村委员会签署的《场地租赁合同》，存在‘租赁期限超过法定期限’等问题。根据小堡村委员会明确要求，请贵方于2020年6月（    ）（空白没有写）前，至小堡村委会，通过签署补充协议或变更协议，加以规范完善，否则法律后果由贵方自行承担。”
  <br/>
  <br/>
  无言提出，2006年小堡村民委员会和他签订的《北京市通州区宋庄镇小堡村文化创意产业园区入园协议书》清清楚楚写着“乙方享有50年的土地使用权”，到2056年，如今才过14年就“毁约”。

  <br/>
  <br/>
  无言说，邻居们都收到律师函要作废原先的合同，估计有约300名艺术家被要求更改合同，形同“割韭菜的刀和砍人的刀都磨好了”。无言也提到，在小堡村艺术工厂路一带、长青藤艺术村六层楼，数百户更收到被宋江镇人民政府发出限期2个月搬迁的通知，他们也大都是面临50年使用权被跳票的艺术家。

 <p>
  <b>
   宋庄艺术家：没有土地私有权 人人自危
  </b>
  <br/>
  <br/>
  无言说，2017年北京驱赶所谓“低端人口”，当时他也去现场看了，非常震惊，不相信是真实发生的事，当时还自认有个房子比较安全，直到今天才发现，任何人没有区别，都在危机当中，不是说你是一个年轻打工族，租了一个二百块钱的房子就不安全，你可能二百万的房子也都不安全。
  <br/>
  <br/>
  无言说，艺术家们的诉求就是能够维持既有合同承诺，且不追缴新增费用。

  <br/>
  近日网上疯传宋庄被拆，有人只穿一条内裤就被赶出去，和抗暴警察对峙的照片，以及对话截图。在九零年代号召艺术家进驻宋庄的艺术家栗宪庭告诉自由亚洲电台，网上很多东西不真实，宋庄还没出现大规模的动手，正想办法解决。
  <br/>
  <br/>
  栗宪庭提到，从一九三零年代开始，共产党都是用这样的土改模式，要把土地所有权捏在自己手里。那是集体土地，租用土地在某种意义上也不是完全合法。但是，当年如果没有官方允许和领导支持，艺术村是无法建立起来的。
  <br/>
  <br/>
  栗宪庭：“说到底，希望找到解决的方法，不要强拆，强拆会造成非常多的浪 费。（这种做法）也不是从宋庄开始的，还没有弄清楚要拆迁或重新订合同这消息、通知从什么地方来，都还没弄清楚，现在还在和有关单位接触 。”

  <b>
   宋庄艺术村始作俑者被警察监控
  </b>
  <br/>
  <br/>
  谈到20多年前号召艺术家到宋庄驻村，栗宪庭说，中国很多事情很复杂，当时宋庄一带落后贫穷，基本上村民很少，艺术家只有少量来住。之后，愈来愈引起外地艺术家注意，他们又做一些美术馆、商业画廊的实验、希望建立一个创作生态环境，包括艺术家创作、学术研究、展出机构，以及画廊销售的商业机构，应该是一个完整的链条，这样才可以让艺术家安心生活。每一个城市都有类似的做法，因为艺术家不停地扩张，流落社会，没有找到工作，他们聚集就成社会问题。艺术区建立这样的艺术生态，实际是为解决社会问题的。
  <br/>
  <br/>
  栗宪庭提到，这一届领导做下去，换另一届领导，可能不这样想，证件再齐全，换个官员可能就全拆了。在其他地方也看到，最根本的问题就是土地不能私人拥有。几年就换领导，没有长远规划，早期领导很好，后来的领导不明白当年的用心，就瞎折腾，而且每个地方政府、乡镇政府，各个都有自己的小算盘。
  <br/>
  <br/>
  多名宋庄艺术家称当年政府承诺土地使用权“50年不变”，现在却反悔。栗宪庭指出，50年不变和现在要改签20年之间的关系怎么回事，还没有弄清楚。
  <br/>
  <br/>
  被问到艺术家是否成立自救会时，栗宪庭说，“不可能成立组织，中国成立组织是敏感政治问题，不可能像香港那样组织起来上街。否则对我们的镇压不是香港警察那么手软的。”
  <br/>
  <br/>
  栗宪庭坦言，他对有些情况不了解，是因为在他号召艺术家到宋庄后，不到几年，官方对他也不信任，不再让他参加任何会议。

  <br/>
  <br/>
  张振宇提到，中国大陆制度是国家持有土地，土地不能买卖。在农村，农民或是所谓的村委会、村级公宣单位，可合法拥有土地使用权70年。有些农民，或是村委会为了地方发展，私下将土地使用权卖给艺术家，有所谓“小产权”、“大产权”的说法，游走法律边缘。但一旦换了领导，政策可能完全推翻，或是更上层的市级、镇级、中央政府有其他土地规划，命令下去，村委会或“农民房东”就得赶人，金钱惹祸引发纠纷。
  <br/>
  <br/>
  张振宇还说，若不是艺术家来打造宋庄，宋庄土地价值不会在这二、三十年间上涨了数百倍。过去十万块买到的土地使用权，现在可能五百万才买得到，有些农民、村委会可能想涨价，不想跟画家续约告到法院的争端，也层出不穷，
  <br/>
  <br/>
  张振宇认为，宋庄事件不能单纯看作是政府迫害艺术家，但艺术家认为合同被迫无预警更改，甚至要求被搬迁没收，当然吃亏。另外，也牵涉到赔偿高低、有没有拆迁补偿等问题，相当复杂。艺术家们是否能维护自己的权力，差异甚大。过去其他艺术村如园明园、七八九、草场地，都出现过类似情况。
  <br/>
  <br/>
 </p>

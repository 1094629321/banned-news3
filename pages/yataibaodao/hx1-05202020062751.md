 <p>
  台湾总统蔡英文、副总统赖清德在520宣誓就职。蔡英文在就职演说中强调，2300万台湾人是生死与共的命运共同体，期许上下一心，当一个光荣的台湾人。最受瞩目的两岸关系部份，蔡英文重申“和平、对等、民主、对话”八字箴言，不会接受“一国两制”。蔡英文承诺遵照中华民国宪法、两岸条例，维持台海和平稳定现状，在国家建设工程上，她宣示立法院将成立修宪委员会凝聚共识。
  <br/>
  <br/>
  蔡英文5千多字、25分钟的第二任就职演说，“重中之重”是以新冠肺炎台湾的表现作为主旋律。第一部份“作为共同体的台湾”，她感谢所有台湾人对政府的信任，让世界看到台湾在最不安的时刻也能保持公民美德。“国家的光荣感，生死与共的共同体，这一段记忆，将会存在我们每一个人心中，团结的感觉，就是这个样子。”
 </p>

 </p>
 <p>

  </b>
 </p>
 <p>
  <br/>
  <b>
   光荣感、共同体、团结台湾 是蔡英文凝聚国家意识的关键词
  </b>
  <br/>
  <br/>
  第二部份提到“空前的挑战和绝佳的机会”，蔡英文指出，这次疫情改变了全球政治经济的秩序，重新排列了经济板块，甚至也改变了国际社会对台湾和周边情势的想像。“治理国家，从来不能依赖激情，而是要在变局中，保持冷静、指出方向。过去4年，这一点，我做到了。”
 </p>

 <p>
  蔡英文表示，下一个4年，在产业发展、社会安全、国家安全、民主深化，这4大面向，她会超前部署，让台湾脱胎换骨。
  <br/>
  <br/>
  蔡英文演说中提到“台湾”49次，“中华民国”5次，“中华民国台湾”1 次。最受瞩目的“两岸关系”，在演说中镶钳在第三部份“国家建设工程”中的第三项“国家安全”，其中的第三点“两岸和平稳定”，仅占306字。
  <br/>
  <br/>
  <b>
   蔡英文就职演说 两岸关系放在中后段仅占6%篇幅
  </b>
  <br/>
  <br/>
  蔡英文：“我要再次重申『和平、对等、民主、对话』”这八个字。我们不会接受北京当局，以『一国两制』矮化台湾，破坏台海的现状，这是我们坚定不移的原则。我们也会持续遵循中华民国宪法，与两岸人民关系条例，来处理两岸事务。这是我们维持台海和平稳定现状的一贯立场。”
  <br/>
  <br/>
  蔡英文说，两岸关系正处于历史的转折点，双方都有责任，谋求长远相处之道，避免对立与分歧的扩大。：“在变局之中，我会坚守原则，并秉持解决问题的开放态度，负起责任，也期盼对岸领导人，能承担起相对的责任，共同稳定两岸关系的长远发展。”
 </p>

  <b>
   首抛“修宪”为第二任重点政策 设在民意机构立法院项下先找共识
  </b>
  <br/>
  <br/>
  蔡英文提到，未来4年，在国家建设的工程方面，立法院即将成立修宪委员会，提供一个平台，让攸关政府制度，以及人民权利的各项宪政体制改革议题，能够被充份对话、形成共识。
  <br/>
  <br/>
  蔡英文最后作出感性呼吁。她说，过去70年来，中华民国台湾，在一次又一次的挑战中，越发坚韧团结。“我们抵抗过侵略并吞的压力、走出独裁体制的幽谷，也一度走在被世界孤立的旷野之中，但无论什么样的挑战，民主自由的价值，一直是我们的坚持。『自助助人、自助人助』的共同体意识，也始终是我们的信念。”
  <br/>
  <br/>
  卫福部长陈时中等防疫有功人员，是她仅邀请的数百名宾客中的贵宾。蔡英文向防疫有功人员致敬，她认为，所有的台湾人都是英雄。她和赖清德，很荣幸接受各位托付。
  <br/>
  <br/>
  蔡英文：“挑战只会越来越多。不过，我们是一个在惊涛骇浪中走过来的国家。我们2300万人，是生死与共的命运共同体。要记得过去这几个月，上下一心、紧紧相依、克服难关的感动。中华民国可以很团结，台湾可以很安全，当一个台湾人可以很光荣，可以抬头挺胸、昂首阔步。”

  <b>
   防疫有功人员成为蔡英文点名贵宾  马英九、韩国瑜皆拒出席
  </b>
  <br/>
  <br/>
  蔡英文过去曾经没开口唱“三民主义”歌的纪录，这次将整首“国歌”唱好唱满，连“吾党所宗”这句也没跳过。至于四年前在蔡英文第一任就职典礼，大力主张制定新宪法、拒绝起身唱“中华民国国歌”的总统府资政辜宽敏，果然如他自己的预期，并未获邀参加蔡的第二任就职典礼。
  <br/>
  <br/>
  总统府表示，这次受疫情影响，就职典礼规模缩小，只邀请执政团队、县市首长、驻台使节及防疫国家队有功人员等200多人，未开放媒体采访。
  <br/>
  <br/>
  六都市长唯独高雄市长韩国瑜缺席。国民党主席江启臣、前总统马英九受邀，但拒绝出席。
  <br/>
  <br/>
  <b>
   蔡英文二任演说 有前四位民选总统的影子
  </b>
  <br/>
  <br/>
  蔡英文第二任就职演说，有前四位选民总统的影子。“生命共同体”是李登辉提出凝聚国家认同的核心概念。陈水扁也曾有过八字箴言“主权、民主、和平、对等”，蔡英文的八字箴言没有“主权”，增加“对话”，其他六字顺序不同。蔡英文第一任提出“维持现状”，被质疑与马英九宣示“维持现状”代表“不统、不独、不武”没有差异。蔡英文在第二任演说，则指出她要维持的是“台海和平稳定”现状。

 <p>
  蔡英文第二任就职演说涉及两岸部份，淡江大学中国大陆研究所荣誉教授赵春山接受自由亚洲电台采访说，没有新东西，“她就是求稳定！认为过去4年台海没有什么冲突，她的期待希望维持，过去对岸批评她上台不承认『九二共识』会地动山摇，没有嘛！”
  <br/>
  <br/>
  <b>
   地动山摇没发生 学者：蔡英文求稳定、不挑衅
  </b>
  <br/>
  <br/>
  赵春山提到，台湾电视台TVBS民调，蔡英文整体表现支持度61%，大陆两岸政策就得到51%支持度过半数支持她，非常高，她觉得不需要改变。对蔡来说，当前最迫切是防疫、经济纾困，不是两岸课题。
  <br/>
  <br/>
  赵春山提到，大陆关心的是蔡英文提到修宪部份，蔡英文就职演说再提遵循宪法、两岸条例。“所谓法理台独这方面，对岸根本不需要担心，蔡不会走这个路。（修宪）有关固有疆域不会动，  动了就不需再讲两岸人民关系条例，不会动大陆方面担心的东西。”
  <br/>
  <br/>
  台湾中央通讯社19号晚上8点，连发三条新闻具名引述总统府发言人预告蔡英文就职演说重点，赵春山表示这很少见，明显是蔡释出的善意：“我让你先有准备，我不会让你措手不及，他把它讲的很详细，昨天”。

  <b>
   总统府前一晚提早公布就职演说  学者：向北京释善意
  </b>
  <br/>
  <br/>
  预期蔡英文的演说对岸是否满意？赵春山说：“我们讲什么，对岸从来没有满意过，我认为他虽然不满意，还是可以接受，因为没有挑衅 。台湾问题现在也不是对岸最重要的，你只要不挑衅就好。”
  <br/>
  <br/>
  政治大学名誉教授丁树范接受自由亚洲电台采访，也预料北京基本上会认为蔡英文第二任演说“还是所谓『未完成答卷』，4年前就这么讲！ 当然会不满意，因为蔡没有讲『九二共识』、没有讲『一中』”，不过，习近平面临的问题可能更多，中国大陆现在要集中力量对抗主要敌人，台湾只要蔡英文没有更具挑衅，就摆着。”
  <br/>
  <br/>
  丁树范也提到，绿委日前提案删除宪法和两岸条例前提“国家统一”改“国家发展”，草案已一读通过却“撤案”，某种程度反应蔡英文向北京和美国保证：“只要我当家，我不会丢出这种敏感东西。”
  <br/>
  <br/>
  对于蔡英文八字箴言不同于陈水扁八字中有“主权”两字，丁树范解读，“对等”就隐含“主权”，蔡应想淡化，因为一提“主权”对岸就跳脚。

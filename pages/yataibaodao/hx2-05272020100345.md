  大陆台商在一场座谈会中呼吁，台商每年为台湾赚1千多亿美元外汇，政府应该要更加维护及重视台商的权益。台商认为，台湾应该在美中之间创造左右逢源的利益，不该只压宝一边。台商透露珠三角至少6000名台商因疫情尚未复工，应尽快与对岸协商，建立绿色通道。
  <br/>
  <br/>
  台湾立法院内政委员会27日赴海基会聆听12名台商的心声。台湾投资企业联谊会常务副会长、深圳台协会长陈忠和在大陆创业逾30年，他感叹每年台商的贡献最高达1300、1400亿美元的贸易顺差，大陆的台商跟在世界各地投资台商都一样，是在为台湾经济力做扩展、做耕耘。

 <p>
  <b>
   台湾政府官员允诺保障台商利益
  </b>
  <br/>
  <br/>
  海基会董事长张小月提到，她佩服很多台商在二、三十年前单枪匹马到大陆创业的胆识和魄力，但几十年下来，中国大陆环境确实改变了。
  <br/>
  <br/>
  张小月：“过去的利基现在已经是挑战，再加上美中贸易战、新冠疫情，很多台商想怎样布局、调整业务，海基会、陆委会等政府部门会协助留在大陆的台商，保障其利益。如果不回台湾到东南亚的台商，海基会也会提供相关服务。”
  <br/>
  <br/>
  陆委会主委陈明通致词时指出，陆委会的职能是维护台海和平，让台商在稳定的环境下，把握商机，一起赚大钱。近年中国大陆投资环境出现大幅变化，近期又遭逢美中贸易战及疫情冲击，对全球供应链造成巨大影响。
  <br/>
  <br/>
  陈明通呼吁台商，回台分散投资，降低单一投资及单一生产基地的经营风险。政府相关部门已陆续推出纾困及振兴经济的相关措施；另外，为改善政府对台商服务，已建立与经济部、海基会联系管道，协助台商因应中国大陆情势变化，强化风险管理，做好经营和布局。

  <b>
   台商盼开放航点缩短检疫
  </b>
  <br/>
  <br/>
  立法委员、行政部门和十二名台商之后进行了闭门交流。多位台商会长表示，目前两岸只开放北京、上海、厦门、成都4个航点，现在疫情已逐步获得控制，台商最密集的珠三角及长三角地区直航航班仍未恢复，加上现有直航航点之检疫政策对外地台商不甚方便，造成许多台商迄今返回大陆工作仍不方便，珠三角就至少六千名台商、台干尚未复工。许多企业复工、融资等问题都需要企业负责人亲力亲为，因此希望能逐步复航，并优先考量恢复南京及深圳（或广州）航点，便利台商们返回工作岗位。台商呼吁，随着疫情的趋缓，希望能建立两岸商务人士的便捷检疫措施，能够缩短隔离检疫的时间。
  <br/>
  <br/>
  深圳台协会长陈忠和会后受访表示，与会台商共提出几大诉求，包括两岸可考虑有限度复航或放行商务人士交流，以及台湾疫情后“拚经济”政策应兼顾台商，执政者应改善两岸关系等。陈忠和建议，台湾可考量有限度的与大陆防疫成果较佳的城市，优先推动复航或是缩短商务人士往来隔离时间，并透过协商，明确两岸都承认的检测标准和机构。
  <br/>
  <br/>
  澳门台商协会会长简廷在会后受访也表示，520过后，两岸重启交流时应该少点政治，更多关注在经济，因为在疫情和中美贸易摩擦的影响下，多数台商都面临订单下滑的困境，急需重回工作岗位，重新调整布局，或寻找转型契机。

  <b>
   围绕陆配的暖心小插曲
  </b>
  <br/>
  <br/>
  立法委员考察海基会出现一段插曲。他们在参访过程中，发现海基会办公室有一箱鲜蚵，感到好奇。海基会表示，这是同仁向一名陈姓大陆配偶采买，希望透过团购帮助陆配家庭。
  <br/>
  <br/>
  海基会人员提及，嫁到台湾在嘉义东石养蚵的陈女士，从小出生在广西。她表示，20岁以前未曾见过大海，嫁到台湾后，与夫家共同经营养蚵事业。除了养蚵外，有商业头脑的陈女士也透过网路行销，经营海蚵养殖体验行程，逐渐打响“白水湖蚵学家”品牌。
  <br/>
  <br/>
  另外在新冠肺炎期间，很多陆配与小孩滞留湖北无法返台，海基会协助搭乘包机及定时定点航班返台。海基会透露，有一位陆配姐妹特别带着彰化家里种的葡萄表达谢意，因为品质很好，同仁之后还团购这名陆配家种的葡萄，以行动相挺。

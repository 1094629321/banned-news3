 <p>
  天津市上週新增一起武汉肺炎(COVID-19)确诊本土案例，中国官方甩锅德国，说病毒源头可能是来自德国进口的冷冻猪肘。德国联邦农业部(BMEL)9日则反驳说，根据科学研究，还没有发现因为食用或接触沾染新冠病毒的肉类及肉制品而被感染的先例”。
  <br/>
  <br/>
  中国天津市上週新增一起武汉肺炎(COVID-19)确诊本土案例，但官方却说，病毒源头可能是来自德国进口的冷冻猪肘。不过德国联邦农业部9日立即反驳，强调根据当前对新冠病毒的科学认知，德国猪肘不太可能是导致感染的因素。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  天津在7日通报1例本土确诊个案为冷冻食品公司的卸货工人，官方指出先前在一批自德国进口的冷冻猪肘外包装上检测出呈弱阳性的武汉肺炎病毒，经过溯源现在能得到的证据指向是由物到人的病毒传播，因此怀疑病毒源头可能是来自德国进口的冷冻猪肘。这批货物是由德国不来梅（Bremen）港口出發，10月19日抵达天津，11月5日由经销商将这批没有开箱的货物全部运往山东德州。
  <br/>
  <br/>
  对于中国称病毒源头可能是来自德国进口的冷冻猪肘，德国联邦农业部9日的新闻稿反驳说，根据德国相关部门掌握情况，天津出现的疫情不太可能是德国猪肘造成，因为目前还没有出现食用或接触遭新冠病毒汙染的肉类和肉製品而被感染的先例，按照目前科学界对新冠病毒的了解，活体猪隻及家禽类都不会感染武汉肺炎病毒，遑论带原病毒，所以也不太可能透过此一途径传染给人类。
  <br/>
  <br/>
  德国联邦动物卫生研究所（Friedrich-Loeffler-Institut）接受农业部委托就动物对新冠病毒（ SARS-CoV-2）的易感性进行研究。 研究发现，新冠病毒无法在食物内或者表面复制繁殖，病毒的繁殖需要活体动物或者人类作为宿主。 研究表明，猪，鸡，鸭和火鸡都不能感染新冠病毒，另一方面，在牛的身上观察到最微小的病毒繁殖，但是，牛不会将病毒传播给其他动物，因此也不会传染给人类。
 </p>

 </p>
 <p>
  此外，德国农业部指出，虽然理论上还有另一种可能，也就是肉品厂或肉品加工厂的工人因为打喷嚏、咳嗽或者不乾淨的双手碰触肉製品，导致病毒附着在肉製品表面，但是这些工厂遵守的卫生规则和保护措施，已经将这种病原体汙染的风险降到最低。
  <br/>
  <br/>
  另外由于冠状病毒在环境中相对较低的稳定性，只在污染后的短时间内才有可能通过接触感染的方式传播。
  <br/>
  <br/>
  农业部还特别强调，中国今年9月中旬便公告禁止进口任何从德国输出的猪肉及相关製品，从德国到中国的海运货物大约需要四到六週的时间，因此推测这批猪肘应是在禁令颁布前就从德国离境的。
  <br/>
  <br/>
  另外，今年9月10日， 德国联邦农业部长克勒克纳（Julia Klöckner）证实，根据官方检测机构德国联邦动物卫生研究所分析的结果，在德国与波兰接壤地区发现的一头野猪感染非洲猪瘟，她说: 我们从施普勒奈瑟县一隻野猪，發现首例非洲猪瘟。由于德国是欧洲最大猪肉出口国之一，布兰登堡邦卫生部长诺纳马舍尔 (Ursula Nonnenmacher)当时就很担心肉品还能不能出口到中国：这必须在国家层面与欧盟进行对话。这将很快决定德国猪肉能否继续出口到中国，而中国目前是一个非常大的出口市场。
  <br/>
  <br/>
  果然，9月11日，中国海关总署及农业农村部立刻公告，禁止直接或间接从德国输入猪、野猪及其产品，以防止德国非洲猪瘟传入中国。所以，自9月12日起，德国猪肉就不能再出口到中国。
  <br/>
  <br/>
  <br/>
  自由亚洲电台记者蔡凌巴黎报导 责编：嘉远  网编：瑞哲

 <p>
  美国为实质拓展台湾国际空间成立的“全球合作暨训练架构”已经满5周年，台、美和去年加入的日本三方1日发布联合声明，展现持续强化三边关系的决心。
  <br/>
  <br/>
  美国、台湾2015年成立，日本去年加入的“全球合作暨训练架构”（GCTF）成立5周年，该合作平台目的旨在发挥台湾的强项和专业，协助区域国家建构、加强多边合作，增进台湾与理念相近国家的实质关系，拓展台湾国际空间。美日台三方代表1日在台湾外交部联合召开记者会，发表联合声明宣示将共同扩大全球合作暨训练架构的规模和合作领域，并纳入更多理念相近国家。
 </p>

  <b>
   AIT：“台湾能帮忙”在疫情期间见真章
  </b>
 </p>
 <p>
  美国在台协会处长郦英杰1日致词表示，全球合作暨训练架构的成立宗旨很简单，也就是台湾外交部近期常提到的“台湾能帮忙”（Taiwan Can Help）。美国一直以来努力争取台湾参与国际组织和论坛的机会，但同时也认为，台湾与世界交流、分享知识，“不应由国际组织领导阶层任意决定”，这样风险太大，所需解决的问题也过于复杂。

 <p>
  <br/>
  郦英杰还说，台湾控制此次疫情的表现是全世界最优秀的，各国争相学习“台湾模式”，然而因为中国施压，台湾今年仍无法参与世界卫生大会，此时此刻更应该感谢有全球合作暨训练架这种平台的存在，这个平台不受政治干预，唯一目的是让专业人士从台湾的经验慷慨受惠。
  <br/>
  <br/>
  郦英杰认为，这个平台是他对美台关系，甚至是全世界最长远、也最有趣的贡献之一。平台的研习与工作坊非常实际，聚焦于重点议题及创造实质成果，参与者都是真的会做事，而不是空有头衔的人，且活动通常以闭门方式进行，让参与者可无顾忌地交流想法。
  <br/>
  <br/>
  郦英杰称赞台湾为可信赖伙伴、民主制度典范及世界良善力量，美台关系是“真朋友，真进展”。

  <b>
   台外长：5年来成果有目共睹
  </b>
 </p>
 <p>
  台湾外交部长吴钊燮说，5年来，全球合作暨训练架构已就公共卫生、执法合作、妇女赋权、数位经济、网路安全及媒体识读等重要议题举办23场国际研习营，共计邀请全球38个国家、超过500位政府官员及专家与会受训，成果有目共睹。
  <br/>
  <br/>
  吴钊燮还说，他担任外交部长后，特别将这个平台的预算增加三倍，希望台湾能与更多国家合作，很高兴日本去年以“正式伙伴”身分加入，瑞典及澳洲等国也陆续加入合办行列，荷兰也规划加入。
  <br/>
  <br/>
  为了使这个平台更制度化运作，吴钊燮宣布，在外交部北美司成立“全球合作暨训练架构小组”作为平台的秘书处，由专责人员协调办理活动，并建立校友网络。
 </p>

  <br/>
  <br/>
  吴钊燮提到，全球合作暨训练架构去年首度移师台美日以外国家，9月在台湾友邦帕劳举办“2019南岛语言复振国际论坛”。外交部正积极规划在拉丁美洲举办活动。拉美地区中，智利及墨西哥去年首度参加这一平台的活动，主题为网路安全与新兴科技。

   日本驻台代表：日本不会离开台湾 也离不开台湾
  </b>
  <br/>
  <br/>
  日本台湾交流协会代表泉裕泰致词时说，日本、台湾、美国有自由民主人权共同价值，日方很荣幸能加入全球合作暨训练架构。这个架构主题务实、符合各国所需；例如对抗病毒，台湾获得国际高度评价，为国际社会做出很大贡献。

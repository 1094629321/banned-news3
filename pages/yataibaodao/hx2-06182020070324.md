 <p>
  在中国全国人大常委会宣布审议港版国安法的同一天，台湾大陆委员会公布“香港人道援助关怀行动专案”，将设立“台港服务交流办公室”，专门负责赴台港人就学、就业、投资事宜。
  <br/>
  <br/>
  就在台湾陆委会召开记者会宣布这个行动专案之前，又传出中国大陆解放军歼10、歼11型战机，进入台湾西南空域防空识别区。这已是本月第4次短暂进入台湾防空识别区，也是今年第11次飞往台海周边扰台。
 </p>

 </p>

 <p>
  <br/>
  <b>
   设立专门办公室负责港人事务
  </b>
  <br/>
  <br/>
  这项专案由主管两岸事务的大陆委员会主委陈明通亲上火线召开记者会说明政策，最大突破是成立专责办公室，并选订香港回归纪念日七月一日开始营运，由公私部门协力处理，政府编预算照顾港人，目前未订预算和人数上限。
  <br/>
  <br/>
  陆委会主委陈明通：“专案办公室将在7月1号开始营运。陆委会辅导成立‘台港经济文化合作策进会’，我们简称策进会。源起以促进台港交流与互动为宗旨，在旗下特别成立对外服务的“台港服务交流办公室”。这个办公室我们设置主任、副主任各一人，下设谘询服务、专案管理、行政事务等三个分组。”
 </p>

 </p>

 <p>
  陈明通在记者会公布专案办公室，设在台北市大安区信义路三段。“我们设有两个服务专线、谘询专线，第一个是（02）2700-3199（善意久久），第二是（02）2397-1088。”
  <br/>
  <br/>
  陈明通说，会增聘人力，并请相关机构支援办理港人来台就学、就业、投资创业，还有跨国企业、国际法人团体来台专案谘询与相关协处工作，以及人道援助，是“一条龙”的服务。
  <br/>
  <br/>
  <b>
   必须是合法入境台湾才能受到协助
  </b>
  <br/>
  <br/>
  对于最受瞩目的协助参与反送中运动的香港抗争者的庇护，陈明通指出，因为政治的理由，他的自由跟安全有受到威胁，政府的协助，（针对）必要的，就是根据港澳条例第18条，这种援助是报请行政院专案处理。
 </p>

 <p>
  至于已知前后约9000名示威者遭警方拘捕，面临检控、起诉等状况，如果人在香港如何向台湾提出相关申请流程？台湾如何协助？陈明通说，“我想最简单的前提，他是进入台湾，我们才根据法律给予协处。我们这个是‘协助’不是‘救援’。”
  <br/>
  <br/>
  对于部份抗争者遭限制出境，面临检控，旅行文件遭扣押，台湾政府对未持有合法身分文件或不是合法管道入境，是否排除协助？或视个案情况专案处理？陈明通说：“这个法律很清楚，他必须是合法入境。”
  <br/>
  <br/>
  陈明通指出，这项专案，是为因应中共强推港版国家安全法造成香港的变局，依总统及院长指示规划，成立办公室提供“客制化”亲切、便捷的服务及基本照顾，并吸引香港资金及专业人才，壮大台湾经济发展。

 <p>
  外界担心，港版国安法通过后，香港形定“一国一制”。陈明通说，国安疑虑问题，审查个案时也会纳入考量。被问到会否有“集体安置”的情况？陈明通表示，不会。
  <br/>
  <br/>
  陈明通还说，此举也是展现台湾关怀港人的决心与善意，因此办公室其中一支服务专线号码2700-3199，即代表台湾对香港的善意久久，也象征各方所坚守的普世价值将绵长亘久。
  <br/>
  <br/>
  此外，陆委会副主委邱垂正承诺，在协处港人赴台案件上，会特别注意保护其个人资讯的安全。

 <p>
  <b>
   在台港人组织：感恩和高兴
  </b>
  <br/>
  <br/>
  在台香港人组织“香港边城青年媒体”在第一时间发出声明，回应台湾提出的人道方案。香港边城青年媒体发言人Justine. K告诉自由亚洲电台：“香港边城青年认为今天陆委会公布的，比之前公布给予来台抗争者协助资讯的更为具体，在此我们感到感恩和高兴。我们期待台湾政府跟有经验协助抗争者的民间团体借经跟合作。”
  <br/>
  <br/>
  香港边青声明还说，香港的社会运动者安全每况愈下，相信台湾有关机构之后必定会遇到更多更新的问题，之前一直协助的民间团体期待和台港服务交流办公室继续得到有效沟通和及时回应。
  <br/>
  <br/>
  台湾“香港抗争者支援工作台湾义务律师团”发言人林俊宏接受自由亚洲电台采访说，对于陆委会公布的方案，确实愿意设立专责办公室，打算要接受港人生活照顾、相关庇护，和就学、就业，处理的不只是第18条，还包括港人赴台一般性移民等整体规划，值得肯定。

  <b>
   台湾义务律师团：呼吁对非法入境港人提供协助
  </b>
  <br/>
  <br/>
  不过，林俊宏提到：“他（陆委会）现在透过一个民间单位（台港经济文化合作策进会）来做这件事情，将来这些港人的申请，对于在法律上的效果是什么，目前我们还在深入了解中，因为涉及到将来策进会如果拒绝港人的申请，到底可不可以提起救济的问题。”
  <br/>
  <br/>
  此外，林俊宏指出，在港版国安法通过后，会有非常多参与反送中的香港抗争者被起诉，或面临相关侦查动作，旅行文件被香港政府扣留，无法循合法管道，就没办法获得台湾这项专案机制的协助。
  <br/>
  <br/>
  林俊宏：“我们是希望还是要有另外一个管道处理将来会非法入境的港人。会有一些比较急迫性的人，他们可能在当地碰到紧急的危害、危险，他们可能也没办法循正常、合法的管道出来了，他们只能走非法管道。这件事情在他们今天公布之前，我们就不断呼吁政府要有一个处理的方式，很可惜今天都还没有看到。”

  执政的民主进步党则发声明指出“援港专案，即刻启动”，延续政府自去年已展开的人道援助港人行动，进一步让“撑香港”措施机制化。民进党支持陆委会，在现有的法律基础上，即刻启动人道援助港人专案，并持续协助民间与政府对话，为完善协助港人措施与因应香港情势变化提供更多可行建议。民进党强调，撑香港，“说跟做，都重要！”

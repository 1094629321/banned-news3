 <p>
  近期美国在外交、国防和经济等领域连番出拳，使中国应接不暇。在这一背景下，中国驻美国大使崔天凯以低姿态表示，要推动两国关系重归正轨，并发展全面建设性关系。崔天凯的表态被认为与 “战狼外交”背道而驰，但有分析认为，这只是中方的缓兵之计。
  <br/>
  <br/>
  中国驻美国大使崔天凯近日应美国智库布鲁金斯学会邀请，与学者就美中关系进行视频交流。
 </p>
 <p>
  崔天凯说，搞“意识形态十字军东征”不能解决当今世界任何问题，注定遭到失败。他强调，中美两国应高度警惕任何试图把两国关系推向对抗冲突的险恶图谋，推动中美关系尽快回归正轨。
 </p>
 <p>
  近年中国的“战狼外交”为人所诟病，但视频里的崔天凯强调，中美两国必须和平共处、共同演进、管控分歧、扩大合作，发展全面、稳定和建设性的关系。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  他说，中方无意介入美国大选，并愿意随时随地与美国现政府合作，寻求解决问题的办法。
  <br/>
  <br/>
  <b>
   学者 : 中共不会改变强硬外交作风
  </b>
  <br/>
  <br/>
  美中两国近期在“香港国安法”等多个议题上针锋相对。特朗普政府对中国的施压一浪接一浪，周三单方面暂停或终止与香港的三条双边协议。在台海问题上，美国第七舰队导弹驱逐舰穿越台湾海峡。美国副国务卿克拉奇则发信呼吁美国高校放弃持有中概股，但香港时事评论员刘锐绍表示，目前看不到北京会改变强硬的外交作风。

  刘锐绍：“崔天凯的说法是缓兵之计。主要就是看到美国对中国动真格，而且不光是针对中国的国有企业，用民营企业包装的很多中共控制的企业也开始针对，譬如阿里巴巴。”
 </p>
 <p>
  崔天凯的一句“意识形态十字军东征”备受谈论。刘锐绍则认为，整个讲话最大问题也是出在这一句。
  <br/>
  刘锐绍：“中共一直以来都认为，什么问题都可以透过硬实力包括经济外交解决，从而主导这个世界，但是西方主要国家之所以有强的实力，主要因为他们软实力硬实力同时具备。习近平上台后有七个方面不能讲，包括普世价值，这恰恰是他未获世界认同的原因。”
  <br/>
  <br/>
  <b>
   邓小平路线和习近平路线并存
  </b>
  <br/>
  <br/>
  虽然”战狼外交”是目前的主旋律，但清华大学政治系原讲师吴强认为，邓小平“韬光养晦”的影响力犹在。
  <br/>
  <br/>
  吴强：“崔天凯在扮演‘和平鸽’的角色，相对温和的鸽派路线，一个鹰派，一个鸽派，基本上是邓小平路线和习近平路线并存的延续。”
  <br/>
  中国外长王毅早前也曾表示，愿与美国重启各层级、各领域的对话机制, 任何问题都可以拿到桌面上来谈。王毅还提到，“要拒绝脱钩，保持合作。”愿与美方在疫情防控、经济复苏等领域开展互利合作。” 外界也因而把崔天凯与他相提并论。
  <br/>
  吴强：“王毅最近也表示要全面激活与美国的对话。事实上他过去一直是以强硬姿态表现出来。包括崔天凯低姿态的讲话仍然把问题归到美国身上，仍然拒绝从中国自身找问题做反省。”
  <br/>
  吴强形容，中国外交官面对的是一条“死胡同”，话语权和回转空间都有限，尤其中国的外交路线主要由习近平决定，外界难以从崔天凯等外交官的讲话揣摩端倪。

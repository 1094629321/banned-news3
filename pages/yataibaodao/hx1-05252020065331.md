 <p>
  中国人大决定制定“香港版国安法”，港人上街抗议再被镇压。台湾的总统蔡英文24日在脸书声援港人，却同时抛出港澳条例可能因港版国安法通过停止适用，引发在野党质疑切割港人，到底与香港或北京站在一起?
  <br/>
  <br/>
  中国人大预计28日审议通过“香港版国安法”，港人24日上街再遭镇压。美国白宫警告北京将对中国经济制裁，英国首相传出有意接收香港“难⺠”。台湾蔡英文总统24日在脸书表示，“香港版国安法”如果实施，香港⺠主自由及司法独立的核心价值将受到严􏰂重侵蚀。为确保台港交流过程中，国家的安全及利益能得到充分保障。
 </p>
 <p>
 </p>
 <p>
  <br/>
  “依据香港澳门关系条例第60条，香港情势一旦发生变化，可停止适用该条例一部份或全部，我们希望香港情势不致走到这 一步。”蔡英文重申，协助受压迫港人方面，会在港澳条例基础提供港人必要协助。
 </p>

 <p>
  这是蔡英文首次抛出“香港澳门关系条例”可能停止适用，引发在野党质疑是“切割”香港。
  <br/>
  <br/>
  国⺠党党主席江启臣25日受访质疑，没有港澳条例，如何与港⺠站在一起?“呼吁⺠进党政府，对香港问题必须有所因应，不要让人觉得，去年选举时大声挺香港，选后让人感觉像要切割香港，停止港澳条例的适用，这等于是把香港切割掉。”
  <br/>
  <br/>
  <b>
   国⺠党质疑蔡英文切割港人
  </b>
  <br/>
  <br/>
  江启臣呼吁，北京要非常谨慎，不乐见香港的自由与法治，与北京所承诺的港人治港等都成为幻灭。台湾不分党派应设法让港澳条例在人道协助、政治庇护能够加以制度化、法制化。

 <p>
  ⺠进党籍立法委员钟佳滨接受自由亚洲电台采访指出，依照台湾宪法增修条文，香港、澳门非 “大陆地区”，也非“台湾地区”，也非“中华人⺠共和国辖下人⺠”，台湾把港澳当作“准外国”看待。在野党希望透过强化港澳条例第18条，给予香港政治难⺠庇护的时候，必须对香港整体法定定位先厘清定义。
  <br/>
  <br/>
  钟佳滨:“他(香港)到底是有别于中国的另一个准国家实体?还是他是中国大陆的一部份?对于中国大陆之下的⺠运人士如果受到政治迫害，台湾在两岸人⺠关系条例当中，就有 明文要给予像王丹这样的⺠运人士给予支持，对于美国或其他⺠主国家也是如此。”
  <br/>
  <br/>
  <b>
   ⺠进党立委:香港变成与中国无分别 国安疑虑须提升
  </b>
  <br/>
  <br/>
  钟佳滨提到，过去台湾依照港澳条例，对港澳人⺠赴台就学、就业、投资、移⺠、入出境管理等，都与中国大陆是分开处理的。“香港一旦失去了一个司法相对独立的自治实体的地位之后，我相信包括台湾在内、美国在内，可能会让北京了解，一旦他通过香港版国安法，过去香港所扮演一个中国跟世界接轨的优势就荡然无存。”
  <br/>
  <br/>
 </p>

  钟佳滨认为，台湾可另立专法，类似美国香港人权⺠主法案，处理对香港政治难⺠的救援。
  <br/>
  <br/>
  台湾朝野立委25日也关切中国预计审议“港版国安法”，国⺠党籍立委赖士葆质询，如果香港变成“一国一制”，香港澳门关系条例要不要“修正”或“废”?国家安全委员会副秘书长蔡明彦答询表示，假如中国人大会议通过“港版国安法”，代表中国对一国两制的承诺破灭，法律面问题都可以讨论。
  <br/>
  <br/>
  时代力党团指出，立法院应要求陆委会每月提出“香港特殊地位检讨报告”，检视香港政治、经济、社会面向的自主性。如果香港在中国影响下已不再保有自主的地位，就该取消港澳条例部份或全部的特殊地位，以保障台湾的国家安全与利益。

   台北市长柯文哲:中国此举加深台湾人⺠讨厌中国程度
  </b>
  <br/>
  <br/>
  台北市长柯文哲则在脸书表示，中共若要强行另订国安法，恐怕只会像“北风与太阳”故事中的北风，风愈吹，⺠心愈冷，逃愈远。台湾邻近香港，中国此举无异加深台湾人⺠讨厌中国的程度。

  <br/>
  蔡英文提到若港版国安法立法可能停止适用港澳条例，学界也有所讨论，陆委会前副主委、清大教授陈明祺认为，提供港人实质协助仍是蔡总统的论述底蕴。

  陈明祺说，中国大陆正加大管控极力促使港澳“内地化”，若再将港澳视为独立于中国大陆以外的个体，意义不大。⺠众对开放中国大陆、港澳人士也会抱有疑虑。政府提出停止适用港澳条例是在有限的作为上，希望对北京产生杠杆作用，让当局不要对香港“催逼甚紧”。
  <br/>
  <br/>
  淡江大学国际事务与战略研究所副教授黄介正指出，香港并不是“中华⺠国宪法”规定主权之外的地方，虽然中国大陆目前由中共控制，但不能忘记至少在1949年以后，香港对 “中华⺠国”所持有的特殊关系，如果忘记这点，把香港特别行政区所有事情都与北京政府画等号，很可能失去台湾的弹性的空间。

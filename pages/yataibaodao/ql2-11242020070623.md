 <p>
  公民记者张展被控“寻衅滋事罪”，目前羁押在上海市浦东新区看守所。辩护律师张科科上周二及本周一两次会见张展后，披露张展一直在绝食，而同囚室人员给她灌食。对于起诉书指张展恶意炒作武汉疫情，张展表示否认。
  <br/>
  <br/>
  张展在被羁押期间以绝食抗议当局对其指控，至今已经超过六个月，引起国际社会和人权组织高度关注。在当局的压力下，代理张展案的多位律师被迫解除代理协议。上周二及本周一，张展案新的辩护律师张科科，在浦东看守所会见当事人并向浦东新区法院邮寄委托手续。
 </p>
 <p>
 </p>

 </p>
 <p>
  <b>
   插管灌输伤及胃部头晕目眩
  </b>
  <br/>
  <br/>
  张科科律师当晚对自由亚洲电台证实，他顺利见到了张展，一提起张展的健康状况，张科科感到无奈。他说：“之前也听说张展长期绝食，被灌食，也有很多朋友关心她。所以我会见她的时候，问到这件事情。她说确实如此，她因为长期绝食，被同监室的人强行喂食。当然更多的时候对她插管灌食，所以导致她持续胃疼。今天也是这样子。 也导致头晕目眩，她说没有力气”。

 <p>
  <br/>
  张科科于本周一上午到浦东新区法院阅卷，下午三点到浦东新区看守所会见张展，一个小时后离开看守所。张科科说，身高约一米八的张展明显瘦了很多，无论是脸型还是身型都“缩小”了。脸色苍白，带著病态。
  <br/>
  <br/>
  <b>
   低血压放风期间无力走出囚室
  </b>
  <br/>
  <br/>
  张科科说，张展白天主要是静坐，可以看书。最近两三个月的放风时段，张展没有离开囚室：“前两个月有放风，但是她没有走出来放风。所以很长时间没有见到阳光了。不知道是不是疫情的原因。医生会给她做血压的测量和血液的检测，她的血压是比较低的，几乎每天都会测量血压。”
  <br/>
  <br/>
  今年2月3日，张展到武汉疫区采访，5月14日被捕。检察院的起诉书指控张展多次通过微信、“twiter”(推特)、“YouTube”(油管)等网络平台，发布大量虚假信息, 并接受境外媒体的采访，恶意炒作武汉市新型冠状病毒肺炎疫情。
  <br/>
  <br/>
  <b>
   起诉书未显示张展犯罪证据
  </b>
  <br/>
  <br/>
  对于当局的指控，张科科说，张展拒绝认罪：“关于（控方）证据这一块，她基本上没有认可的证据，她不认可。但是起诉书的内容也没有具体讲（证据），她觉得自己没有做错什么，她不认为自己有罪，更不承认指控的相关内容。”

  张科科说，张展感谢家人的支持和外界的关心。她希望外界不要过于关注她，更多帮助更需要的人。
  <br/>
  <br/>
  关注此案的甘肃维权律师李大伟对本台说，张展在行使法律赋予的权利，因此无罪：“武汉封城后，她冒着生命危险进入武汉，当时在武汉城内的人不惜一切代价要逃离武汉。她用语言、文字、视频向世人展示了武汉究竟发生了什么，武汉封城之后的真实状况。”
  <br/>
  <br/>
  为张展重获自由而呼吁的海外人权组织“人道中国”理事王剑虹对本台说，张展被捕至今已六个多月：“我们听到张科科律师会见她之后，终于传出来比较具体的张展身体情况和看守所内情形。我非常悲愤，张展一直在绝食、抗争，说他虚弱到无法走路，虚弱到连放风的时候不能走出去。我看了非常悲愤。”
  <br/>
  <br/>
  《上海市浦东新区人民检察院量刑建议书》称，张展“犯罪事实清楚、证据确实、充分、 应当以寻衅滋事罪追究其刑事责任，其法定刑为五年以下有期徒刑、拘役或者管制。建议判处被告人张展四年以上五年以下有期徒刑。

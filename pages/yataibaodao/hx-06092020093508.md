 <p>
  台湾国防部证实，美军一架运输机9日上午穿越台湾领空。此外，9日上午另有解放军军机多架次短暂进入西南空域。美军按照程序事先有申请，解放军则遭国军驱离。
  <br/>
  <br/>
  根据追踪飞机动态的推特帐号Golf9和网站AirNav资讯，美军1架C-40A“快船”运输机9日上午从日本冲绳起飞，罕见自基隆飞向台湾西海岸，穿越台湾领空，从台南外海离去。

 <p>
  国防部发言人史顺文接受自由亚洲电台采访表示，美军飞越台湾领空并非首次：“当然这不是首次，美机进入我空域，系按程序完成申请，期间并未降落我任何机场。”
  <br/>
  <br/>
  中国军机也出现在台湾空域。史顺文：“9日上午侦获中共苏恺30战机多架次，短暂进入台湾西南空域。除广播警告外，国军空中侦巡战机全程掌握并采积极驱离应处。”
  <br/>
  <br/>
  国防部在6月9日中午12点30分先发布解放军苏恺30的消息，十分钟后，12点40分再发布美军飞越台湾领空经程序申请并未降落的讯息。
  <br/>
  <br/>
  中、美军机同一天在台湾周边出现，高度敏感。台媒报导大都指向美军前脚走，解放军机随后到，但军方对谁先谁后都不透露。
  <br/>
  <br/>
  <b>
   军事评论员：美军越过台湾领空是单纯借道
  </b>
  <br/>
  <br/>
  军事评论员亓乐义9日接受自由亚洲电台采访说，就他了解，美军是从台湾东部海域飞往泰国，是单纯的通航事件，中方、美方各做各的，没有关连。

  亓乐义：“一般媒体有过多联想，这是美国军机的行政专机，可能有搭载他们的高级长官，从冲绳到泰国，一般走台湾东部。因为这几天国军空军有实弹演习，走到这边可能有风险不方便，所以才改道，这是一个单纯的通航事件 。当然他能够飞越我们领空附近海域，经过申请我们也同意，表示台湾和美国之间是有相当不错的伙伴关系，要不然我们也不会让他进来，他也不会提出申请”。
  <br/>
  <br/>
  至于解放军苏恺30战机有多架次经台湾西南部海域，亓乐义解读，这是解放军这几年一直有的动作，目的在增加他们战区海空或空军本身的战力，在这一带的巡航也一直都有。但中共确实在台海这边巡航跟演练的次数会愈来愈多，确实需要关注。
  <br/>
  <br/>
  亓乐义： “到现在为止，美国跟共军并没有在台海正面遭遇过，大部份都是你来我往。今天在同一天，也不是正面遭遇，所以之间是没有关系的，但是双方在台海确实在强化战备演训。”
 </p>
 <p>
  根据“AirNav” 公布的飞行轨迹资讯显示，编号“CNV7642”的美军1架C-40A从日本冲绳那霸市起飞，飞进台湾基隆上空，沿着台湾西海岸飞行，分别穿越基隆、新北、台北、桃园、新竹、苗栗、台中、彰化、嘉义、台南等地，最后在台南外海离开台湾，朝着南海前进。经常追踪飞行轨迹的“Golf9”在推特上表示，“CNV7642”飞越台湾西海岸，这是一个非常少见的飞行航线。
 </p>
 <p>
  <b>
   两种不同解读
  </b>
  <br/>
  <br/>
  日本《产经新闻》台北支局长矢板明夫接受自由亚洲电台采访，他认为美中两强军机同一天在台湾出现并不单纯。对美军飞越台湾领空，他解读：“美台军事交流进入新的阶段”。
 </p>
 <p>
 </p>

  <br/>
  矢板明夫：“这当然理由有各种各样，但是美军机在台湾上空得到台湾军方允许，以及飞跨了十个县市，这件事情毕竟是近年很少见的行动，这其实可以说美台军事交流，进入一个比较新的阶段，当然这对中国是有刺激性的行为。五二零以后，或罢韩以后，美台军事交流可能会更加频繁，中国可能会对此表示更加不满，中国今后这些小动作可能会继续。”
  <br/>
  <br/>
  矢板明夫分析，美国国会在2017、2018通过国防授权法，其中要求美国军舰和台湾军舰要实现互访。当时中国驻美公使李克新对美国国会表示，美台军舰互访，违反中美建交的基本精神，“美国军舰抵达高雄之日，就是解放军武力统一台湾之时”。2018年美军一艘研究船开进高雄港，引发轩然大波，第二个月韩国瑜当选高雄市长。台湾基进党向韩国瑜建议推进高雄港跟美军交流，韩消极回应。后来韩要推军事观光，其实就是给陆客看高雄港，其中很多中企在里面，所以高雄港一直动不了。
  <br/>
  <br/>
  矢板明夫认为，韩国瑜被罢免，可以使美台军事交流变的更顺畅。“当时他（韩国瑜）的当选，变成了阻碍台美军事交流的一块巨大的绊脚石。现在这个绊脚石被搬走了，今后台美的军事交流可能会更顺畅地往下推动在高雄建美军军港的议题。当然建军港需要很多条件，但是一个重要的难题已经解决了。”
 </p>
 <p>
  台湾前国防部副部长林中斌接受自由亚洲电台采访则有不同看法。他说，美军、解放军军机9日的行动，与罢韩扯不上关系。过去美军也曾进入台湾领空，像是特殊事件、维修、意外等。
 </p>

 <p>
  林中斌：“如果解放军机苏恺30在先，美军进入可以理解是一个反应。但如果没有前面苏恺进入的事情先发生，美军进入就是另一个状况，那就是说美军进入领空没有因为中共空军威胁，有别的事情。”
  <br/>
  <br/>
  <b>
   汉光演习将首次发布“空中威胁告警系统”
  </b>
  <br/>
  <br/>
  美中军机同一天出现在台湾及其附近空域，外界担心中美会否擦枪走火开战，林中斌重申绝对不会：“老共如果在东沙或其他方面有军事行动，等于是一些假动作，他要往右走的时候，做一些小动作，大的方向是往中间靠。小英政府也有很多迹象，也往那边表示善意。”
  <br/>
  <br/>
  此外，台湾国防部9日另召开记者会，说明国军一年一度汉光演习，与军民联合防空万安43号演习相关规划。汉光演习将于7月13至17日举行实兵操演，9月14至18日举行电脑兵棋对抗演练。为了避免人群聚集，今年的万安演习将不实施人车管制，并将首次发布“空中威胁告警系统”。
  <br/>
  <br/>
  国防部全动室简任主任韩冈明说：“考量疫情，如果按往年要做人车管制，还要做其他防护演练，会有群聚问题。所以今年参考警政署建议，我们也利用这个机会做全国性警报发放。这也是我们第一次这样做，也算是一种验证。”
  <br/>
  <br/>
  至于此次演习的想定，韩冈明说，想定不方便说明，会针对外来的飞弹，不管那一种形式，空中作战司令部会针对飞弹来袭发布警讯。

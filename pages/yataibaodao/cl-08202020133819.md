 <p>
  被称为 “欧洲最后的独裁者” 白俄罗斯总统卢卡申科，在总统大选后，虽然获得中国领导人习近平最早表达祝贺，却因外界质疑选举舞弊，面临国内示威抗议不断的局面，连欧洲联盟都在19日宣布不承认白俄罗斯总统选举结果。
  <br/>
  <br/>
  8月9日白俄罗斯（Belarus）总统大选结果揭晓，由掌权26年的总统卢卡申科（Alexander Lukashenko）再度连任，却被爆黑箱作票，首都明斯克（Minsk）爆发反政府示威，政府血腥镇压群众，欧盟领袖19日召开特别峰会，讨论如何回应白俄罗斯引发争议的大选。
 </p>
 <p>
 </p>
 <p>
  <br/>
  卢卡申科的政治对手季哈诺夫斯卡娅（Svetlana Tikhanovskaya）在紧急峰会之前，从她流亡所在的邻国立陶宛发出呼吁，敦促欧盟领导人不要承认这次总统选举，并呼吁在国际监督下举行新的选举。
 </p>

  <br/>
  在质疑选举既不自由也不公平的背景下，欧盟宣布站在示威者一边，拒绝承认选举结果，欧洲理事会（Euopean Council）主席米歇尔（Charles Michel）说，“我们不承认白俄罗斯当局提供的选举结果，白俄罗斯民众值得更好的。”
  <br/>
  <br/>
  欧盟还警告，很快将对需为暴力、镇压及选举舞弊负责的许多官员祭出制裁，并提供5300万欧元资金支持白俄罗斯的公民社会、独立媒体和防疫医疗。
  <br/>
  <br/>
  长期以来，白俄罗斯一直是俄罗斯最忠诚的邻国，在与欧盟成员国波兰与立陶宛的边界大规模加强军力。卢卡申科却对俄罗斯存有戒心，虽然在８月４号的演讲中，指责俄罗斯试图干预白俄罗斯选举，但普京却已表示愿向卢卡申科提供军事援助镇压，欧盟领袖则试图在不激怒莫斯科的情况下，支持白俄罗斯的民主力量。
  <br/>
  <br/>
  但是，欧盟希望避免乌克兰历史重演，6年前的乌克兰亲俄领袖在群众起义中被赶下台，引发俄国军队介入，到目前为止仍然是欧洲血腥冲突的战场，卢卡申科则说，他不会允许在白俄罗斯上演乌克兰的"广场革命"。
 </p>

  <br/>
  欧盟力挺白俄罗斯的民主势力，引来中国的反对，中国外交部发言人赵立坚说，“反对外部势力对白俄罗斯社会制造分裂和动荡。”
  <br/>
  <br/>
  毕竟在官方公布卢卡申科连任成功后，中国领导人习近平在外国领导人中最早向卢卡申科表达祝贺，并在贺电中强调中国高度重视与白俄罗斯关系。而在这次白罗斯大选前，卢卡申科还曾公开炫耀与习近平的交情，称自己与习近平早已相识，习还提到可帮白俄罗斯开发武器，并拨款援助。
  <br/>
  <br/>
  白俄罗斯在中国欧洲外交中的重要性相应提高，主要是美中角力下，东欧国家开始批评中国，与美国同一阵线，中国未来可能寻求白俄罗斯的支持。欧美国家最近纷纷抨击中国在新疆独裁甚至灭绝种族的举措，白俄罗斯却在七月的联合国人权理事会上，代表46个国家共同发言，可以看出白俄罗斯支持中国的新疆政策。
  <br/>
  <br/>
  但是，欧盟和西方世界如果对卢卡申科实施新的制裁行动，白俄罗斯经济将雪上加霜。被中国视为一带一路的支点国家，中国势必要支持这个可以让中国深入欧洲内陆的国家，因此，与欧美媒体报导数万人走上街头抗议政府的方向不一样，中国官方媒体新华网17日刊登新闻，报道白俄罗斯人上街支持卢卡申科连任、支持政府维护国家安宁与和平，也就不足以为奇了。

 <p>
  香港警方近日针对批评北京的人士发起一系列逮捕行动，欧盟除了发表声明表示密切关注案件的发展，也紧锣密鼓建立对侵犯人权者加以制裁的欧盟版马格尼茨基人权问责法案。欧盟外交与安全政策高级代表博雷利就指出，法案可能在12月10日国际人权日之前通过。
  <br/>
  <br/>
  香港警方1日逮捕8名民主派人士，欧盟对外行动署发言人2日发表声明说，将密切关注有关案件的进展，确保民主、人权和司法公正。
  <br/>
  <br/>
  8名被逮捕的民主派议员和前议员涉及今年5月立法会内务委员会冲突事件，警方指控他们在立法会议上的反抗行为违法，涉嫌触犯藐视罪以及干预议员、立法会人员或证人罪，定于5日在法庭审讯。如果罪名成立，被捕人士可能面临长达一年的监禁，而结果可能会影响其余泛民主派议员的议席，根据香港《基本法》的规定，任何议员被判超过1月个以上的徒刑，其他议员可以动议免除该名议员的职务。
 </p>

  欧盟对外事务部就被捕事件發声明: "欧盟会密切留意發展，包括是否有确保民主原则和尊重人权，强调保持公正是法治的主要原则。 "

 <p>
  中国外交部则重申，香港特区警方依法执法，不容任何人干扰。香港的司法事务也不容欧盟方面干预。
  <br/>
  <br/>
  针对批评北京人士发起的一系列拘捕行动仍在进行中，香港警方表示，不排除有更多人被捕。警方3日就短暂逮捕了香港电台（RTHK）时事节目《铿锵集》编导蔡玉玲，指控她在调查2019年反送中示威期间，发生在新界元朗的721事件一周年报道节目中，涉嫌为取得道路交通条例下的证明书而作出虚假陈述，香港多个记者工会则批评警方打压媒体报道。
  <br/>
  <br/>
  欧盟不只发出声援的声明，也紧锣密鼓建立对侵犯人权肇事者加以制裁的马格尼茨基人权问责机制，欧盟外交与安全政策高级代表博雷利（Josep Borrell ）10月31日就发表文章指出，在12月欧盟外长会议召开之前，可能通过这个法案，这是纪念12月10日国际人权日的恰当方式。
  <br/>
  <br/>
  欧盟外交部长会议去年12月9日一致通过将立法制裁侵犯人权者，推动欧盟版的马格尼兹基人权问责法，当时博雷利指出，"在好几个成员国的正式请求之下，我们已经同意启动全球制裁制度的准备工作，以解决严重侵犯人权的行径，欧盟的这项法案，就相当于美国的马格尼茨基法案。今天我们已经决定，我们对于展开准备工作，有强烈的共识。"
  <br/>
  <br/>
  欧盟现在正处于建立全球马格尼茨基人权问责机制的最后阶段，法案已于10月19日送交欧盟理事会，供成员国讨论和通过。
  <br/>
  <br/>
  博雷利指出，欧盟的法案不同于美国，只关注严重侵犯人权的行为，而美国的制度还包括打击腐败。博雷利强调，除了贸易、对话、财政支持等手段外，欧盟将制裁作为促进尊重人权的更广泛手段之一。欧盟决定实施金融制裁和旅行禁令，或实施涉及其他部门性限制，例如限制出售用于内部镇压的武器或其他设备。
  <br/>
  <br/>

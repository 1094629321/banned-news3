 <p>
  正在台湾访问的美国卫生部长阿札(Alex Azar)周一(10日)上午晋见蔡英文总统，蔡英文期盼台美双方未来在共同研发及生产疫苗与药品上，能有显著的进展，她并重申，政治不该凌驾健康之上。阿札下午赴台湾卫生福利部签署双边合作备忘录，被问到美国宣布退出世卫组织，如果成立新组织台湾是否有机会加入，阿札表示，将持续透过双边与多边合作继续为世界公卫努力，有机会就会与台湾讨论参与意愿。
  <br/>
  <br/>
  蔡英文总统10日上午在总统府接见阿札访问团一行人，蔡英文以拱手方式与访问团寒暄，会面过程中蔡英文与访宾都全程戴口罩。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  蔡英文在致词中提及，前几天，美国国务卿蓬佩奥（Mike Pompeo）也提到，美方一直希望台湾能参与世界卫生大会讨论，但遭中国阻挡。这一点也是台湾感到很遗憾的事情。
 </p>

  蔡英文：“在这波疫情中，台湾已经证明，不仅可以守住国内疫情防线，也有能力协助其他国家对抗疫情。政治因素不应该凌驾在健康人权之上，WHA排除台湾的参与，是违背医卫普世人权的原则。”
  <br/>
  <br/>
  蔡英文还特别感谢美国总统特朗普、国务卿蓬佩奥及阿札部长等许多美国友人，持续肯定“台湾模式”在全球对抗疫情上的贡献，并且大力支持台湾的国际参与。期待接下来，台美在共同研发和生产疫苗以及药品上，都能够有显著的进展。
 </p>
 <p>
  蔡英文强调，在双方共同合作努力下，不仅是在防疫，在各领域的交流，都可以有更多的突破、更丰硕的成果，一起为印太区域的和平稳定发展，贡献力量。

 <p>
  <b>
   阿札代表特朗普总统 传递对台强力支持与友谊
   <br/>
  </b>
  <br/>
  美国卫生部长阿札在致词时首先提及，今天非常荣幸能够来到这里，传达川普总统对台湾的强力支持与友谊。对于蔡总统今年开始第二任期，他也表达恭贺之意。诚如美国国务卿蓬佩奥在今年5月恭贺蔡总统就任的贺词中提到，蔡总统在领导台湾这个蓬勃的民主社会时所展现的勇气及视野，对于区域、乃至于全球都是深刻的启发。他也对前总统李登辉逝世表达哀悼之意，并推崇李登辉为“台湾的民主之父，同时也是二十世纪全球民主浪潮中重要的领袖。”
  <br/>
  <br/>
  阿札：“在特朗普总统的带领下，美国已透过各种方式，展现出对台湾民主成就的欣羡之意。川普总统签署法案，强化台美之间的伙伴关系。另外，美国在台协会也在2018年启用一栋全新的大楼，展现美国对双方友谊的高度珍视。”
  <br/>
  <br/>
  阿札指出，此行访台以及与总统会面的讨论重点，着重在台湾的医卫与防疫成就，以及双方未来如何进一步合作，来共同预防、侦测，并一起回应各项公卫挑战。

 <p>
  <b>
   美台签署“医卫合作了解备忘录”AIT：真友谊、真进展
  </b>
  <br/>
  <br/>
  阿札下午赴卫生福利部疾管署与台湾抗疫有成的陈时中部长会面，两人并出席共同见证，美台签署“医卫合作了解备忘录”，双方由美国事务委员会主任委员杨珍妮及美国在台协会(AIT)台北办事处长郦英杰代表双方签署，显示台美双边伙伴关系紧密、坚实与友好。
  <br/>
  <br/>
  卫福部认为，台湾和美国在医疗卫生方面已有20多年的合作，这次签署“医疗合作了解备忘录”，是首次提高行政层级到美国卫生部。美国代表团发新闻稿指出，台湾在国际社会的角色很重要，也证实一个自由民主资讯透明的国家，可以成功对抗新冠肺炎。郦英杰还特别以中文强调，此时的台美关系是“真友谊、真进展”。
  <br/>
  <br/>
  <b>
   阿札访台 凸显台湾对世界公卫重要性
  </b>
  <br/>
  <br/>
  在签署仪式之后，阿札随即召开访台的首次记者会，有媒体提及过去美国也曾有阁员级官员访台但是相对低调，阿札这次访台与不少台湾官员会面，这是否代表美国对台湾政策改变，如果是的话，为什么是现在？
  <br/>
  <br/>
  阿札表示，这次的访问代表美国肯定美台之间深厚的友谊以及伙伴关系，包括安全、经济、卫生、以及民主开放透明等价值。阿札强调，美台双方在卫生领域长期以来有有着密切合作的历史，在这次疫情流行中，美国要支持并且肯定能透过透明的方式共享价值的伙伴。阿札也提及，台湾在四年前遭世卫大会移除观察员身份，美国的应处。
 </p>

  阿札：“在特朗普总统的指示下，我和国务卿蓬佩奥疾呼，要恢复台湾在世卫大会观察员的身份。但是中国共产党还有世卫组织阻止我们这么做，这对特朗普政府对世卫组织感到最为挫折的一点，另外世卫组织也没有进行改革。]
  <br/>
  <br/>
  阿札举了个例子，2018年刚果爆发伊波拉疫情，当时世卫干事长谭德赛还疾呼希望所有会员国可以慷慨解囊，美国捐了钱、中国没有。台湾虽被剥夺观察员的身份，非常慷慨地想捐100万美元给世卫，在北京的指示下，世卫组织拒绝台湾的捐款。
  <br/>
  <br/>
  阿札：“我这次代表特朗普总统造访台湾，要特别凸显台湾对于世界公卫社群的重要地位，还有展现台湾过去与未来能做出的贡献。最后要强调美国和台湾之间，强劲与密切的伙伴关系和友谊。”
  <br/>
  <br/>
  <b>
   成立新组织取代世卫邀台参与？阿札：会与台湾讨论
  </b>
  <br/>
  <br/>
  阿札也回应台媒提问，美国总统特朗普日前宣布退出世卫，甚至可能成立新的国际公卫组织，而台湾有机会参与吗？
  <br/>
  <br/>
  阿札认为，美国总统特朗普是因美国对世卫组织没有意愿也没有能力进行改革，感到非常挫折，且世卫也没有办法确保会员国有公正及透明的行为。阿札说，美国一直以来，及未来都会持续是全球公卫社群最大的资助者。美国目前仍是世卫成员，因世卫有其多边机制，退出需要一点时间，待往后正式离开后，会寻找其他适合的方式，与其他国家进行双边或多边方式合作，继续透过全球卫生安全倡议等议程，为全球公卫努力。

  在此同时，日本产经新闻9日引用不具名人士表示，台美双方可能讨论成立新的国际组织以取代“世界卫生组织”（WHO）。台湾外交部严正澄清，该报导与事实不符，外交部表示遗憾。

  <b>
   阿札访台 共机歼11、歼10一度逾越海峡中线
  </b>
  <br/>
  <br/>
  台湾的空军司令部10日证实，中共空军所属歼11、歼10型机，10日上午陆续短暂逾越海峡中线，国军空军除以地面防空飞弹全程监控外，并于第一时间广播警告，运用空中侦巡兵力强势驱离。
  <br/>
  <br/>
  空军司令部指出，共机刻意侵扰及破坏台海现况之行为，已严重破坏区域的安全与稳定。而中国外交部发言人赵立坚10日被问及阿札访问台湾一事重申，中方一贯坚决反对美台官方往来，已就有关情况向美国提出严正交涉。
